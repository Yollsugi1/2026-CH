# 국순당 프로젝트 관리 가이드
> Creatorhood 내부용 — Claude 자동화 연동 가이드

## 1. 프로젝트 개요

국순당 콘텐츠 제작 프로젝트는 **이중 트랙 일정 관리** 체계로 운영됩니다.

**Track 1 — 클라이언트 공유용** ("2026 국순당 콘텐츠" DB)
- 국순당 담당자(연정님)와 공유하는 업로드 스케쥴 중심의 DB
- 시리즈: 순당포착 / 취하지아니한가 / 1막1장 / 기타

**Track 2 — 내부 제작 관리용** ("내부 업무관리" DB)
- 업로드일이 밀리지 않도록 사전에 제작 스케쥴을 선제적으로 체크하는 DB
- 상태 흐름: 예정 → 촬영 → 1차공유 → 2차공유 → 최종공유 → 업로드 → 완료

---

## 2. 핵심: 3단계 공유 시스템

국순당 측에는 **30%-80%-100%** 의 3단계로 공유한다고 안내되어 있음.
우리 내부적으로는 **20%-50%-80%** 단계에서 준비하여 리소스를 세이브하고 선제적으로 대응.

### 단계 매핑

| 우리 내부 완성도 | 국순당에 말한 완성도 | 내부 DB 태그 | 공유 시점 | 피드백 기한 |
|---|---|---|---|---|
| **20%** | **30%** (1차 공유) | `1차공유` | 촬영 D+3 또는 업로드 D-14 | 공유 D+2 |
| **50%** | **80%** (2차 공유) | `2차공유` | 업로드 D-10 | 공유 D+2 |
| **80%** | **100%** (최종 공유) | `최종공유` | 업로드 D-5 | 업로드 D-3 |

### 각 단계에서 국순당에 요청하는 피드백

1. **1차 공유 (30%)**: 컷 편집 흐름 확인 — 흐름이 맞는지, 컷 추가/수정/삭제 희망 구간
2. **2차 공유 (80%)**: 디테일 확인 — 자막, 색감, 전환효과, B-roll 등
3. **최종 공유 (100%)**: 피드백 반영 확인 → 업로드 예약

> **중요**: 1차에서 흐름이 확정되어야 2차 편집으로 진행. 후반부 마이너 수정이 연쇄적 수정을 유발하므로 가장 빠른 단계에서 공유하는 것이 핵심.

## 3. 업로드일 기준 역산 타임라인

```
D-21~14  촬영 완료 + 소재 정리
D-14     1차 편집 완료 (내부 20%) → 1차 공유 (→30%)
D-12     클라이언트 피드백 마감
D-12~10  2차 편집 (내부 50%)
D-10     2차 공유 (→80%)
D-8      클라이언트 피드백 마감
D-8~5    최종 편집 (내부 80%)
D-5      최종 공유 (→100%)
D-3      최종 승인 → 업로드 예약
D-day    업로드
```

## 4. 자동화 구성 (Claude 스킬)

현재 세팅된 Claude 스킬:

| 스킬명 | 용도 | 트리거 |
|---|---|---|
| `kuksundang-schedule-manager` | 제작 일정 관리 + 공유 준비 알림 | "국순당 일정", "공유 준비", "마감 확인" |
| `kuksundang-daily-report` | 매일 오전 Slack 리포트 | 매일 오전 9시 or "데일리 리포트" |
| `kuksundang-weekly-report` | 주간 종합 리포트 | 매주 월요일 or "주간 리포트" |
| `kuksundang-client-update` | 클라이언트 보고 페이지 생성 | "클라이언트 업데이트" |

### 사용법
Claude에게 다음과 같이 요청:
- "국순당 오늘 일정 확인해줘"
- "이번 주 공유 준비해야 하는 거 있어?"
- "국순당 주간 리포트 보내줘"
- "국순당 클라이언트 업데이트 만들어줘"

## 5. 시리즈별 특이사항

### 순당포착 (고객인터뷰형)
- 사전 리드타임: 기획+섭외 약 +7일 추가
- 산출물: 스틸컷 1 + 영상 2편
- 업장 리스트업 병행 필요

### 취하지아니한가 (먹방토크형)
- 외국인 호스트 스케쥴 조율 필수
- 한국어 능통 + 한국문화 이해도 높은 호스트

### 1막1장 (TPO 공감형)
- 월별 시즌 테마 매칭 (3월 신학기, 5월 나들이 등)
- 영상 10-15초, 공간/분위기 중심
- 핵심 지표: 시청지속시간(완료율)
