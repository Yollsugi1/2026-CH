# 국순당 자동화 스킬 설치 가이드
> 팀원 개별 Claude 환경에 설치하는 방법

## 필요한 파일 목록

```
.claude/skills/
├── kuksundang-schedule-manager/
│   ├── SKILL.md                              ← 제작 일정 관리 스킬
│   └── references/
│       └── production-workflow.md             ← 워크플로우 레퍼런스
├── kuksundang-daily-report/
│   └── SKILL.md                              ← 데일리 리포트 스킬
├── kuksundang-weekly-report/
│   └── SKILL.md                              ← 주간 리포트 스킬
└── kuksundang-client-update/
    └── SKILL.md                              ← 클라이언트 보고 스킬
```

## 설치 방법

### 1. 파일 복사
이 폴더에 있는 `skills/` 디렉토리를 자신의 `.claude/skills/` 경로에 복사합니다.

### 2. 필수 커넥터 연결
Claude Cowork에서 다음 커넥터가 연결되어 있어야 합니다:
- **Notion** — 데이터베이스 조회 및 페이지 생성
- **Slack** — 채널 메시지 발송

### 3. 사용법

Claude에게 자연어로 요청하면 됩니다:

| 요청 예시 | 실행되는 스킬 |
|---|---|
| "국순당 오늘 할 일 정리해줘" | kuksundang-daily-report |
| "이번 주 공유 준비해야 하는 거 있어?" | kuksundang-schedule-manager |
| "국순당 주간 리포트 보내줘" | kuksundang-weekly-report |
| "클라이언트 업데이트 만들어줘" | kuksundang-client-update |
| "국순당 마감 임박한 거 확인" | kuksundang-schedule-manager |

## 핵심 참고사항: 3단계 공유 시스템

우리 내부 완성도와 국순당에 공유하는 완성도가 다릅니다:

| 우리 (실제) | 국순당 (안내) | 의미 |
|---|---|---|
| 20% | 30% | 컷 편집 흐름 확인 |
| 50% | 80% | 디테일 편집 확인 |
| 80% | 100% | 최종 확인 |

> 클라이언트 커뮤니케이션 시 반드시 30/80/100으로 표현할 것.
> 내부 논의 시에는 20/50/80으로 소통 가능.
