# 카카오 노션 협업 교육 커리큘럼

## 교육 개요

| 항목 | 내용 |
|------|------|
| **교육명** | 노션으로 만드는 팀 협업 시스템 — 실시간 협업부터 데이터 아카이빙까지 |
| **교육 목적** | 노션을 활용하여 팀 단위의 실시간 협업 체계를 구축하고, 프로젝트·태스크·회의록·레퍼런스를 연결하는 데이터 기반 업무 구조를 설계할 수 있다 |
| **대상** | 전사 임직원 (직무·직급 무관 / 실무 중심) |
| **규모** | 차수당 약 30명 |
| **시간** | 3시간 (이론 1h + 실습 2h) |
| **형식** | 오프라인 강의 + 실습 (노션 템플릿 제공) |
| **요구사항** | 노션 계정, 노트북 지참 |

> 본 커리큘럼은 범용 기업교육 기준으로 구성되었으며, 교육 확정 시 조직의 업무 환경과 기존 노션 활용 수준에 맞춰 커스터마이징이 가능합니다.

## 커리큘럼 구성

### 모듈 1. 노션 기본기 — 같은 언어로 시작하기 (30분)
> 이론 + 데모 | 0:00 ~ 0:30

**학습 목표**: 노션의 핵심 구조를 이해하고, 실습에 필요한 기본 조작에 익숙해진다.

| 순서 | 내용 | 시간 |
|------|------|------|
| 1-1 | **노션은 왜 협업 툴인가** — 문서 도구를 넘어서는 노션의 본질 (워크스페이스, 실시간 협업, 권한 체계) | 10분 |
| 1-2 | **블록과 페이지 구조** — 노션의 모든 것은 블록이다 (블록 타입, 페이지 계층, 드래그&드롭) | 10분 |
| 1-3 | **생산성을 높이는 핵심 기능** — 단축키, 마크다운, 멘션, 코멘트, 슬래시 명령어 | 10분 |

**실습 체크포인트**: 슬래시 명령어로 토글·콜아웃 블록 추가, 옆 사람을 멘션하여 코멘트 남기기

### 모듈 2. 데이터베이스 — 협업의 기반을 세우다 (30분)
> 이론 + 실습 | 0:30 ~ 1:00

**학습 목표**: 노션 데이터베이스의 핵심 개념을 이해하고, 업무에 맞는 DB를 직접 구성할 수 있다.

| 순서 | 내용 | 시간 |
|------|------|------|
| 2-1 | **데이터베이스란?** — 테이블 vs 보드 vs 캘린더 vs 갤러리 (뷰의 힘) | 8분 |
| 2-2 | **속성(Property) 설계** — 텍스트, 셀렉트, 날짜, 사람, 상태 등 핵심 속성 | 7분 |
| 2-3 | **필터·정렬·그룹** — 같은 데이터, 다른 시각 (나에게 필요한 뷰 만들기) | 7분 |
| 2-4 | **[핵심] 관계형 속성(Relation)** — DB와 DB를 연결하는 노션의 강력한 무기 | 8분 |

**실습**: 프로젝트 DB 만들기 → 속성 설정 → 보드 뷰·테이블 뷰 전환 → 필터 적용

### 모듈 3. 팀 협업 구조 설계 — 우리 팀의 업무 허브 만들기 (40분)
> 실습 중심 | 1:10 ~ 1:50

**학습 목표**: 팀 단위 협업을 위한 노션 워크스페이스 구조를 직접 설계하고 구축한다.

| 순서 | 내용 | 시간 |
|------|------|------|
| 3-1 | **팀 홈페이지 설계** — 팀 대시보드 페이지 구성 (공지, 멤버, 바로가기) | 10분 |
| 3-2 | **프로젝트 ↔ 태스크 연동** — 관계형 DB를 활용한 업무 추적 시스템 구축 | 15분 |
| 3-3 | **회의록 시스템** — 회의록 DB를 프로젝트와 연결하여 논의 → 실행 흐름 만들기 | 15분 |

**실습 결과물**: 팀 홈페이지 + 프로젝트DB + 태스크DB + 회의록DB가 서로 연결된 팀 협업 허브

## 교육 특징

### 1. 실무 중심 실습
- 강의 시간의 **2/3가 직접 실습**으로 구성
- 모든 실습은 **즉시 업무에 적용 가능한 템플릿** 기반으로 진행
- 교육 후 실습 템플릿을 그대로 팀에 가져가 활용 가능

### 2. 체계적인 학습 설계
- **개인 → 팀 → 조직** 순서로 확장하는 자연스러운 학습 흐름
- 모듈 간 연결: 앞에서 만든 DB가 다음 모듈에서 실제로 활용됨
- 완성형 예시를 먼저 보여주고, 직접 구축하는 Top-down 방식

### 3. 맞춤형 교육 가능
- 교육 전 수강생 수준 사전 설문을 통해 기본기 비중 조절 가능
- 조직의 기존 노션 활용 방식에 맞춰 시나리오 커스터마이징 가능
- 직무별 세분화 교육으로 확장 가능 (PM, 마케팅, HR 등)
