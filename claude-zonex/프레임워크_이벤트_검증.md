## 1. 이벤트 실현가능성 평가 기준

### 1.1 컴바인 (2박 3일, 10-16명) 실현가능성 평가

| 평가 항목 | 평가 질문 | 판단 기준 | 현재 상태 |
|---|---|---|---|
| **시간 충분성** | 6개 라운드 x N종목을 2박 3일에 소화할 수 있는가? | 라운드당 소요시간 시뮬레이션 + 여유 시간 20% 이상 | 미검증 |
| **인원 밀도** | 10-16명이 동시에 또는 순차적으로 움직일 때 장소 수용 가능한가? | 동시 진행 가능 종목 수, 대기 인원 동선 | 미검증 |
| **장비 가용성** | 필요 장비 전수 조달 가능한가? (어썰트 바이크, 바벨, 로프 등) | 장비 리스트 대비 조달 확인율 100% | 미검증 |
| **촬영 양립성** | 측정과 촬영을 동시에 수행할 때 양쪽 품질이 유지되는가? | 카메라 수, 촬영 크루 규모, 조명 배치 | 미검증 |
| **안전성** | 고강도 종목(1RM 데드리프트, 파워 클린 등)에 대한 안전 프로토콜이 있는가? | 스포터 배치, 의무 인력, 응급 동선 | 미검증 |
| **선수 컨디션 관리** | 2박 3일간 6개 라운드를 수행하는 선수의 피로 누적이 결과 왜곡을 일으키지 않는가? | 라운드 간 최소 회복 시간, 라운드 순서 전략 | 미검증 |
| **날씨/외부 변수** | 실내 vs 실외 종목이 섞여 있을 때 날씨에 따른 대체안이 있는가? | 대체 프로토콜 유무 | 미검증 |

## 2. 측정 프로토콜 검증 체크리스트

### 2.1 검증 기준 정의

모든 측정 종목은 아래 4가지 기준을 통과해야 한다:

| 기준 | 정의 | 실패 시 결과 |
|---|---|---|
| **공정성** | 체중/체형/종목 배경에 의한 불공정이 통제되었는가 | 특정 종목 출신에게 구조적 유리/불리 → 스탯 신뢰도 하락 |
| **안전성** | 부상 위험이 관리 가능한 수준인가 | 부상 발생 → 법적 리스크 + 시리즈 중단 위험 |
| **재현성** | 같은 조건에서 반복 측정 시 유사한 결과가 나오는가 | 결과 신뢰도 하락 → Hexagon 스탯의 가치 상실 |
| **촬영 적합성** | 카메라에 잡히는 시각적 임팩트가 있는가 | 측정은 정확하나 콘텐츠로 무의미 |

## 3. 스코어링 시스템 설계 원칙

### 3.1 핵심 결정 사항

| 결정 | 옵션 A | 옵션 B | 각각의 함의 |
|---|---|---|---|
| **스케일 기준** | 백분위 (참가자 내 상대적 위치) | 절대 기준 (사전 정의된 기준표) | 백분위: 참가자 수에 따라 점수 의미 변동 / 절대: 외부 비교 가능하나 기준 설정 난이도 높음 |
| **체중 보정** | 점수에 체중 보정 계수 적용 | 보정 없이 원 점수 사용 | 보정: 공정하지만 직관성 저하 / 무보정: 직관적이지만 체중 차 반영 안 됨 |
| **능력 간 가중치** | 6개 능력 동일 비중 | 특정 능력에 가중치 부여 | 동일: 단순하고 공정 / 가중치: "완성형 운동인"의 정의에 따라 설득력 있을 수 있으나 논쟁 유발 |
| **동일 능력 내 복수 종목** | 종목별 점수 평균 | 종목별 점수 합산 후 스케일링 | 평균: 종목 수에 무관 / 합산: 종목 수가 많은 능력에 유리 |

### 3.2 통계적 고려 사항

| 요소 | 설명 | 검증 필요 사항 |
|---|---|---|
| **분포 가정** | 10-16명의 소규모 샘플에서 백분위가 유의미한가? | N=12일 때 백분위 1점 차이 = 실질적 차이가 아닐 수 있음 |
| **이상치 처리** | 한 능력에서 압도적으로 뛰어난 선수가 있으면 나머지 전원의 점수가 몰림 | 상한/하한 클리핑 or z-score 변환 검토 |
| **종목 간 비교 형평성** | 비프테스트 레벨 12 = 데드리프트 체중의 2배 = FRAN 3분? 이 등가를 누가 설정하는가 | 자문위원/전문가 패널 검증 필요 |
| **시즌 간 비교** | Season 1 ENGINE 85 = Season 2 ENGINE 85인가? | 절대 기준이 아니면 시즌 간 비교 불가 → 백분위의 근본 한계 |
| **복합 종목 점수화** | FRAN 타임을 0-100으로 어떻게 변환? 선형? 로그? | 엘리트(2분) vs 일반(6분)의 격차를 어떤 곡선으로 반영할지 |

### 3.3 실무적 고려 사항

| 요소 | 설명 |
|---|---|
| **실시간 계산** | 컴바인 현장에서 측정 직후 점수를 계산해 방송에 활용할 수 있어야 함 → 자동화 스프레드시트 or 앱 필요 |
| **시각적 표현** | 0-100 숫자가 육각형 레이더 차트에서 직관적으로 보여야 함 → 점수 간 차이가 시각적으로 유의미한 크기여야 |
| **내러티브 해석성** | 점수가 해설에서 바로 스토리텔링에 활용 가능해야 함 → "ENGINE 92면 상위 10% 수준" 같은 해석이 가능한 구조 |
| **선수 납득** | 선수가 자기 점수에 납득할 수 있어야 함 → 채점 기준 사전 공개, 이의 제기 프로세스 |
| **외부 비교 가능성** | "Zone X ENGINE 92는 크로스핏 게임즈 참가자 대비 어느 수준?"이라는 질문에 답할 수 있으면 가치 상승 |

### 3.4 추천 접근

1단계 (Season 0): **참가자 내 백분위 기반**, 소규모이므로 단순하게 시작
- 각 종목 순위 → 백분위 환산 → 능력당 복수 종목 평균 → 0-100 스케일링
- 체중 보정은 적용하지 않되, 종목별 처방(체중 비율)으로 간접 보정

2단계 (Season 1+): 데이터 축적 후 **절대 기준 + 백분위 혼합**으로 진화

## 5. 컴바인 vs 리그 운영 분리 프레임

### 5.1 근본적 차이

| 차원 | 컴바인 (Season 0) | 리그 (Season 1+) |
|---|---|---|
| **목적** | 데이터 수집 + 콘텐츠 촬영 | 경쟁 + 콘텐츠 |
| **긴장의 원천** | "내 능력이 드러난다" (자기 자신과의 싸움) | "상대를 이겨야 한다" (직접 대결) |
| **결과의 성격** | 점수/랭킹 (순위) | 승패 (이분법) |
| **운영 우선순위** | 정확한 측정 > 엔터테인먼트 | 공정한 경기 > 정확한 측정 |
| **실패 허용 범위** | 측정 실패 = 재측정 가능 | 경기 실패 = 패배 확정 (재경기 없음) |
| **선수 컨디션** | 3일간 누적 피로 관리 필수 | 1일 이벤트 내 컨디션 관리 |
| **장비 전환 빈도** | 라운드별 대규모 전환 (6회) | 매치별 소규모 전환 (8-12회) |
| **촬영 방식** | 다큐 스타일 (인터뷰 + 리액션 + 측정) | 스포츠 중계 스타일 (경기 중심) |

### 5.2 운영 방식 차이

| 항목 | 컴바인 | 리그 |
|---|---|---|
| **타임테이블** | 라운드 단위 블록 스케줄 (1라운드 = 2-3시간) | 매치 단위 롤링 스케줄 (1매치 = 20-30분 턴어라운드) |
| **선수 관리** | 전원이 모든 라운드에 참가 → 동시 관리 | 매치 카드에 따라 순차 출전 → 개별 관리 |
| **장비 셋업** | 라운드 간 대규모 장비 전환 (바이크 → 바벨존 → 오픈 플로어) | 매치 간 소규모 전환 (CYA 선택에 따라 장비 변경) |
| **기록 시스템** | 6능력 x N종목 x 10-16명 = 수백 개 데이터 포인트 | 매치별 Phase 타임/결과만 기록 |
| **의사결정** | 사전 확정된 프로토콜 (라운드 순서/종목 고정) | 실시간 의사결정 (CYA 선택, 매치 순서 조정) |
| **콘텐츠 흐름** | 촬영 후 편집 (후처리 중심) | 촬영 = 콘텐츠 (실시간 중심) |

### 5.3 분리 운영 시 주의 사항

- 컴바인 장비/장소 세팅과 리그 장비/장소 세팅은 **다른 매뉴얼**이 필요
- 컴바인에서의 저지 기준과 리그에서의 저지 기준이 **동일**해야 스탯-경기 간 일관성 유지
- 컴바인 스태프와 리그 스태프의 역할이 다름 → **별도 교육/브리핑** 필요
- 컴바인은 "모든 선수가 항상 촬영 대상"이지만, 리그는 "경기 중인 선수만 촬영 대상" → 카메라 운용이 완전히 다름

## 7. 현재 기획에 대한 이벤트 이슈/질문/도발

이벤트 기획자로서 "현장에서 실제로 돌아가느냐"를 기준으로, 현재 기획 문서에서 발견한 이슈들.

### 7.1 컴바인 — 시간/물량

**Q1. 10-16명을 2박 3일에 6능력 전부 측정하는 게 시간적으로 가능한가?**

- 라운드별 종목 구조: ENGINE(비프테스트 전원 + 바이크 1:1 페어) + GRIT(FRAN 히트 2-3명씩) + FORCE(데드 엘리미네이션 + 벤치 1:1) + BLAST(클린 엘리미네이션 + 박스점프 1:1 + 메드볼) + REFLEX(L-Drill 개인 + 미러/추격 1:1 토너먼트 + BlazePod) + CONTROL(8동작 체크리스트 전원 순차)
- **러프 시뮬레이션** (16명 기준):
  - ENGINE: 비프테스트 전원 30-40분 + 바이크 1:1 8매치 x 5분 = 40분 + 세팅 → **약 2시간**
  - GRIT: FRAN 히트 5-6히트 x 10분 = 60분 + 세팅 → **약 1.5시간**
  - FORCE: 데드리프트 엘리미네이션 8-10라운드 x 5분 = 50분 + 벤치 1:1 8매치 x 5분 → **약 2.5시간**
  - BLAST: 파워클린 엘리미네이션 + 박스점프 토너먼트 + 메드볼 → **약 2.5시간**
  - REFLEX: L-Drill 16명 x 3분 + 미러/추격 토너먼트 + BlazePod → **약 3시간**
  - CONTROL: 8동작 x 16명 x 3분/동작 = 6.4시간 → **약 3-4시간** (동시 진행 시 단축 가능)
  - **순수 측정만 약 15시간**
  - 여기에 장비 전환, 인터뷰, 식사, 휴식, 브리핑, 돌발 상황을 더하면 **실제 필요시간 20-25시간**
  - 2박 3일 = 실 가동시간 약 24-30시간 (Day 1: 8시간, Day 2: 10시간, Day 3: 8시간)
  - → **빡빡하지만 가능. 단, 여유 시간이 거의 없어 돌발 시 스케줄 붕괴 위험**

**Q2. CONTROL의 Movement Mastery Checklist는 16명 x 8동작이면 128회 시도다. 시간은?**
- 동작당 준비+시도+판정+리액션 = 최소 2-3분
- 128회 x 2.5분 = 320분 = **5시간 20분**
- 동시 진행(스테이션 분할)으로 절반 단축해도 **약 2시간 40분**
- CONTROL 라운드만으로 거의 반나절. 현실적인가?

**Q3. 라운드 순서가 선수 컨디션에 미치는 영향은?**
- Day 1에 ENGINE + GRIT → Day 2에 FORCE + BLAST → Day 3에 REFLEX + CONTROL (현재 EP 기준)
- Day 1에 심폐+근지구력을 몰아치면 Day 2 근력/폭발력에 피로 영향
- "Day 1에 바이크 올아웃 + FRAN을 다 뛰고, 다음 날 데드리프트 1RM을 쳐?" → 공정성 의문
- **대안**: 라운드 순서를 선수 회복을 고려해 재배치하거나, 라운드 간 최소 회복시간 규정

### 7.2 컴바인 — 장비/장소

**Q4. 어썰트 바이크 2대 이상 동일 기종 확보가 가능한가?**

- 대여? 구매? 협찬? 비용은?

**Q5. 로프 클라임용 장소 확보가 가능한가?**
- 천장 높이 최소 4m, 구조적으로 로프 매달기 가능한 빔/구조물 필요
- 일반 피트니스 센터에서는 불가능한 경우 많음
- 크로스핏 박스라면 가능 → 장소가 크로스핏 박스여야 하는 제약?

**Q6. 장소에 "상징성"을 넣겠다고 했는데 (블랙컴뱃의 폐교회처럼), 장비 인프라와 상징성을 양립할 수 있는 장소가 있는가?**
- 상징적인 장소(창고, 공장, 폐건물 등) = 피트니스 인프라 제로
- 피트니스 인프라 있는 장소(크로스핏 박스, 체육관) = 상징성 부족
- 둘 다 잡으려면: 큰 크로스핏 박스를 빌려서 세트 디자인으로 분위기 전환? → 추가 비용

### 7.3 리그 — 매치 운영

**Q7. 8-12매치를 하루에 촬영하면 총 몇 시간이 걸리는가?**
- 매치당 소요: 준비 5분 + Phase 1(3분) + 전환(2분) + Phase 2(3분) + 전환(2분) + Phase 3(3분) + 결과/리액션(3분) + 정리(2분) = **약 23분**
- 매치 간 장비 교체 (CYA 선택에 따라 장비 변경) = 5-10분
- 10매치 x 30분 = **약 5시간** (순수 경기 시간)
- 선수 워밍업, 브리핑, 휴식, 식사 포함 → **실제 약 7-9시간**
- → 이른 아침부터 저녁까지 꽉 찬 하루. 가능은 하지만 여유 없음.

**Q8. Choose Your Attack 선택을 현장에서 어떤 물리적 절차로 진행하는가?**
- 문서에서 "상체/심폐/하체 공격 중 선택"이라고 했는데, 구체적 절차가 없음
  - 선수가 말로 외치는가? ("심폐 공격!")
  - 카드를 뒤집는가?
  - 디스플레이에 터치하는가?
  - 사전에 봉투에 넣어두는가?
- **이 절차가 콘텐츠의 가장 핵심적인 순간**이므로 연출적으로도 강력해야 함
- 현장에서 실행 가능한 CYA 선택 메커니즘 설계 필요

**Q9. CYA 선택 후 장비 전환에 걸리는 시간은?**
- Phase 2에서 "심폐 공격" 선택 → 어썰트 바이크 + 더블언더 세팅
- Phase 3에서 "상체 공격" 선택 → 바이크 치우고 로프/링 세팅
- 이 전환이 **경기 흐름을 끊지 않으면서** 진행 가능한가?
- Phase 간 전환 시간이 곧 선수의 휴식 시간이 됨 → 전환 시간이 경기 결과에 영향

**Q10. Phase 간 전환 시간을 고정할 것인가, 유동적으로 둘 것인가?**
- 고정(예: 정확히 90초): 공정하지만 복잡한 장비 전환 시 시간 부족 가능
- 유동적(장비 세팅 완료 시): 매치마다 휴식 시간 달라짐 → 공정성 문제

### 7.4 안전/법적

**Q11. 참가자 전원의 상해보험 가입이 가능한가?**
- 이벤트 단위 단체 상해보험 존재 여부 확인 필요
- 크로스핏 대회/하이록스 대회는 어떤 보험을 쓰는가? → 레퍼런스 필요
- 1RM 데드리프트, 파워 클린, 로프 클라임 같은 고위험 종목이 보험 적용 범위에 포함되는가?

**Q12. 파워 클린 1RM 엘리미네이션에서 올림픽 리프팅 경험이 전혀 없는 선수가 참가해도 되는가?**
- 기술 미숙 상태의 1RM 시도 = 손목/어깨/등 부상 위험 매우 높음
- **사전 기술 스크리닝 or 최소 기술 기준** 없이 전원 참가 시키면 안전 사고 위험
- 대안: 미숙자는 감량 기준(체중의 X% 이하) 적용, or 대체 종목

**Q13. 미러/추격 드릴에서 신체 접촉이 발생하면?**
- "제한 공간에서 2명이 맞붙음" → 충돌 불가피
- 신체 접촉 규칙, 반칙 기준, 부상 시 중단 기준이 없음
- 격투기 배경 선수와 러너가 추격전을 하면 물리적 능력 차이로 위험한 상황 발생 가능

### 7.5 스코어링

**Q14. 컴바인 현장에서 실시간 점수 계산이 가능한 시스템이 준비되어 있는가?**
- EP 10 "THE HEXAGON"에서 스크린에 점수를 하나씩 리빌하는 연출
- 이를 위해: 모든 라운드 결과 → 실시간 점수 변환 → 그래픽 반영 파이프라인 필요
- 수기 엑셀? 자체 앱? 외부 서비스?
- **그래픽 팀과 데이터 팀의 실시간 연동**이 필요한데, 그 인프라가 있는가?

**Q15. 능력별 1위("王") 발표 시 동점이면?**
- 컴바인 소규모(10-16명)에서 동점 가능성 높음
- 타이브레이커 규칙이 없으면 현장에서 즉석 판단해야 → 공정성 논란

### 7.6 인력/구조

**Q16. 저지/심판의 판정 기준이 문서화되어 있는가?**
- CONTROL의 합격/불합격 기준 → "풀 깊이 스쿼트"란 정확히 어디까지?
- 풀업 기준: 턱이 바를 넘으면? 쇄골이 바에 닿으면?
- 파워 클린 성공 기준: 풀 스탠딩? 프론트 랙 포지션 유지 2초?
- **기준서 없으면 저지마다 다르게 판정 → 선수 항의 → 방송 사고**

**Q17. 해설/캐스터를 누가 하는가? 언제 확보하는가?**
- 기획 문서에 "2인 해설: 실황 캐스터 + 전문가 해설"이라고 했는데
- 크로스핏/피트니스를 아는 캐스터 풀이 있는가?
- 컴바인 촬영(5월 말)까지 남은 시간 약 2.5개월 — 섭외 + 준비 가능?

### 7.7 빌드업 → 컴바인 전환

**Q18. 빌드업에서 간단 측정한 데이터와 컴바인 정밀 측정 데이터의 일관성은?**
