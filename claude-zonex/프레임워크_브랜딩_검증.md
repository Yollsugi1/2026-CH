# Zone X 브랜드 검증 프레임워크

> 작성일: 2026.03.18
> 작성: 브랜딩 에이전트 (Claude)
> 용도: Zone X의 모든 산출물이 "하나의 세계관"에 부합하는지 검증하는 기준 문서
> 핵심 질문: **"이것이 Zone X다운가?"**

---

## 0. 프레임워크의 전제

Zone X는 하나의 브랜드 아래 두 개의 제품을 운영한다:
- **컴바인** (The Proving Ground): 측정 기반 엔터테인먼트 시리즈
- **리그** (Choose Your Attack): 전략 기반 1:1 경기 리그

이 둘은 성격이 다르지만 같은 세계관에 속해야 한다. 이 프레임워크는 "Zone X"라는 브랜드의 일관성을 감사하고, Phase별 톤 전환이 자연스러운지, 그래픽 자산이 세계관에 부합하는지, 채널 아이덴티티가 통일되어 있는지를 검증한다.

## 1. 브랜드 일관성 감사 체크리스트

### 1.1 언어 일관성

| 검증 항목 | 기준 | 확인 방법 |
|---|---|---|
| **브랜드명 표기** | "Zone X" (대문자 Z, 대문자 X, 공백 포함). "ZONE X", "ZoneX", "zone x" 불가 | 모든 문서/영상/그래픽에서 검색 |
| **Hexagon 6능력 표기** | 카테고리명(ENGINE, GRIT 등)은 항상 대문자 full cap. 능력명(Stamina, Endurance 등)은 Title Case | 스탯 카드, 해설 자막, 문서 전체 |
| **CYA 공격 유형명** | "심폐 공격 / 상체 공격 / 하체 공격" 또는 "Cardio Attack / Upper Body Attack / Lower Body Attack" — 혼용 시 한 콘텐츠 내에서 통일 | 경기 그래픽, 해설 스크립트 |
| **라운드명** | 컴바인: THE FURNACE, BREAKING POINT, THE WALL, DETONATION, THE CHASE, THE MASTER TEST — 정확한 영문 대문자 표기 | 시리즈 타이틀 카드, 자막 |
| **아키타입명** | The Engine, The Tank, The Explosive, The All-Rounder, The Specialist, The Dark Horse — "The" 포함, Title Case | 선수 소개, 해설, SNS |
| **시리즈명** | "The Proving Ground" (컴바인), "Choose Your Attack" (리그 핵심 메커니즘) — 정확한 표기 | 모든 터치포인트 |
| **"완성형 운동인" 프레이밍** | 한국어 커뮤니케이션에서 일관되게 사용. "최고의 운동인", "최강의 선수" 등 유사 표현과 혼용 금지 | 대외 문서, 자막, SNS 카피 |

### 1.2 비주얼 일관성

| 검증 항목 | 기준 | 확인 방법 |
|---|---|---|
| **컬러 팔레트 준수** | 기존 설정: #0A0A0A(다크), #E94560(액센트 레드). 모든 그래픽에서 이 팔레트 기반 | 디자인 산출물 전수 검사 |
| **타이포 준수** | 타이틀: Akira Expanded, 기술 텍스트: OCR A Std (pre-pivot 설정). 변경 시 명시적 결정 필요 | 썸네일, 카드, 배너 |
| **다크 테마 일관성** | Zone X = 어두운 톤 기반. 밝은 배경의 그래픽은 예외적 케이스로만 | 전체 비주얼 |
| **Hexagon 시각화 규격** | 육각형 레이더 차트의 꼭짓점 순서, 라인 스타일, 채움 컬러가 모든 매체에서 동일 | 스탯 카드, 매치 프리뷰 |
| **로고 사용 규정** | 최소 크기, 여백, 배경색 위 사용 기준 정의 필요 (현재 미확정) | 모든 로고 적용 건 |

### 1.3 톤 일관성

| 검증 항목 | 기준 | 확인 방법 |
|---|---|---|
| **브랜드 보이스** | 자신감 있되 허세 없음. 데이터와 논리 기반. 도발적이되 비하 없음. "이 시장의 빈 자리" 톤 | 카피, 내레이션, SNS |
| **운동인 존중** | 모든 종목의 선수를 동등하게 존중. 특정 종목 비하 절대 금지. "다른 배경에서 같은 판에 선다"는 프레임 | 인터뷰 편집, 자막, 해설 |
| **경쟁의 품격** | 공격성은 게임 메커니즘 안에서만. 인신공격, 과도한 악감정 조장 금지. UFC의 프로모션이되 WWE의 대본은 아님 | 선수 인터뷰 유도, 편집 방향 |

### 1.4 세계관 일관성

| 검증 항목 | 기준 | 확인 방법 |
|---|---|---|
| **Hexagon = 배경 데이터** | 6능력 스탯이 경기 룰에 직접 개입하지 않음을 항상 명확히 | 해설 스크립트, 그래픽 표현 |
| **CYA = 경기 그 자체** | Choose Your Attack 선택이 경기의 전략적 핵심임을 강조 | 본 매치 영상 구조 |
| **3단계 연결** | 빌드업 → 컴바인 → 리그가 하나의 서사 아크임을 모든 콘텐츠에서 의식 | 에피소드 간 떡밥 연결 |

### 2.2 전환 체크포인트

톤이 바뀌는 세 지점에서 다음을 확인한다:

**전환 1: 빌드업 → 컴바인 (5월 말)**

| 체크 항목 | 질문 |
|---|---|
| 톤 에스컬레이션 | 빌드업의 "가벼운 만남"에서 컴바인의 "진검승부"로 톤이 자연스럽게 올라가는가? |
| 캐릭터 연결 | 빌드업에서 소개된 선수가 컴바인에 등장할 때, 시청자가 "아, 그 사람!"이라고 인식하는가? |
| 정보 누적 | 빌드업에서 심어둔 떡밥(강점/약점 선언)이 컴바인 결과와 대비되어 드라마를 만드는가? |
| 비주얼 전환 | 숏폼 핸드헬드 → 시리즈 시네마틱으로의 비주얼 업그레이드가 느껴지는가? |

**전환 2: 컴바인 → 본 리그 (6월 중순)**

| 체크 항목 | 질문 |
|---|---|
| 스탯의 실전 전환 | Hexagon 스탯이 "측정 결과"에서 "전략적 데이터"로 의미가 전환되는가? |
| 매치업 기대감 | 컴바인 마지막 에피소드에서 공개된 매치 카드가 시청자에게 "이건 봐야 해"라는 감정을 만드는가? |
| 포맷 인식 전환 | "시리즈물"에서 "스포츠 리그"로 시청자의 인식 프레임이 바뀌는가? |
| 선수 → 전사 | 선수가 "측정 대상"에서 "전략을 구사하는 경쟁자"로 포지셔닝이 바뀌는가? |

### 3.2 자산별 디자인 요건

#### Hexagon 스탯 카드

- **사이즈**: 1:1 (인스타 피드), 9:16 (스토리/릴스), 16:9 (유튜브 오버레이)
- **필수 요소**: 선수 사진, 이름, 종목, 아키타입명, 6꼭짓점 레이더 차트, 각 능력 점수(0~100), 평균 점수
- **디자인 기준**: 다크 배경(#0A0A0A 기반), 액센트 레드(#E94560)로 강조, Hexagon 채움은 반투명 컬러
- **변형**: 선수 개인용(1명) + 비교용(2명 나란히)
- **핵심**: 시청자가 캡처해서 SNS에 공유하고 싶은 디자인이어야 한다. "이 선수 스탯 봐" 리액션 유도

#### CYA 선택 카드

- **사이즈**: 16:9 (영상 내 삽입), 1:1 (SNS)
- **필수 요소**: 선택한 공격 유형명 + 아이콘, 선택한 선수 이름, 해당 공격 유형의 동작 2개 표시
- **디자인 기준**: 선택 순간의 임팩트 — 컬러 플래시, 충격파 효과 등. 3가지 공격 유형별 고유 컬러 지정 필요
- **핵심**: "심폐 공격을 선택했습니다!" 자막과 동시에 화면에 뜨는 순간이 경기의 가장 짜릿한 포인트

#### 공격 유형별 컬러 제안 (검증 필요)

| 공격 유형 | 제안 컬러 | 근거 |
|---|---|---|
| 심폐 공격 | 블루 계열 | 산소, 심장, 호흡 연상 |
| 상체 공격 | 레드 계열 | 근육, 힘, 강렬함 연상 |
| 하체 공격 | 그린/옐로 계열 | 스피드, 폭발, 에너지 연상 |

> 주의: 이 컬러가 Zone X의 메인 액센트 레드(#E94560)와 충돌하지 않는지 확인 필요. Section 7 참조.

#### 6능력 아이콘 세트

- **스타일**: 미니멀 라인 아이콘 또는 솔리드 아이콘 (모노크롬 기본, 컬러 버전 병행)
- **각 능력별 심볼 제안**:
  - ENGINE: 엔진/심장/불꽃
  - GRIT: 이빨 악물기/주먹/반복 화살표
  - FORCE: 바벨/방패/산
  - BLAST: 번개/폭발/위쪽 화살표
  - REFLEX: 번개+방향 전환/미러/화살 회전
  - CONTROL: 저울/다이아몬드/원 안의 점
- **용도**: 스탯 카드, 라운드 타이틀, SNS 아이콘, 영상 자막 오버레이

## 5. 컴바인 vs 리그 브랜드 분리-통합 프레임

### 5.1 핵심 원칙: "한 지붕, 두 성격"

Zone X는 하나의 우산 브랜드다. 컴바인과 리그는 같은 세계관의 "챕터"이지, 별개의 브랜드가 아니다. 하지만 각각의 콘텐츠적 성격은 분명히 다르다.

```
             ┌───────────────────────────────┐
             │          ZONE X               │
             │    "완성형 운동인을 가린다"      │
             │    (우산 브랜드)                │
             └───────┬───────────┬───────────┘
                     │           │
         ┌───────────┴──┐  ┌────┴───────────┐
         │  THE PROVING  │  │ CHOOSE YOUR    │
         │  GROUND       │  │ ATTACK         │
         │  (컴바인)     │  │ (리그)         │
         │              │  │                │
         │  "누가 진짜  │  │ "누가 이기는가" │
         │   인가?"     │  │                │
         └──────────────┘  └────────────────┘
```

### 5.2 통합 요소 (반드시 공유)

| 요소 | 공유 방식 | 이유 |
|---|---|---|
| **Zone X 로고** | 컴바인/리그 모두 동일 로고 사용 | 하나의 브랜드 인식 |

| **선수 프로필** | 컴바인에서 만들어진 프로필이 리그까지 이어짐 | 서사의 연속성 |
| **"완성형 운동인"** | 핵심 프레이밍 동일 | 메시지의 일관성 |
| **톤앤매너 기반** | 자신감, 데이터 기반, 도발적이되 품격 있는 톤 | 브랜드 보이스 통일 |

### 5.4 전환 가이드라인

컴바인 → 리그 전환은 EP 10~11(Hexagon 리빌 + 매치 카드 공개)에서 일어난다. 이 전환 에피소드에서:

1. **비주얼 전환**: 컴바인 UI가 서서히 리그 UI로 교체. EP 10에서 Hexagon이 공개될 때, 매치 프리뷰 카드 디자인을 처음 노출
2. **사운드 전환**: 컴바인의 서스펜스 음악이 리그의 전투적 음악으로 전환
3. **프레이밍 전환**: "누가 진짜인가?"에서 "누가 이기는가?"로 내레이션 전환
4. **CYA 첫 노출**: 컴바인 마지막 에피소드에서 CYA 메커니즘을 처음 설명. "이 스탯이 경기에서 이런 의미를 갖는다"

### 7.2 해결해야 할 브랜딩 질문들

**Q1. "Hexagon"이라는 네이밍은 시각적으로 충분히 강력한가?**
- 현재: 6능력 체계의 이름이 "The Hexagon"이고 시각화도 육각형 레이더 차트
- 도전: "Hexagon"은 일반 명사. 검색 시 기하학, 인텔 CPU, 게임 등과 경쟁. "Zone X Hexagon"으로 항상 풀네임을 써야 하는가?
- 대안 검토: "The Six", "Hex Profile", "Zone X Code" 등 — 또는 "The Hexagon"이 충분히 맥락 안에서 작동하니 문제없는가?

**Q2. 6능력별 고유 컬러가 필요한가?**
- 현재: 6능력에 개별 컬러가 지정되지 않음. Hexagon 차트는 단색 또는 그라디언트
- 필요성: 6능력별 컬러가 있으면 아이콘/차트/인포그래픽에서 즉시 구분 가능
- 위험: 6색이 추가되면 Zone X의 메인 컬러 팔레트(다크+레드)와 충돌 가능
- 결정 필요: 6능력 컬러를 도입한다면, 메인 팔레트의 "보조색" 레벨로 정의해야 함

**Q3. CYA 3공격 유형의 컬러 정체성은 어떻게?**
- Section 3.2에서 제안한 심폐(블루)/상체(레드)/하체(그린) 컬러가 메인 레드(#E94560)와 충돌
- 상체 공격을 레드로 하면 Zone X 메인 컬러와 같아져서 "공격 유형 컬러"인지 "브랜드 컬러"인지 혼동
- 대안: 3공격 유형은 아이콘/형태로 구분하고 컬러는 사용하지 않는 방법도 있음

**Q4. "Choose Your Attack"은 브랜드명인가 메커니즘명인가?**
- 현재: 경기 내 전략 메커니즘의 이름이자, 리그 전체를 대표하는 표현
- 도전: 리그 자체의 이름은 "Zone X"인데, CYA가 별도 브랜드처럼 인식될 수 있음
- 정리 필요: CYA는 "Zone X 경기의 핵심 룰"로 포지셔닝하되, 리그 자체와 동일시하지 않는 선 유지

**Q5. 촬영 장소의 브랜드 역할**
- 블랙컴뱃은 폐교회/폐공장 같은 장소 자체가 브랜드 정체성
- Zone X 컴바인의 장소가 TBD인데, 장소의 상징성이 세계관에 큰 영향
- 질문: "Zone X다운 장소"는 어떤 곳인가? 체육관? 산업 공간? 야외? — 장소가 곧 브랜드

**Q6. 해설자/MC의 브랜드 역할**
- UFC에서 조 로건/브루스 버퍼가 브랜드의 일부인 것처럼, Zone X의 보이스가 될 사람이 필요
- 이 사람의 캐릭터가 Zone X의 톤을 정의함
- 질문: 수현이 직접 하는가? 별도 캐스터를 세우는가? 둘 다?

**Q7. 컴바인과 리그의 채널 분리 여부**
- 현재 가정: Zone X라는 하나의 채널에서 빌드업/컴바인/리그 모두 업로드
- 대안: 컴바인은 별도 시리즈 채널? 리그만 Zone X 메인 채널?
- 추천: Season 1에서는 하나의 채널이 맞음. 콘텐츠 볼륨이 충분해지면(Season 2+) 분리 검토

### 7.3 도발적 질문 — "이것이 Zone X다운가?"

1. **"6개 능력이 너무 많지 않은가?"** — 시청자가 6개를 다 기억하고 구분할 수 있는가? 3~4개로 줄이면 더 직관적이지 않은가? (반론: 6이어야 Hexagon이고, 6이어야 "완성형"의 의미가 살아남)

2. **"The Proving Ground, The Furnace, Breaking Point... 영어 이름이 한국 시청자에게 작동하는가?"** — 블랙컴뱃은 한국 콘텐츠지만 영어 네이밍. UFC도 한국에서 영어 그대로 소비됨. 하지만 Zone X는 아직 브랜드 인지도가 0인 상태에서 영어 이름이 진입 장벽이 될 수 있음

3. **"Hexagon 스탯이 컴바인 촬영 전에 존재할 수 있는가?"** — 빌드업에서 간단 측정을 하지만 정식 스탯은 컴바인에서 나옴. 빌드업 기간에 Hexagon 그래픽을 미리 사용하면 혼란? (답: 빌드업에서는 Hexagon 그래픽을 사용하지 않고, "6능력"이라는 개념만 교육)

4. **"Zone X의 X는 아직 세 가지 의미(Xtreme/미지수/크로스오버)를 다 담고 있는가?"** — Pivot 후 "카디오" 포커스가 약해지면서 "Xtreme(심폐 Zone 5를 넘어서는)"의 의미가 희석됨. X의 의미를 재정의할 필요가 있는가?

5. **"스폰서십 제안서의 컨텍스트가 pivot 이전 버전에 머물러 있지 않은가?"** — 대외 문서들이 "8팀 팀전 78편+"을 여전히 기반으로 하고 있다면, 신속히 업데이트해야 함
