## Part 1: 방향성 재검토 (260318 P100 차별화 해결 추가)

### 1.1 기존 기획의 지향점

**High-level concept**: "피지컬 트레이닝 버전의 스우파" / "카디오를 경쟁하는 스우파"

**핵심 논리 구조**:
- 시장 갭: "하는 시장(하이록스/크로스핏)은 폭발, 보는 시장은 비어 있다"
- 포지셔닝: 방송(피지컬100)도 아니고, 대회(하이록스)도 아닌 — "팀 기반 시즌제 스포츠 엔터테인먼트 IP"
- 구조: 8개 팀 x 카디오 미션 x 10주 시즌제 리그
- BM: 스폰서십 + 콘텐츠 78편+
- 레퍼런스: Kings League, Black Combat, DRL

**팀 아이데이션 (세한)의 A+B=C**:
"각 종목을 대표하는 최정예 크루가 카디오라는 중립 전장에서 가장 우월한 종목을 증명하는 시네마틱 스포츠 서바이벌"

### 1.2 발생한 우려들

**우려 1: 방송 프로그램과의 변별력**
- 싱크로게임(TVING) 시청 후, Zone X가 담으려는 그림과 근본적으로 유사하다는 것 발견
- 방송사/OTT는 훨씬 큰 자본과 캐스팅 파워를 가짐 — 포맷 레벨에서는 경쟁 불가

**우려 2: 스테이크(stakes)의 부재 — 핵심 규명**
> "이 쇼의 최종점을 왜 봐야 하지?"

| | 싱크로게임/피지컬100 | MMA/블랙컴뱃 |
|---|---|---|
| 우승의 의미 | 이 쇼에서 이김 | 실제 세계 랭킹과 명성이 바뀜 |
| 패배의 비용 | 이 쇼에서 탈락 | 커뮤니티 내 지위, 다음 경기 기회 |
| 참가자 동기 | 출연료/노출 | 커리어, 종목의 명예 |
| 시청자 투자 | "재밌는 방송" | "내가 응원하는 선수/종목의 싸움" |

→ Zone X가 가져야 할 것: 쇼 바깥에서도 의미 있는 스테이크

**우려 3: 카디오 포맷 자체의 시청 재미**
- "카디오 종목은 반복적인 동작이라 crossfit games를 봐도 재미가 떨어짐"
- 게임 설계로 해결하려 하면 다시 방송 포맷으로 귀결

**우려 4: 경기의 정당성**
- 세한이 제기한 "어떤 종목이 사람의 능력을 가장 키워줄까?"는 게임으로 어떤 것을 경쟁하게 할 것이냐와 직결
- Zone X라는 명칭보다 게임의 정당성이 먼저

**우려 5: 피지컬100 연상 문제**
- 팀전이든 개인전이든 "피트니스 서바이벌" 구조에서 벗어나기 어려움

### 1.3 새로운 방향성

**핵심 전환**: 스우파(방송 포맷 이식) → MMA/UFC (리그 그 자체를 창조)

MMA가 복싱/레슬링/주짓수 등 투기 종목들의 환경에서 하나의 리그를 만들어낸 것처럼,
Zone X는 크로스핏/하이록스/러닝/MMA 등 생활체육 전반에서 하나의 카디오 리그를 만드는 것.

> MMA가 "가장 완전한 격투가"를 가리는 리그인 것처럼, Zone X는 "가장 완성형 운동인"을 가리는 리그다.

| | 기존 ("스우파 이식") | 신방향 ("MMA형 리그") |
|---|---|---|
| 주인공 | 팀 vs 팀 게임 | 선수 개인 + 종목 배경의 리그 |
| 구조 | 팀전 서바이벌 토너먼트 | 1:1 매치 → 랭킹 → 챔피언십 |
| 서사의 원천 | 팀 내/팀 간 게임 드라마 | 선수 커리어, 종목 명예, 랭킹 다툼 |
| 포맷 인상 | 방송 프로그램 | 스포츠 리그 |
| 차별점 출처 | 게임 설계 | 리그 구조 자체 |

### 1.4 확정된 방향들

**Q1. 참가 단위** — 개인 선수를 default 단위로. 블랙컴뱃 초기처럼 "프로 오디션" 형태의 토너먼트로 참여자를 수급하는 과정 자체를 콘텐츠화. 하이브리드 경로.

**Q3. 토너먼트 vs 리그** — Q1과 연결. 시즌1 = 오디션형 토너먼트 → 이후 본 리그.

**참여자 범위** — 크로스핏/하이록스 뿐 아니라 러너, MMA, 주짓수, 농구/축구 사회인리그 플레이어까지.

**인원 클래스** — MMA 체급처럼 1인전/2인전/4인전으로 경쟁 구분 (상세 설계 예정).

### 1.5 Physical 100 차별화 — "리그화"가 해결한다 (260318 신규)

1.2에서 제기된 우려 5(피지컬100 연상 문제)에 대한 근본적 해결.

**문제 정리**:
- "피트니스 서바이벌" → 피지컬 100 연상 불가피
- 포맷 레벨에서의 차별화(게임 설계)만으로는 "방송 프로그램의 변형"이라는 인식을 벗어나기 어려움
- 근본적으로 다른 차원의 차별화가 필요

**해결: "리그화(League-ification)"**

| 항목 | Physical 100 (쇼) | Zone X (리그) |
|---|---|---|
| **형태** | 1회성 TV/OTT 프로그램 | **영구 리그** (시즌제, 벨트, 방어전) |
| **측정 기준** | 에피소드마다 다른 도전과제 | **Hexagon 6능력이라는 단일 잣대** |
| **스테이크** | 이 쇼의 우승 + 상금 | **랭킹, 벨트, 커리어** |
| **참가자 동기** | 방송 출연, 상금, 노출 | **종목 명예, 커뮤니티 내 위상, 다음 경기** |
| **서사 구조** | 탈락/생존 (단발) | **성장/하락/복귀 (누적)** |
| **종료 후** | 방송 출연 이력 | **다음 경기가 있다. 랭킹이 남는다.** |

**왜 "리그화"가 근본적 차별점인가**:

1. **스테이크의 차원이 다르다**: 피지컬 100은 "이 방송에서 이기면 끝". Zone X는 "이번 시즌 랭킹이 내 커리어에 누적된다". 참가자에게 쇼 바깥의 의미가 있다.

2. **서사가 누적된다**: 피지컬 100 참가자는 시즌 1 끝나면 시즌 2 출연 보장이 없다. Zone X 선수는 3연승 행진, 챔피언 도전, 연패 후 복귀 — 선수에게 스토리가 쌓인다.

3. **시청자 투자가 깊어진다**: 한번 보고 끝이 아니라 "이 선수가 이기면 이 의미가 있다"는 서사적 투자가 생긴다. 팬이 생기고, 예측이 생기고, 논쟁이 생긴다.

4. **커뮤니티가 형성된다**: 리그에는 팬이 있다. 팬은 선수를 응원하고, 매치업을 예측하고, 결과를 논쟁한다. 쇼에는 시청자가 있고, 리그에는 팬이 있다.

5. **참가자의 정체성이 달라진다**: 피지컬 100 출연자 = "피지컬 100 출연한 사람". Zone X 선수 = "Zone X 챔피언", "Zone X #3 랭커". 선수들이 스스로를 "Zone X 선수"로 정체화할 때 진짜 리그가 된다.

> **핵심**: UFC가 격투기에 한 것을 Zone X가 피트니스에 한다.
> Physical 100 우승자는 다음 시즌이 없다. Zone X 챔피언은 방어전이 있다.
> 이것이 "쇼"와 "리그"의 근본적인 차이이고, Zone X의 존재 이유다.

**현재 시장이 리그를 필요로 하는 이유**:
- 산재된 대회들 (크로스핏 게임즈, 하이록스, 스파르탄...)은 서로 기준이 다르다
- 방송 프로그램 (피지컬 100)은 1회성이다
- **"누가 더 운동을 잘하는지" 겨룰 수 있는 영구적인 장이 없다**
- Zone X는 그 "장"을 만드는 것이다

## Part 2: 게임 포맷 설계 (260318 CYA 진화 아이디어 추가)

### 2.1 경기의 철학과 정체성

카디오는 모든 운동의 공통 엔진이다. Zone X는 그 "공통 전장"에서 다른 배경의 선수들이 붙는 리그.

**크로스핏 게임즈/하이록스와의 차이**: 그들은 "레이스" — 더 빠른 사람이 이긴다. Zone X는 "배틀" — 상대의 약점을 찌르고, 에너지 배분 전략으로 승부를 뒤집는다.

**핵심 긴장 구조 (MMA 비유)**:

| MMA | Zone X |
|---|---|
| 타격가 vs 그래플러 → 상성 | 심폐지구력형 vs 근지구력형 → 상성 |
| 초반 KO 노리다 체력 소진 | 초반 전력 투구로 리드하다 후반 역전 당함 |
| 체급으로 경쟁 구분 | 1인/2인/4인전으로 경쟁 구분 |
| 스탠드업 vs 그라운드 선택 | Phase별 동작 유형 선택 (Choose Your Attack) |

### 2.2 경기 구조: Phase 시스템 + Choose Your Attack

**기본 흐름**: Phase 1 → Phase 2 → Phase 3 (에스컬레이팅 생존 구조)

```
Phase 1 — 고정 시그니처 (오프닝 벨)
 └→ 마일스톤 선두 → "Choose Your Attack" 선택권 획득
Phase 2 — 선택된 동작 수행
 └→ 마일스톤 선두 → 선택권 획득
Phase 3 — 선택된 동작 수행
 └→ 먼저 기권/실패하면 패배. 양쪽 완주 시 총 시간 판정.
```

**Phase 1 — 시그니처 시작 (항상 동일)**
- 예: 어썰트 바이크 15칼 + 버피 10개
- Zone X의 "오프닝 벨". 하이록스 1km 런처럼 시그니처.

**마일스톤 선두 → Choose Your Attack**
Phase를 먼저 끝낸 선수가 다음 Phase의 동작을 2~3개 선택지 중에서 고른다.

| 선택지 | 동작 예시 | 성격 |
|---|---|---|
| "상체 공격" | 로프클라임 + 월워크 | 러너/사이클러에게 치명적 |
| "심폐 공격" | 더블언더 + 바이크 추가 | 근력형에게 부담 |
| "하체 공격" | 박스점프 + 러닝 | 상체형에게 상대적 유리 |

**핵심 메커니즘**:
1. 내가 빠를수록 상대가 더 힘들어진다 (눈에 보임)
2. 하지만 초반에 너무 달리면 내 체력이 먼저 바닥남 (페이싱 트레이드오프)
3. 상대가 마일스톤을 찍고 내 약점을 공격하는 순간이 "피격"처럼 느껴짐
4. Phase마다 주도권이 뒤바뀔 수 있어 엎치락뒤치락 구조 자동 생성

**이것이 WOD가 아니라 Combat인 이유**:

| 크로스핏 Open / 하이록스 | Zone X |
|---|---|
| 상대: 타이머/리더보드 (간접) | 상대: 바로 옆에서 같이 죽고 있는 사람 (직접) |
| 페이싱 기준: 내 최적 전략 | 페이싱 기준: 상대 상태를 보고 전략을 바꿈 |
| 긴장의 원천: 시간과의 싸움 | 긴장의 원천: 상대와의 싸움 |
| 실패: 시간 내 미완료 | 실패: 상대보다 먼저 무너짐 |

**정당성 유지**:
- 선택지의 모든 동작은 합법적인 피트니스 테스트
- 어떤 동작이 걸리든 결국 더 강한 사람이 이김 — 선택은 약점을 찌를 뿐 실력을 대체하지 않음
- 상성이 있어도 충분히 강하면 불리한 동작에서도 이길 수 있음

**직관성 유지**:
- 선택지 2~3개, 직관적 이름 ("상체 공격", "심폐 공격", "하체 공격")
- 화면에 아이콘/텍스트로 "A선수 선택: 상체 공격!" 표시
- 중계에서 "왜 이걸 골랐는가"를 1문장으로 설명 가능

### 2.3 Choose Your Attack — Alternatives

**Alt 1: Pre-Match Draft (사전 선택형)**
경기 전에 두 선수가 번갈아가며 Phase 동작을 드래프트. LoL 밴픽 방식.
- 장점: 경기 전 전략 콘텐츠 생성, 경기 자체는 순수 체력 승부, 정당성 최강
- 단점: 경기 중 역동성 약함, Combat 질감 감소
- 적합: 정당성 최우선, 콘텐츠 밀도는 경기 외부에서 만들 때

**Alt 2: Blind Pick (동시 선택형)**
Phase 전환 시 두 선수가 동시에 다음 동작 선택, 동시 공개.
- 장점: 극적인 공개 순간, 읽기 싸움 발생
- 단점: 서로 다른 동작 시 비교 공정성 문제, 룰 복잡
- 적합: 전략 게임성 극대화

**Alt 3: Escalating Stakes (리스크 베팅형)**
리드 선수에게 "기본 난도 vs 강화 난도" 베팅권. 강화 구간 승리 시 2배 포인트.
- 장점: 도박적 긴장감, 피니시 노리다 역전 당하는 드라마
- 단점: 포인트 시스템 필요, 생존형의 단순함 사라짐
- 적합: 타이틀전/빅매치에서 부분 도입

**Alt 4: Rotating Advantage (교대 선택형)**
마일스톤과 무관하게 선택권이 자동 교대.
- 장점: 가장 공정, 룰 단순
- 단점: 마일스톤 경쟁 의미 약화
- 적합: 첫 시즌 단순 시작

**비교 요약**:

| | Choose Your Attack (원안) | Alt 1 Draft | Alt 2 Blind | Alt 3 Stakes | Alt 4 Rotate |
|---|---|---|---|---|---|
| Combat 느낌 | 강 | 중 | 강 | 강 | 중 |
| 정당성 | 강 | 최강 | 중 | 중 | 강 |
| 직관성 | 강 | 강 | 약 | 중 | 최강 |
| 역전 가능성 | 강 | 중 | 중 | 강 | 중 |
| 룰 복잡도 | 중 | 낮 | 높 | 높 | 최저 |

**추천**: Choose Your Attack을 기본으로, Alt 3를 챔피언십/타이틀전 빅매치에서 부분 도입.

### 2.4 CYA 진화 아이디어 — "진짜 스포츠가 되기 위한 전략적 심도" (260318 신규)

현재 CYA는 가안(ver.1)이다. 리그가 시즌을 거듭하면서 "관전자가 전략을 읽는 재미"가 깊어져야 진짜 스포츠가 된다. 아래 3가지 진화 아이디어를 탐색한다.

#### Sub-idea 1: 종목/동작 단위 세부 선택

현재 CYA는 "심폐 공격 / 상체 공격 / 하체 공격" 3가지 대분류에서 선택한다. 이걸 더 세분화하면?

```
현재:  심폐 공격 → (풀에서 2동작 랜덤 조합)
진화:  심폐 공격 → 구체적 동작 선택
       예: "더블언더 75 + 바이크 20cal" vs "버피 20 + 로잉 500m" 중 택 1
```

**효과**:
- 선수가 상대의 특정 동작 약점을 정밀 공격할 수 있음
- "저 선수는 더블언더를 못 한다" → 더블언더가 포함된 조합을 선택 = 정밀 저격
- 시청자가 "왜 이 조합을 골랐는지" 분석하는 재미가 생김

**리스크**: 룰 복잡도 상승. Season 1에서는 대분류만, Season 2+에서 세부 선택 도입 검토.

#### Sub-idea 2: 공격 유형 × 난도 조합 (리스크-리워드)

공격 유형 선택에 "난도 베팅"을 결합한다.

```
선택: 심폐 공격 — 기본 난도 (기본 처방)
      심폐 공격 — 강화 난도 (처방 1.5배, 단 상대가 받아들여야 함)
      심폐 공격 — 극한 난도 (처방 2배, 양쪽 모두에게 적용)
```

**제안/수락 메커니즘** (마인드게임):
1. A선수: "심폐 공격 — 강화 난도" 제안
2. B선수: 수락 or 거부 (거부 시 기본 난도로 실행)
3. 수락하면: 양쪽 모두 강화 난도 수행 → 자신감 대결
4. 거부하면: 거부 자체가 "약하다"는 인상 → 심리전

**콘텐츠 효과**:
- "강화 난도를 제안했다! 상대가 받아들일까?" — 극적인 순간
- 공격 선택 + 난도 베팅 → 2차원 전략 (현재 1차원)
- 시즌이 깊어질수록 "누가 강화를 잘 거는가" 메타 형성

#### Sub-idea 3: 교대 선택 + 거부권

Phase별로 선택권을 교대하되, **시즌당 1회 "거부권"** 사용 가능.

```
Phase 2: A선수가 공격 유형 선택
Phase 3: B선수가 공격 유형 선택
거부권: 시즌 중 1회, 상대의 공격 선택을 거부하고 다시 선택하게 할 수 있음
```

**거부권의 전략적 깊이**:
- 언제 거부권을 쓰는가 = 메타게임의 핵심
- "시즌 초반에 쓰면 그 경기를 잡지만, 나중에 타이틀전에서 쓸 수 없다"
- 시청자: "거부권을 아직 안 썼다... 언제 쓰지?" → 시즌 내내 긴장감
- 상대가 거부권 보유 중이면 → 공격 선택 시 "거부당할 가능성"까지 계산해야 함

#### "종목 번갈아 고르기 머리싸움" — 전략의 본질

위 3가지 Sub-idea의 공통 핵심:

> **CYA가 단순한 "약점 찌르기"를 넘어서, 상대의 사고방식을 읽는 머리싸움이 되어야 한다.**

**체스 비유**:
- 현재 CYA = 3수 앞을 보는 전략 (Phase 2 선택이 Phase 3에 영향)
- 진화한 CYA = 시즌 전체를 보는 전략 (이번 경기의 선택이 다음 경기 상대에게 정보를 준다)

**"진짜 스포츠가 되기 위한 조건"**:
1. **반복 가능한 포맷**: 같은 룰에서 무한히 다른 경기가 나와야 함 ✓ (CYA 기본)
2. **학습 곡선**: 시청자가 볼수록 더 깊게 읽을 수 있어야 함 → Sub-idea 1-3이 해결
3. **메타게임**: "이번 시즌의 유행 전략"이 생겨야 함 → 시즌 내 패턴 형성 + 파괴
4. **관전 문화**: "왜 이 선택을 했는지" 분석하는 커뮤니티가 생겨야 함 → 해설 콘텐츠의 재료

**도입 로드맵**:

| 시즌 | CYA 버전 | 전략적 심도 |
|---|---|---|
| Season 1 | 기본 CYA (3대분류 선택 + 마일스톤 기반) | 약점 찌르기 (1차원) |
| Season 2 | + 세부 동작 선택 (Sub-idea 1) | 정밀 저격 (2차원) |
| Season 3+ | + 난도 베팅 or 거부권 (Sub-idea 2/3) | 머리싸움 + 메타게임 (3차원) |

## Part 3: 상성 시스템 상세 설계

### 3.1 설계 원칙

Choose Your Attack의 3가지 공격 유형 각각에 **동작 풀(Exercise Pool)**을 정의한다. 공격을 선택하면, 해당 풀에서 2개 동작이 Phase 워크아웃으로 구성된다.

**원칙**:
1. **직관적 구분**: 시청자가 "심폐 공격 = 오래 달려야 한다", "상체 공격 = 무거운 걸 들어야 한다"를 바로 이해
2. **공정한 난도**: 3가지 공격의 전체 부담(소요시간, 에너지 소모)이 대략 동등
3. **변동성**: 매 매치마다 풀 안에서 동작이 달라져 시청자 예측 불가
4. **커뮤니티 인지도**: 크로스핏/하이록스/피트니스 커뮤니티에서 알려진 동작 우선
5. **시각적 임팩트**: 화면에서 힘들어하는 것이 "보여야" 함

### 3.2 심폐 공격 (Cardio Attack) — ENGINE + GRIT 영역

> "숨이 턱까지 찬다. 멈추면 진다."

**핵심**: 장시간 심폐 출력 + 반복 피로 누적. 근력형/폭발력형에게 치명적.

| # | 동작 | 기본 처방 (스케일링 가능) | 성격 | 누구에게 치명적 |
|---|---|---|---|---|
| C1 | **더블언더 (Double-Under)** | 50 / 75 / 100회 | 스킬 + 심폐 | Tank (줄넘기 경험 부족) |
| C2 | **어썰트 바이크** | 15 / 20 / 25 cal | 순수 심폐 출력 | Tank, Specialist |
| C3 | **로잉 (Rowing)** | 250m / 500m | 전신 심폐 | Tank (기술 + 지구력) |
| C4 | **버피 (Burpee)** | 15 / 20 / 25회 | 전신 심폐 킬러 | 모든 유형에 부담 |
| C5 | **월볼 (Wall Ball)** | 20 / 30 / 40회 | 심폐 + 하체 지구력 | Tank, Specialist |
| C6 | **쓰러스터 (Thruster)** | 10 / 15 / 20회 (체중 비율) | 심폐 + 전신 복합 | 순수 근력형 |

**Phase 조합 예시**:
- 더블언더 75회 + 어썰트 바이크 20cal → 스킬 + 순수 심폐
- 버피 20회 + 월볼 30회 → 전신 심폐 킬러
- 로잉 500m + 쓰러스터 15회 → 지구력 + 전신 복합

### 3.3 상체 공격 (Upper Body Attack) — FORCE + CONTROL 영역

> "무거운 것을 들고, 높은 곳을 오르고, 뒤집힌 채로 버틴다."

**핵심**: 상체 절대 근력 + 움직임의 정교함. 심폐형/러너에게 치명적.

| # | 동작 | 기본 처방 | 성격 | 누구에게 치명적 |
|---|---|---|---|---|
| U1 | **로프 클라임** | 1 / 2 / 3회 | 상체 당기기 — 시각적 최강 | Engine, Explosive |
| U2 | **월 워크 (Wall Walk)** | 3 / 5 / 7회 | 어깨 지구력 + 코어 | Engine, Explosive |
| U3 | **핸드스탠드 푸시업 (HSPU)** | 5 / 8 / 12회 | 오버헤드 밀기 근력 | Engine |
| U4 | **링 머슬업** | 1 / 3 / 5회 | 당기기→밀기 전환 (스킬+근력) | Engine, Explosive |
| U5 | **바벨 오버헤드 프레스** | 설정 무게 × Max Rep | 순수 상체 밀기 근력 | Engine |
| U6 | **스트릭트 풀업** | 10 / 15 / 20회 | 상체 당기기 기본 | 일반적 테스트 |

**Phase 조합 예시**:
- 로프 클라임 2회 + 월 워크 5회 → 당기기 + 밀기 복합
- 링 머슬업 3회 + 핸드스탠드 푸시업 8회 → 고난도 체조 동작
- 바벨 프레스 × Max Rep + 스트릭트 풀업 15회 → 순수 상체 근력

### 3.4 하체 공격 (Lower Body Attack) — BLAST + REFLEX 영역

> "뛰고, 점프하고, 버티고. 다리가 먼저 죽으면 끝이다."

**핵심**: 폭발적 하체 파워 + 방향 전환·스피드. 상체 특화형에게 치명적.

| # | 동작 | 기본 처방 | 성격 | 누구에게 치명적 |
|---|---|---|---|---|
| L1 | **박스점프 오버** | 10 / 15 / 20회 | 폭발적 점프 | Tank (둔함) |
| L2 | **러닝** | 200m / 400m | 스피드 + 하체 지구력 | Tank |
| L3 | **프론트 스쿼트** | 설정 무게 × Max Rep | 하체 근력 + 코어 | Engine (절대 근력 부족) |
| L4 | **워킹 런지** | 20 / 30 / 40m (바벨/덤벨) | 하체 지구력 + 밸런스 | 모두에게 부담 |
| L5 | **슬레드 푸시** | 설정 무게 × 25m | 하체 raw power | Engine |
| L6 | **피스톨 스쿼트** | 좌우 3 / 5 / 7회 | 밸런스 + 하체 근력 | 균형 훈련 안 한 선수 |

**Phase 조합 예시**:
- 박스점프 오버 15회 + 러닝 400m → 폭발력 + 스피드
- 프론트 스쿼트 × Max Rep + 워킹 런지 30m → 하체 근력 + 지구력
- 슬레드 푸시 25m + 박스점프 오버 10회 → raw power + 폭발력

### 3.5 Phase별 동작 조합 규칙

**경기 기본 흐름**:

```
Phase 1: 시그니처 (고정) — 어썰트 바이크 15cal + 버피 10회
  └→ 마일스톤 선두 = Choose Your Attack 선택권 획득
Phase 2: 선택된 공격 유형의 풀에서 2동작 조합
  └→ 마일스톤 선두 = Choose Your Attack 선택권 획득
Phase 3: 선택된 공격 유형의 풀에서 2동작 조합
  └→ 먼저 기권/실패 = 패배. 양쪽 완주 시 총 시간 판정.
```

**조합 규칙**:

| 규칙 | 내용 | 이유 |
|---|---|---|
| 풀 내 2동작 | 한 Phase에 같은 풀에서 2동작 선정 | 공격 유형의 정체성 유지 |
| 같은 계열 중복 불가 | 로프 클라임 + 풀업 (둘 다 당기기) = 불가 | 동작의 다양성 보장 |
| Phase 간 동작 반복 불가 | Phase 2에 버피가 나왔으면 Phase 3에는 다른 동작 | 시청자 피로 방지 |
| 같은 유형 연속 선택 가능 | Phase 2 심폐 → Phase 3 심폐 가능 (동작은 다름) | 약점 집중 공략 전략 허용 |
| 동작 리볼브 | 같은 공격 유형이라도 매 이벤트마다 조합이 달라짐 | 신선함 유지, 시청자 예측 불가 |

**난도 에스컬레이팅**:
- 각 동작에 3단계 처방 (기본 / 중간 / 고강도)
- Phase 3은 Phase 2보다 한 단계 높은 처방 적용
- 타이틀전은 전체적으로 고강도 적용

### 3.6 아키타입별 상성 매트릭스

컴바인 Hexagon 스탯 기반 아키타입이 3가지 공격 유형과 어떻게 상호작용하는가:

```
             │  심폐 공격         상체 공격         하체 공격
─────────────┼──────────────────────────────────────────────────
 Engine      │  ✓✓✓ 홈그라운드    ✗✗✗ 치명적       ✗✗ 불리
 Tank        │  ✗✗✗ 치명적       ✓✓✓ 홈그라운드    ✗✗ 불리
 Explosive   │  ✗✗ 불리          ✗ 약간불리        ✓✓✓ 홈그라운드
 All-Rounder │  ✓ 보통           ✓ 보통            ✓ 보통
 Specialist  │  특화 영역에서 무적, 나머지에서 위험 (극단적)
```

**핵심 매치업 시나리오**:

| 매치업 | 공격 선택 전략 | 시청자에게 보이는 드라마 |
|---|---|---|
| **Engine vs Tank** | Engine → 심폐 공격 (상대 약점). Tank → 상체 공격 (상대 약점). **Phase 1 승자가 경기를 지배.** | "Phase 1에서 누가 이기느냐가 모든 것을 결정한다" |
| **Engine vs Explosive** | Engine → 심폐 (지구전). Explosive → 하체 (속전속결). Phase 3까지 가면 체력 소모로 Engine 유리. | "시간이 Engine의 편이다. Explosive는 빨리 끝내야 한다" |
| **Tank vs Explosive** | Tank → 상체. Explosive → 하체. 둘 다 심폐에 약해서 Phase 1이 난장판. | "둘 다 심폐가 약해서 Phase 1부터 카오스" |
| **All-Rounder vs Specialist** | All-Rounder → Specialist의 약점 공격. Specialist → 자기 강점 공격. Specialist가 압도적 차이를 만들어야 All-Rounder의 균형을 이김. | "All-Rounder는 져도 근소하게 진다. Specialist는 이기면 압도적" |
| **Engine vs Engine** | 둘 다 심폐 강자. 상체/하체 공격을 걸어야 차이가 나는데 둘 다 약함. Phase 1 초장기전. | "누가 먼저 지치느냐. GRIT의 차이가 여기서 드러난다" |
| **Tank vs Tank** | 둘 다 근력 강자. 심폐 공격 걸면 둘 다 힘들지만, 거기서 누가 조금 더 버티느냐. | "힘은 비슷하다. 결국 체력에서 갈린다" |

### 3.7 전략적 심도 — 왜 CYA가 작동하는가

**레이어 1: 약점 찌르기 (가장 직관적)**
- "상대의 ENGINE이 45다. 심폐 공격을 건다." → 시청자 누구나 이해

**레이어 2: 자기 강점 vs 상대 약점의 딜레마**
- 내가 Tank형이고 상대도 FORCE가 높다면 → 상체 공격은 상대에게도 나쁘지 않음
- 차라리 심폐 공격을 걸어서 "둘 다 힘든 영역에서 내가 조금이라도 낫다"는 베팅 가능
- → 단순한 가위바위보가 아니라 **읽기 싸움**

**레이어 3: 경기 내 전환 (CYA의 핵심 매력)**
- Phase 2에서 심폐 공격을 받은 Tank가 간신히 버텨냄 → Phase 3에서 본인이 상체 공격 선택
- "Phase 2를 버텼으니 Phase 3에서 역전할 수 있다" → 역전의 구조적 보장
- 시청자는 "끝났다"고 생각했는데 Phase 3에서 뒤집히는 순간 = 가장 강력한 콘텐츠

**레이어 4: 메타게임 (시즌이 깊어질수록)**
- "저 선수는 항상 심폐 공격을 고른다" → 상대가 대비
- 패턴을 읽고 허를 찌르는 선수 등장
- 해설이 "이번에는 뭘 고를까" 예측 → 시청자 참여

**레이어 5: Phase 1의 페이싱 딜레마**
- Phase 1을 빠르게 끝내야 CYA 선택권을 얻지만
- 너무 빨리 가면 Phase 2-3에서 체력 소진
- "일부러 Phase 1을 천천히 가서 체력을 아끼는 전략"도 유효 (선택권을 포기하되 체력을 남김)
- 시청자: "왜 느리게 가지?" → 해설: "체력을 아끼는 겁니다. Phase 3에서 승부를 볼 생각이에요"

## Part 4: 매치 구분 기준 & 매치메이킹

### 4.1 Division 설계

Season 1은 단순하게 시작. 참가자 수가 늘어난 후(Season 2+) 세분화.

**Season 1 기본 구분**:

| 구분 | 기준 | 비고 |
|---|---|---|
| 성별 | 남/녀 분리 | 공정성 필수 |
| 인원 | 1:1 | Season 1 기본 포맷 |
| 체급 | 없음 (Open Division) | 컴바인 인원 10-16명으로 세분화 불필요 |
| 랭킹 | 컴바인 종합 순위 기반 초기 시드 | 이후 전적으로 업데이트 |

> **왜 체급을 두지 않는가**: Zone X의 핵심 메시지는 "모든 종목의 운동인이 같은 판에서 붙는다". 체급으로 나누면 이 메시지가 약해진다. Season 1에서는 체중 차이보다 능력 차이가 더 중요한 게임 설계를 통해 공정성을 확보한다 — CYA 선택으로 약점을 공략하면 체중 차이를 뛰어넘을 수 있다.

**Season 2+ 확장 예정**:
- 체급 분리: 75kg 이하 / 75-90kg / 90kg 이상 (3체급)
- 팀전 도입: 2v2 (Duo), 4v4 (Squad)
- 티어 분리: Open / Challenger / Elite (→ Part 7 상세)
- 더 많은 참가자 → 더 세분화된 디비전

### 4.2 매치메이킹 철학

> "토너먼트가 아니다. UFC처럼 매치메이커(수현)가 가장 재미있는 대결을 만든다."

**매치메이킹 3원칙**:

| 원칙 | 설명 | 예시 |
|---|---|---|
| **상성 기반** | Hexagon 스탯이 극명히 대비되는 매치업 우선 | ENGINE 92 vs FORCE 95 = 전형적 Engine vs Tank |
| **서사 기반** | 빌드업/컴바인에서 쌓인 라이벌리·떡밥 활용 | "컴바인에서 신경전 있었던 둘이 드디어 붙는다" |
| **경쟁력 기반** | 일방적 학살이 아니라 승부 예측이 어려운 매치 | 종합 순위가 비슷하지만 유형이 다른 매치 |

**매치 카드 구성 (이벤트당)**:

```
메인 이벤트 (1경기):   가장 기대되는 매치업 — 랭킹 상위 or 타이틀전
코메인 (1-2경기):     주목도 높은 서사 매치 or 랭킹전
메인카드 (3-4경기):    컴바인 상위 시드 선수 중심
언더카드 (3-4경기):    컴바인 하위 시드, 신규 참가, 데뷔전
```

### 4.3 랭킹 시스템

**초기 랭킹** = 컴바인 종합 순위

**이후 업데이트** = 전적 기반:

| 랭킹 변동 요소 | 가중치 | 설명 |
|---|---|---|
| 승리 | 기본 상승 | 모든 승리는 랭킹을 올림 |
| 상위 랭킹 상대 격파 | 추가 보너스 | 강자를 이기면 크게 오름 |
| 연승/연패 | 가속/감속 | 3연승 이상이면 빠르게 상승 |
| 피니시 방식 | 작은 보너스 | KO(기권 유도) > 판정(시간차 승) |
| 비활동 | 소폭 하락 | 2이벤트 이상 불참 시 |

**랭킹 표기**: #1 ~ #(총 참가자 수). 성별 분리 랭킹.

### 4.4 인원 클래스 확장 (Season 2+)

| 클래스 | 인원 | Choose Your Attack 변형 | 핵심 차별점 |
|---|---|---|---|
| **Solo** | 1:1 | 기본 CYA 룰 | 순수 개인 능력 |
| **Duo** | 2:2 | 2명이 합의해서 공격 선택 | 팀 전략 + 역할 분배 |
| **Squad** | 4:4 | 릴레이 — 각 Phase에 다른 선수 투입 가능 | 선수 투입 순서 전략 |

**Duo 전략적 깊이**:
- Engine+Tank 페어 → 모든 공격에 한 명은 강함 = 밸런스형
- Engine+Engine 페어 → 심폐 압도적이지만 상체 공격에 둘 다 취약 = 도박형
- 팀 구성 자체가 전략 = 드래프트/팀 빌딩 콘텐츠 가능

**Squad 전략적 깊이**:
- Phase 1에 Engine형, Phase 2에 Tank형, Phase 3에 All-Rounder를 투입하는 릴레이
- "누가 어디에 들어가느냐"가 감독의 전략 = 축구 감독의 교체 카드 같은 재미

## Part 6: 콘텐츠 설계 — 경기 외부 (260318 MMA형 서사 빌드업 상세화)

### 6.1 콘텐츠 패키지 (이벤트당)

| 콘텐츠 | 시점 | 형태 | 목적 |
|---|---|---|---|
| **매치업 프리뷰** | 이벤트 1주 전 | 숏폼 60초 + 롱폼 5분 | Hexagon 비교, 상성 분석, 시청자 예측 유도 |

| **하이라이트 클립** | 본 매치 공개 직후 | 숏폼 15-30초 | CYA 선택, 역전 장면, KO 순간 → 바이럴 |

| **다음 이벤트 티저** | 리캡 후 2-3일 | 숏폼 30초 | 다음 매치 카드 힌트/공개 |

### 6.2 본 매치 영상 구조 (1경기당)

```
[30초]  매치업 인트로 — Hexagon 비교 + 상성 해설
[2분]   Phase 1 시그니처 — 누가 먼저 끝내는가
[10초]  ★ CYA 선택 순간 — "심폐 공격을 선택했습니다!"
[2-3분] Phase 2 수행
[10초]  ★ CYA 선택 (Phase 2 승자 or Phase 1 승자 연속)
[2-3분] Phase 3 수행
[30초]  결과 + 리액션 + 승자 인터뷰
```

**총 8-12분 per match.** 이벤트당 8-12경기 = 약 80-120분 분량.

### 6.3 핵심 콘텐츠 자산 — 반복 재활용 가능한 것들

**① Hexagon 스탯 카드** (그래픽 템플릿)
- 매 매치 프리뷰에 사용
- 소셜 미디어 공유 최적화 (1:1 정사각형)
- 시청자가 직접 캡처해서 공유하게 만드는 디자인

**② CYA 선택 순간 클립** (15초 숏폼 — 최고 바이럴 자산)
- "심폐 공격을 선택했습니다!" + 상대방 표정 + 해설 리액션
- 매 경기에서 최소 2개 생산 (Phase 2, Phase 3 선택)
- 릴스/쇼츠 최적화 → 알고리즘 타기 좋은 포맷

**③ 역전 장면 클립**
- Phase 전환 시 역전되는 순간 — "끝난 줄 알았는데!"
- 전후 컨텍스트 포함 30초 클립

**④ KO(기권) 모먼트**
- 상대가 기권하는 순간의 감정 = 콘텐츠의 정점
- 기권자의 리액션 + 승자의 리액션 = 드라마

**⑤ 랭킹 변동 카드** (그래픽 템플릿)
- 이벤트 후 순위 상승/하락 시각화
- 연승 행진, 업셋(하극상) 강조

## Part 7: 리그 티어 시스템 아이데이션 (260318 신규)

### 7.1 왜 티어가 필요한가

**현재 구조의 한계 (Season 1)**:
- 컴바인 통과 10~16명이 전부 같은 리그에서 경쟁
- 실력 차이가 크면 일방적 경기 발생 → 콘텐츠 품질 하락
- 새로운 참가자가 들어올 통로가 없음 (시즌 끝나야 다음 컴바인)
- 리그가 커지면 참가자 관리가 비효율적

**티어 시스템으로 해결하는 것**:
- 실력 수준별 경쟁 → 매치 퀄리티 보장
- 진입 통로 상시 개방 → "누구나 도전할 수 있는 리그"
- 승격/강등 서사 → 콘텐츠 자산 (강등의 위기, 승격의 감동)
- 리그의 규모 확장을 구조적으로 지원

### 7.2 3티어 구조 — e스포츠 랭킹전 레퍼런스

| 티어 | 성격 | 인원 규모 (장기 비전) | 이벤트 빈도 |
|---|---|---|---|
| **Elite** | 최상위 리그 — 챔피언 벨트 경쟁 | 8~12명 | 격주 이벤트 (메인 콘텐츠) |
| **Challenger** | 중간 리그 — Elite 승격을 위한 경쟁 | 12~20명 | 월 1-2회 |
| **Open** | 진입 리그 — 누구나 도전 가능 | 제한 없음 | 컴바인/오디션 형태 |

**e스포츠 레퍼런스**:
- 롤(LoL): 아이언 → 브론즈 → ... → 다이아 → 마스터 → 그랜드마스터 → 챌린저
- 발로란트: Iron → Bronze → ... → Diamond → Ascendant → Immortal → Radiant
- Zone X에 적용: Open → Challenger → Elite (3단계로 단순화)

### 7.3 티어별 상세 설계

#### Elite Tier — "Zone X의 얼굴"

| 항목 | 내용 |
|---|---|
| **진입 조건** | Challenger 시즌 1~2위 자동 승격 / 챔피언 초대 |
| **이벤트** | 격주 1회 — 이벤트당 6-10매치 |
| **챔피언십** | 시즌 파이널에서 챔피언 결정 + 벨트 |
| **강등** | 시즌 최하위 2명 → Challenger로 강등 |
| **콘텐츠 비중** | 메인 콘텐츠 — 프리뷰, 분석, 서사 빌드업 집중 |

#### Challenger Tier — "다음 레벨로 가는 길"

| 항목 | 내용 |
|---|---|
| **진입 조건** | Open 컴바인 상위 / Elite 강등 선수 |
| **이벤트** | 월 1-2회 — 이벤트당 8-12매치 |
| **승격** | 시즌 1~2위 → Elite 자동 승격 |
| **강등** | 시즌 최하위 → Open으로 이동 (다음 컴바인 재도전) |
| **콘텐츠 비중** | 서브 콘텐츠 — 승격 경쟁 중심 서사 |

#### Open Tier — "모두가 시작하는 곳"

| 항목 | 내용 |
|---|---|
| **진입 조건** | 공개 모집 / 직접 섭외 → 컴바인 참가 |
| **형태** | 컴바인(시즌당 1-2회) |
| **승격** | 컴바인 상위 성적 → Challenger 배정 |
| **콘텐츠 비중** | 발굴/서프라이즈 서사 ("어디서 이런 사람이?") |

### 7.4 체급 × 티어 매트릭스 (장기 비전)

리그가 충분히 커지면 체급과 티어를 교차해서 운영:

```
              경량급 (~70kg)   중량급 (70~85kg)   중량급+ (85kg~)
─────────────┼────────────────┼──────────────────┼───────────────
 Elite       │  EL-경량 (6~8) │  EL-중량 (6~8)   │  EL-중량+ (6~8)
 Challenger  │  CH-경량 (8~12)│  CH-중량 (8~12)  │  CH-중량+ (8~12)
 Open        │  ── 전체 혼합 (컴바인에서 체급 분리) ──
```

> Season 1에서는 전체가 Open Division (체급 구분 없음, 티어 구분 없음).
> Season 2에서 티어 도입 (체급 구분은 아직 없음).
> Season 3+에서 인원이 충분해지면 체급 × 티어 매트릭스 적용.

### 7.5 승격/강등 메커니즘

**자동 승격/강등**:
- 시즌 종료 시 각 티어의 상위 2명 자동 승격
- 각 티어의 하위 2명 자동 강등

**교차전 (Promotion Battle)**:
- 시즌 마지막 이벤트에서 "승격전" 매치: Challenger #1 vs Elite 하위 랭커
- 직접 대결로 승격/잔류 결정 → 극적 콘텐츠

**특별 초대**:
- 화제성 있는 외부 인물(유명 운동인, SNS 인플루언서 등) → Elite 직접 초대 가능
- 단, 컴바인에서 Hexagon 스탯은 반드시 측정 (정당성 유지)

### 7.6 티어 시스템이 만드는 콘텐츠 서사

| 서사 유형 | 예시 | 감정 |
|---|---|---|
| **승격 서사** | "Open에서 시작해서 Elite까지 올라간 선수" | 성장의 감동 |
| **강등 서사** | "Elite에서 밀려난 선수의 복수전" | 긴장 + 공감 |
| **교차전** | "Challenger 1위 vs Elite 최하위 — 직접 결정" | 한판 승부의 긴장감 |
| **신성 등장** | "컴바인에서 놀라운 스탯을 기록한 무명" | 서프라이즈 + 기대 |
| **복귀** | "강등 후 Challenger에서 다시 올라오는 여정" | 리질리언스 |

> **핵심**: 티어 시스템은 단순한 관리 도구가 아니라 **서사 생성 엔진**이다. 승격/강등/복귀의 구조가 시즌마다 자동으로 드라마를 만들어준다.
