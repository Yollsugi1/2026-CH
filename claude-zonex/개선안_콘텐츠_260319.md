## 2. 빌드업 콘텐츠 플래닝

### 2.1 에피소드 상세 구조 (260318 컨셉 반영)

8주 × 주 2~3회 = 16~24편. 아래 구조를 제안한다.

**Phase 1 (4월 1~3주, 6~8편): "질문을 던지다"**

목표: 6능력 개념 노출 + 채널 인지도 확보

| EP | 중심 능력 | 인터뷰이 유형 | 에피소드 제목 (가안) | 핵심 포인트 |

| 2 | FORCE | 파워리프터 | "200kg를 드는 사람한테 버피를 시켜봤다" | 근력 이야기 + 버피 1분 측정 → "힘만으로 부족" |

## 3. 컴바인 콘텐츠 플래닝

### 3.1 ACT 1 자존심 배틀 반영한 시리즈 재구조화

기존 11부작에서 260318 ACT 1 재설계를 반영하여 **10부작 + 보너스 1편** 구조로 조정.

```
ACT 1: 집결 + 자존심 배틀 (EP 1-2)      → Day 1 오전 촬영
ACT 2: 6개 라운드 (EP 3-8)              → Day 1 오후 ~ Day 3
ACT 3: 리빌 + 선전포고 (EP 9-10)         → Day 3 마지막
보너스: 비하인드 (EP 11)                  → 전 기간 B-roll 편집
```

**변경 핵심: 11부작 → 10+1부작**
- 기존 EP6(중간 성적표)를 독립 편에서 → EP5 후반부에 통합. 중간 성적표만으로 한 편을 채우면 "쉬어가는 편"이 됨. 측정 장면 없이 인터뷰+순위만으로는 시청자 이탈 위험.
- ACT 2 에피소드가 7편 → 6편으로 줄어 편집 부하 1편 감소.

| 2주차 | EP 3-5 편집 (ROUND 1~3 + 중간 성적표) | 3편 완성 | 에디터 A + 에디터 B |
| 3주차 | EP 6-8 편집 (ROUND 4~6) | 3편 완성 | 에디터 A + 에디터 B |
| 4주차 | EP 9-10 편집 (ACT 3) + EP 11 보너스 | 3편 완성 | 지운PD + 에디터 A |

총 편집 기간: 약 4주. 주 2회 업로드 시 → 5주간 업로드. **컴바인 시리즈 완결까지 약 5주**.

### 3.4 외부 프로덕션 협업 범위 제안

| 역할 | 내부 | 외부(PIP 등) |
|---|---|---|
| 총 연출 | 수현 | - |
| 카메라 A (메인 경기) | - | 외부 1명 |
| 카메라 B (보조 경기 + 관전자) | 지운PD | - |
| 카메라 C (인터뷰 전담) | 세한 or 수민 | - |
| 사운드 | - | 외부 1명 (핀마이크 + 붐) |
| 현장 진행 | 수민 or 세한 | - |
| 그래픽 사전 제작 | 우석/훈택 | - |

**최소 외부 인력: 2명 (카메라 1 + 사운드 1)**
**이상적 외부 인력: 3~4명 (카메라 1 + 사운드 1 + 조명/장비 1 + 보조 1)**

견적 기준 (참고용):
- 카메라맨 일당: 30~50만 원 × 3일 = 90~150만 원
- 사운드 일당: 25~40만 원 × 3일 = 75~120만 원

- 장비 렌탈 + 장소 + 식대 별도

### 4.2 콘텐츠 볼륨 현실화 — 7종 → 4종 추천

기존 7종을 그대로 유지하면 격주 이벤트 기준 2주에 영상만 20~30편. 현 인력(지운PD + 에디터 3명)으로 불가능.

**축소 제안: 핵심 4종 영상 + 보충 2종 이미지/텍스트**

| # | 콘텐츠 | 형태 | 주기 | 비고 |
|---|---|---|---|---|
| 1 | **본 매치 (풀)** | 롱폼 8~10분 | 이벤트당 메인 5매치만 풀 편집 | 나머지 매치는 하이라이트만 |
| 2 | **하이라이트 클립** | 숏폼 60~90초 | 이벤트당 5~8편 | CYA 선택 + 역전 + KO 순간 |
| 3 | **리캡 + 분석** | 롱폼 5~8분 | 이벤트당 1편 | 전체 이벤트 종합. 랭킹 변동 포함 |
| 4 | **매치 프리뷰** | 숏폼 60초 | 메인 3매치만 | Hexagon 비교 + 관전 포인트 |
| (보충) | **랭킹 카드** | 인스타 카드 이미지 | 이벤트 후 | 디자이너가 템플릿으로 제작 |

**삭제/통합 대상**:
- "선수 각오 인터뷰" → 매치 프리뷰 숏폼에 통합 (별도 영상 불필요)
- "다음 이벤트 티저" → 리캡 영상 엔딩에 포함

**결과: 이벤트당 영상 제작 수**
- 풀매치 5편 + 하이라이트 5~8편 + 리캡 1편 + 프리뷰 3편 = **14~17편**

- 2주 제작 주기에 지운PD + 에디터 3명으로 **타이트하지만 가능**

### 5.2 빌드업→컴바인 인물 연결 해결안

**3단계 인물 분류**:

| 분류 | 인원 | 빌드업 처리 | 컴바인 처리 |
|---|---|---|---|
| **A. 확정 참가자** | 8~10명 | Phase 2-3에서 집중 소개 | 정상 등장 |
| **B. 서프라이즈 참가자** | 2명 이하 | 빌드업에서 촬영은 하되 **비공개** | EP 1에서 "누구도 예상 못한 참가자" 리빌. 빌드업 영상은 플래시백으로 삽입 |
| **C. 빌드업만 출연자** | 6~12명 | Phase 1에서 자연스럽게 등장 | 컴바인에 불참. **별도 서사 처리 불필요** — 빌드업은 "다양한 운동인 인터뷰 시리즈"로 독립적 가치가 있음 |

**핵심**: 빌드업을 "컴바인 예선"이 아니라 "운동인 인터뷰 시리즈"로 포지셔닝하면, 컴바인에 안 오는 사람이 있어도 "왜 안 왔지?" 물음이 크지 않다. 시리즈 초반(Phase 1)은 탐색적 인터뷰, 후반(Phase 3)으로 갈수록 컴바인 확정자 중심으로 자연스럽게 전환.

### 5.3 CYA 시청자 이해 장벽 해결안

**빌드업부터 CYA 개념을 심는 3단계 교육 전략**:

| 단계 | 시점 | 방법 | 목표 |
|---|---|---|---|
| **Step 1: 능력 분류** | 빌드업 Phase 1 | "이 사람은 심폐형", "이 사람은 근력형" — 에피소드 자체가 유형을 보여줌 | "운동인에는 유형이 있다" |
| **Step 2: 상성 인식** | 빌드업 Phase 2 | ENGINE vs FORCE 비교 편집 → "심폐형에게 무거운 걸 들게 하면?" | "유형마다 약점이 다르다" |
| **Step 3: 공격 개념** | 컴바인 EP 10-11 | Hexagon 리빌 후 "이 스탯이면 심폐 공격에 약하다" → CYA 직접 설명 | "약점을 찌르는 게 Choose Your Attack" |

**바이럴 클립 전략 수정**:
- 처음 보는 사람용 바이럴 = **시각적으로 강렬한 순간** (데드리프트 맥스, 비프테스트 탈락, 역전 순간)
- 이미 팬인 시청자용 바이럴 = **CYA 선택 순간** (전략적 맥락이 필요)
- 즉, 바이럴 클립은 2종류로 분리 제작

### 5.4 마케팅 "더 많이" vs 콘텐츠 "줄여야" 충돌 해결

QA 충돌점 2.1 — 마케팅은 주 5회+ 업로드, 콘텐츠/이벤트는 축소를 주장.

**하이브리드 해결안**:

| 유형 | 주간 빈도 | 제작 부하 | 효과 |
|---|---|---|---|
| 영상 (풀매치+하이라이트+리캡) | 주 3~4편 | 높음 → 에디터 | 시청 시간 + 알고리즘 |
| 인스타 카드 (랭킹, 매치업, 스탯) | 주 2~3장 | 낮음 → 디자이너 템플릿 | 인게이지먼트 + 공유 |
| 인스타 스토리 (투표, 퀴즈) | 매일 1개 | 최소 → 에디터 or 세한/수민 | 채널 활성도 + 알고리즘 |

영상은 품질 중심으로 주 3~4편. 나머지 채널 활성도는 이미지/스토리로 채운다. 이렇게 하면 "매일 무언가 올라가는" 채널이 되면서도 영상 제작 부하는 관리 가능.
