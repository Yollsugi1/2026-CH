# Zone X 콘텐츠 검증 프레임워크

> 작성일: 2026.03.18
> 작성 관점: 콘텐츠 PD — "이걸 영상으로 어떻게 만들 건데?" + "시청자가 왜 끝까지 볼 건데?"
> 용도: 기획 산출물을 콘텐츠 제작 관점에서 교차 검증하는 체크리스트 및 루브릭

---

## 1. 콘텐츠 품질 기준 — Phase별

### 1.1 빌드업 숏폼 (60~90초, 주 2~3회, 4~5월)

| 평가 항목 | 기준 | 합격선 | 비고 |
|---|---|---|---|
| **후킹 파워 (0~5초)** | 스크롤을 멈추게 하는가? 측정 결과 or 인상적 장면이 첫 프레임에 있는가? | 3초 이내 시각적/언어적 후크 존재 | 릴스/쇼츠 알고리즘상 3초 내 이탈 결정 |
| **인물 매력도** | 이 사람을 더 보고 싶은가? 종목 배경 + 개성 + 말투가 기억에 남는가? | "이 사람 누구야?" 댓글이 나올 수준 | 모든 운동인이 콘텐츠 적합은 아님 — 선별 기준 필요 |
| **6능력 연결** | 6가지 능력 체계(Hexagon)와 자연스럽게 연결되는가? | 에피소드 내 최소 1개 능력 키워드 명시적 노출 | "억지로 교육"이 아니라 대화 흐름 속 자연스러운 노출 |
| **떡밥 설치 밀도** | 컴바인/리그에서 회수할 수 있는 발언이 있는가? | "자신 있는 능력" + "약점" 중 최소 1개 구체적 발언 | "전 다 잘해요" 같은 모호한 답은 떡밥이 안 됨 |
| **시각적 다양성** | 매주 비슷한 화면이 아닌가? (체육관 → 필드 → 러닝코스 → 도장 등) | 연속 3편 동일 장소 유형 회피 | 종목 다양성이 시각적 다양성을 자동 생성해야 함 |
| **측정 장면 효과** | 간단 측정이 "재미"를 만드는가? | 성공/실패 어느 쪽이든 리액션이 살아 있어야 함 | 측정 자체보다 결과에 대한 감정이 핵심 |
| **CTA 효과** | 다음 편 or 컴바인을 기대하게 만드는가? | 마지막 5초에 다음 예고 or 궁금증 유발 장치 | "다음 주 이 사람" or "이 사람이 컴바인에 오면?" |

**등급 기준**:
- S: 5/7 이상 충족 + 후킹 파워 필수 합격 → 즉시 업로드
- A: 4/7 충족 → 편집 보강 후 업로드
- B: 3/7 충족 → 재촬영 또는 편집 대폭 수정
- C: 2/7 이하 → 보류 (다른 소스 우선)

### 1.2 컴바인 롱폼 시리즈 (10~12부작, "The Proving Ground")

| 평가 항목 | 기준 | 합격선 |
|---|---|---|
| **에피소드 자체 완결성** | 이 편만 봐도 재미가 있는가? (시리즈 안 봤어도) | 라운드 자체의 긴장감 + 서프라이즈가 독립적으로 작동 |
| **시리즈 연속성** | 이전 편과의 서사적 연결이 느껴지는가? | 이전 라운드 결과 레퍼런스 or 선수 발언 콜백 최소 1회 |
| **감정 곡선** | 한 편 안에서 감정의 고저가 있는가? | 최소 2개의 감정 전환점 (예: 예상대로→서프라이즈, 탈락→인터뷰 감동) |
| **선수 서사 밀도** | 라운드 사이 인터뷰가 선수를 "사람"으로 보이게 하는가? | 라운드 전(기대/불안) + 라운드 후(리액션) 인터뷰 양쪽 포함 |
| **편집 리듬** | 측정 장면과 인터뷰가 적절히 교차하는가? | 연속 3분 이상 동일 유형 장면 없음 (측정만 3분 or 인터뷰만 3분 금지) |
| **빌드업 콜백** | 빌드업 숏폼에서 한 말/장면을 활용하는가? | 해당 선수의 빌드업 클립 삽입 최소 1회 |
| **클리프행어** | 다음 편을 보고 싶게 끝나는가? | 다음 라운드 티저 or 순위 변동 암시 |
| **음악/사운드** | 라운드별 톤 차이가 음악으로 느껴지는가? | THE FURNACE는 뜨거운 톤, THE CHASE는 긴박한 톤 등 |

**추가 기준 — EP별 특수 체크**:

| EP | 특수 평가 |
|---|---|
| EP 1-2 (집결) | 12~16명 전원 최소 1장면 노출? 첫인상 인터뷰 확보? |
| EP 6 (중간 성적표) | 3개 라운드 데이터의 시각적 정리? 순위 변동의 드라마 연출? |
| EP 10 (Hexagon 리빌) | 스탯 공개 순간의 연출력? 아키타입 네이밍이 자연스럽게 발생? |
| EP 11 (Season 1 공개) | 매치 카드 공개의 엔터테인먼트 가치? 선전포고 장면의 진정성? |

### 1.3 리그 매치 영상 (8~12분/매치, 격주 이벤트)

| 평가 항목 | 기준 | 합격선 |
|---|---|---|
| **매치업 인트로** | 30초 안에 두 선수의 Hexagon 비교 + 핵심 상성이 전달되는가? | "이 매치의 관전 포인트는 ___다"가 명확 |
| **CYA 선택 연출** | "Choose Your Attack" 선택 순간이 극적으로 연출되는가? | 선택 전 표정 + 선택 순간 사운드/그래픽 + 선택 후 리액션 3단계 |
| **Phase 전환 텐션** | 마일스톤 선두 전환 시 시청자가 체감하는가? | 화면상 리드/팔로 표시 + 해설 코멘트 |
| **KO/기권 순간** | 한 쪽이 무너지는 순간의 감정이 전달되는가? | 슬로모션 + 리액션샷 + 승자/패자 양쪽 인터뷰 |
| **전략 해설** | CYA 선택의 이유가 시청자에게 설명되는가? | 해설 또는 자막으로 "왜 이 공격을 골랐는가" 1문장 |
| **결과 → 다음 연결** | 이 매치 결과가 랭킹/다음 매치에 어떤 영향을 미치는지 전달? | 매치 종료 후 랭킹 변동 or 다음 매치업 암시 |
| **시각적 피로도** | 같은 동작 반복이 지루하지 않은가? | 카메라 앵글 변화 (최소 3앵글) + 관전자 컷어웨이 |

## 2. 에피소드 구조 평가 체크리스트

모든 에피소드(빌드업/컴바인/리그)에 공통 적용하는 내러티브 구조 체크리스트.

### 2.1 Hook 효과 (첫 5초)

- [ ] 시각적 후크: 강렬한 이미지 or 동작이 첫 프레임에 존재하는가?
- [ ] 언어적 후크: "이 사람이 ___ 했다" 식의 호기심 유발 멘트가 있는가?
- [ ] 결과 스포일러: 측정 결과나 서프라이즈를 미리 보여줘서 "어떻게 이렇게 됐지?"를 유발하는가?
- [ ] **안티패턴**: 로고 → 장소 소개 → 인물 소개 순서로 시작 (지루함 → 이탈)

### 2.2 내러티브 비트 (기승전결)

| 비트 | 질문 | 체크 |
|---|---|---|
| **설정** | 이 사람/이 상황은 무엇인가? | [ ] |
| **갈등/도전** | 무엇이 걸려 있는가? 뭘 해야 하는가? | [ ] |
| **절정** | 성공/실패, 예상대로/서프라이즈의 순간이 있는가? | [ ] |
| **정리** | 이 에피소드의 한 줄 임팩트가 남는가? | [ ] |

### 2.3 감정 곡선 평가

```
[호기심] → [이해] → [긴장] → [해소/서프라이즈] → [여운/기대]
   ↑          ↑        ↑              ↑                 ↑
  Hook      소개      도전          결과            CTA/예고
```

- [ ] 감정이 **상승 → 하강** 한 번 이상 반복하는가? (플랫하면 지루함)
- [ ] 가장 감정이 높은 순간이 **에피소드 중반~후반**에 위치하는가? (앞에 몰리면 뒷부분 이탈)
- [ ] 마지막 5초에 **다음을 기대하게 하는 감정**이 남아 있는가?

### 2.4 CTA (Call To Action) 효과

- [ ] 다음 에피소드를 보고 싶게 만드는가?
- [ ] 댓글/공유를 유발하는 질문 or 논쟁점이 있는가?
- [ ] "이 사람이 컴바인에 오면?"/"이 매치 누가 이길까?" 같은 커뮤니티 참여 유도가 있는가?

## 3. 내러티브 아크 일관성 기준

### 3.1 3-Phase 시청자 여정 검증

빌드업→컴바인→본리그 across 3 phases에서 시청자 여정이 끊기지 않는지 평가.

```
Phase        시청자 감정     핵심 전달      끊김 위험 체크
─────────────────────────────────────────────────────────────
빌드업       관심 + 호감     "이 사람들은   [ ] 인물이 기억에 남는가?
(숏폼)                       누구인가"     [ ] 6능력 프레임이 자연스럽게 심겼는가?
                                           [ ] "컴바인이 궁금하다"는 감정이 생기는가?
             ────────── 전환점 1: 컴바인 참가자 발표 ──────────
컴바인       이해 + 기대     "누가 진짜     [ ] 빌드업에서 본 사람이 등장하는가?
(롱폼)                       인가"         [ ] 빌드업 발언이 콜백되는가?
                                           [ ] Hexagon 스탯이 "캐릭터"로 느껴지는가?
                                           [ ] 매치업 예측이 가능해지는가?
             ────────── 전환점 2: Season 1 매치 카드 공개 ──────────
본 리그      몰입 + 확인     "누가 이기     [ ] 컴바인 스탯이 매치에서 활용되는가?
(매치)                       는가"         [ ] CYA 선택이 스탯 기반 전략으로 읽히는가?
                                           [ ] 예상대로/서프라이즈 모두 드라마가 되는가?
```

### 3.2 선수별 서사 추적

각 선수마다 3-Phase에 걸친 서사 아크가 존재해야 함:

| 체크 항목 | 기준 |
|---|---|
| **빌드업 → 컴바인 연결** | 빌드업에서 한 발언(강점/약점)이 컴바인 결과와 대비/확인되는가? |
| **컴바인 → 리그 연결** | Hexagon 스탯이 CYA 선택 전략의 근거가 되는가? |
| **아키타입 일관성** | The Engine, The Tank 등의 별명이 실제 경기 행동과 일치하는가? |
| **성장/변화** | 시즌 내에서 선수의 변화(약점 보완, 전략 진화)가 보이는가? |

### 3.3 끊김 리스크 체크리스트

| 리스크 | 상황 | 대응 |
|---|---|---|
| **인물 망각** | 빌드업에서 본 사람인데 컴바인까지 6주 공백 → 시청자가 잊음 | 컴바인 EP1에 빌드업 하이라이트 패키지 삽입 |
| **개념 혼동** | 6능력과 CYA 3공격의 관계를 시청자가 이해 못함 | 컴바인→리그 전환기에 "스탯이 매치에 어떻게 영향을 미치는가" 해설 영상 필요 |
| **톤 단절** | 빌드업(가벼운 릴스) → 컴바인(시리즈물) → 리그(스포츠 중계) 톤 차이가 급격 | 각 Phase 첫 편에 톤 전환을 의도적으로 연출 (브릿지 콘텐츠) |
| **플랫폼 단절** | 유튜브 쇼츠로 보다가 롱폼으로 전환 시 이탈 | 컴바인 시작 전 "시리즈 예고편" 숏폼으로 유도 |

## 4. 프로덕션 실현가능성 루브릭

### 4.1 현재 인력 구조

| 역할 | 인원 | 주요 업무 |
|---|---|---|
| 지운 PD | 1명 | 영상 촬영·편집 총괄 |
| 에디터 | 약 3명 (Zone X 에디터 포함) | 편집 실행 |
| 외부 프로덕션 (PIP 등) | TBD | 컴바인 촬영 보강 |
| 세한·수민 | 2명 | 빌드업 촬영 현장 (인터뷰어+촬영) |
| 우석/훈택 | 1명 | 그래픽·디자인 (Hexagon 카드, 템플릿) |

### 4.2 Phase별 제작 부하 평가

#### 빌드업 (4~5월, 8주)

| 항목 | 분량 | 필요 리소스 | 실현 판정 |
|---|---|---|---|
| 촬영 | 주 2~3건 방문 촬영 | 세한+수민 (각 1~2시간/건) | **가능** — 2인이 주 2~3회 가능 |
| 편집 | 주 2~3편 숏폼 (60~90초) | Zone X 에디터 1명 | **가능** — 숏폼 편집은 1편당 3~5시간 |
| 디자인 | 선수 소개 카드, 인스타 포스트 | 우석/훈택 | **가능** — 템플릿 세팅 후 반복 적용 |
| 병목 | 세한·수민의 주중 가용 시간 | 학업/다른 업무 확인 필요 | **확인 필요** |

**빌드업 실현가능성: HIGH (4/5)**

#### 컴바인 (5월 말, 2박 3일 → 10~12편 편집)

| 항목 | 분량 | 필요 리소스 | 실현 판정 |
|---|---|---|---|
| 촬영 | 2박 3일, 6개 라운드 + 인터뷰 + 브릿지 | 최소 2~3카메라, 별도 인터뷰 세팅 | **외부 보강 필수** — 1카메라로는 불가능 |
| 편집 | 10~12편 롱폼 (각 15~30분) | 지운PD + 에디터 2~3명 | **병목 구간** — 2~3주 집중 편집 필요 |
| 음향 | 현장 동시녹음 + 배경음악 라이선스 | 현장 사운드 장비 + 음악 소싱 | **확인 필요** — 현장 마이크 멀티 세팅 |
| 그래픽 | Hexagon 스탯 그래픽, 라운드별 타이틀, 스코어보드 | 디자이너 집중 투입 | **사전 제작 필수** — 촬영 전 템플릿 완료 |
| 장소 | 상징성 있는 촬영 장소 (폐교, 창고 등) + 장비 세팅 | 장소 섭외 + 장비 이동 + 셋업 1일 | **확인 필요** — 예산·물류 |

**컴바인 실현가능성: MEDIUM (3/5) — 외부 프로덕션 확보 여부에 달림**

#### 본 리그 (6월~, 격주 이벤트)

| 항목 | 분량 | 필요 리소스 | 실현 판정 |
|---|---|---|---|
| 촬영 | 이벤트당 8~12매치, 하루 촬영 | 최소 2카메라 + 해설 부스 + 관전자 촬영 | **루틴화 필요** — 첫 이벤트가 세팅 기준 |

## 7. 현재 기획에 대한 콘텐츠 이슈, 질문, 도발

### 7.1 스케줄/분량 관련 크리티컬 이슈

**이슈 1: 컴바인 11부작 + 리그 시작 타이밍 충돌**

> 컴바인 10~12편을 주 2회 업로드하면 5~6주 소요. 6월 초 시작 → 7월 중순까지.
> 본 리그 "6월 중순~" 시작은 컴바인 업로드가 끝나기 전에 리그를 시작한다는 뜻.

- **질문**: 컴바인 시리즈 업로드 중에 리그 촬영을 병행하는 건가? 아니면 컴바인 완결 후 리그 시작인가?
- **문제**: 컴바인 EP11에서 "Season 1이 시작된다!" 선언하는데, 시청자가 아직 EP7까지밖에 안 봤으면?
- **제안**: 리그 시작을 7월 중순 이후로 미루거나, 컴바인을 주 3회로 올려 4주 안에 끝내는 방안 검토

**이슈 2: 이벤트당 7종 콘텐츠가 지속 가능한가?**

> 현재 설계: 매치업 프리뷰 / 선수 각오 인터뷰 / 본 매치 / 하이라이트 클립 / 리캡+분석 / 랭킹 업데이트 / 다음 이벤트 티저 = 7종

격주 이벤트 × 7종 콘텐츠 = **2주에 7종 제작**.
이벤트당 8~12매치이므로 본 매치만 8~12편 + 하이라이트 8~12편 = 16~24편의 영상.
여기에 프리뷰, 리캡, 랭킹, 티저까지 더하면 **2주에 20~30편**.

- **질문**: 지운PD + 에디터 3명으로 2주에 20~30편이 가능한가?
- **제안**: 이벤트당 핵심 5매치만 풀 편집, 나머지는 하이라이트 or 모아보기로 축소. 7종 콘텐츠 중 "각오 인터뷰"와 "다음 이벤트 티저"를 통합.

**이슈 3: 빌드업→컴바인 인물 중복률**

> 빌드업에서 16~24편을 만들면 최소 16~24명 인터뷰. 컴바인 참가자는 10~16명.
> 빌드업에 나왔지만 컴바인에 안 오는 사람이 8~12명 발생.

- **질문**: 빌드업에서 시청자가 좋아했는데 컴바인에 안 오는 선수에 대한 서사 처리는? "이 사람은 왜 안 왔지?"라는 물음에 답이 있어야 함.
- **제안**: 빌드업 에피소드를 "인터뷰 에피소드(누구든)"와 "컴바인 확정자 프리뷰(확정된 사람만)"로 구분. 5월 이후에는 확정자 중심 빌드업으로 전환.

### 7.2 콘텐츠 설계 관련 질문

**질문 4: 빌드업 에피소드가 컴바인 참가자를 전원 미리 소개해야 하나?**

- 전원 소개 → 컴바인에서 "서프라이즈 참가자" 카드를 쓸 수 없음
- 일부만 소개 → 미소개 선수는 컴바인에서 배경지식 없이 등장 → 감정 투입 약해짐
- **제안**: 10~12명 중 6~8명은 빌드업에서 소개 + 2~4명은 "서프라이즈 참가자"로 컴바인 EP1에서 리빌. 서프라이즈 참가자의 빌드업 영상은 컴바인 시리즈 내에서 플래시백으로 삽입.

**질문 5: "중간 성적표" EP6의 위치가 적절한가?**

- 현재: 6개 라운드 중 3개가 끝난 시점 (EP 3-5 = 3라운드 + EP6 = 중간정리)
- 문제: EP6가 "쉬어가는 편"이 될 위험. 측정 장면 없이 인터뷰+순위만으로 한 편을 채울 수 있는가?
- **제안**: EP6를 독립 편으로 하기보다, EP5 후반부에 중간 순위 공개를 통합하고, EP6을 바로 BLAST 라운드(DETONATION)로 진입.

**질문 6: 컴바인 Day 1 / Day 2 촬영 분배**

- 현재 기획: EP 3-5 (ENGINE, GRIT, FORCE) = Day 1, EP 7-9 (BLAST, REFLEX, CONTROL) = Day 2
- Day 1에 전신 체력 소모 라운드가 3개 연속 → **선수 체력이 Day 2 결과에 영향**
- **질문**: 선수 피로도에 의한 성적 왜곡을 어떻게 처리할 것인가? 이것도 "콘텐츠"인가(지치는 것도 실력), 아니면 공정성 문제인가?
- **제안**: Day 1과 Day 2의 라운드 배치를 전략적으로 조합 (예: Day 1 = ENGINE + FORCE + CONTROL / Day 2 = GRIT + BLAST + REFLEX) — 피로도 영향 분산.

### 7.3 프로덕션 리스크 관련 도발

**도발 7: 2박 3일 안에 10~12편 분량의 소스를 확보할 수 있는가?**

- 6개 라운드 × 10~16명 참가 = 최소 60~96개의 측정 장면
- + 라운드 간 인터뷰 (라운드 전/후 × 10~16명 × 6라운드 = 120~192개 인터뷰 클립)
- + 브릿지 장면 (선수 간 대화, 식사, 대기실 등)
- = **수백 개의 클립 소스**를 2박 3일에 촬영

1카메라 + 지운PD만으로는 물리적으로 불가능.

- **최소 장비**: 경기 촬영 2카메라 + 인터뷰 전담 1카메라 = 3카메라
- **최소 인원**: 카메라 3명 + 사운드 1명 + 현장 디렉터 1명(수현) + 장비/세팅 1명 = 6명
- **PIP 프로덕션과의 구체적 협업 범위가 확정되어야 함**

**도발 8: 해설 시스템이 Season 1부터 필요한가?**

- 현재 기획: "2인 해설: 실황 캐스터 + 전문가 해설"
- **질문**: 캐스터는 누구인가? 전문가 해설은 누구인가? Season 1에서 캐스터 없이 자막+내레이션으로 대체 가능한가?
- 블랙컴뱃도 초기에는 해설 없이 시작 → 규모가 커지면서 해설 도입
- **제안**: Season 1은 내레이션 + 자막 + 선수 인터뷰로 구성. 해설은 Season 2부터 도입 검토.

**도발 9: 숏폼 클립 바이럴 전략이 구체적이지 않다**

- "CYA 선택 순간 클립 = 최고 바이럴 자산"이라고 되어 있지만:
  - 15초 숏폼에서 CYA 선택이 의미 있으려면 **그 전에 Hexagon 개념을 알아야 함**
  - 처음 보는 사람에게 "심폐 공격을 선택했습니다!"가 왜 중요한지 어떻게 설명?
- **제안**: 바이럴 클립은 CYA 선택보다 **시각적으로 강렬한 순간** (데드리프트 맥스, 비프테스트 마지막 탈락, 박스점프 1:1 등)이 먼저. CYA 선택 클립은 이미 팬인 시청자용.

**도발 10: 편당 8~12분 매치 영상이 너무 길지 않은가?**

- UFC 경기는 보통 5분 라운드 × 3 = 15분인데 **선수를 이미 알고 있는 팬**이 봄
- Zone X는 신생 콘텐츠 → 시청자가 선수를 덜 알 수 있음 → 8분도 길게 느낄 수 있음
- **제안**: Season 1은 "풀매치 8~12분 + 하이라이트 3분" 두 버전 제작. 하이라이트를 메인으로 홍보하고 풀매치는 팬용으로 포지셔닝. 조회수 데이터로 Season 2 방향 결정.

### 7.4 전략적 질문

**질문 11: "콘텐츠 70% = 경기 외부"라는 원칙이 Season 1에서 실현 가능한가?**

> 기획서에 "경기 자체(본 매치)는 전체 콘텐츠의 30%에 불과. 나머지 70%는 경기 전/후"라 되어 있음.
> 하지만 Season 1에는 선수가 10~16명 → 선수별 콘텐츠 양이 제한적.
> "프로모션 콘텐츠"를 만들 만큼 각 선수의 팬덤이 형성되어 있는가?

- **현실**: Season 1의 70%는 경기 자체 + 하이라이트 중심이 될 수밖에 없음. 프리뷰/리캡은 "있으면 좋지만 없어도 되는" 보너스로 접근해야 함.
- **제안**: Season 1에서는 본 매치 품질에 80% 집중. 프리뷰/리캡은 텍스트+이미지(인스타 카드)로 대체하여 영상 제작 부하 줄이기.
