# PART 1. Zone X — 이걸 왜 하는데?

## 한 줄

**종목 상관없이 — "가장 완성된 몸을 가진 사람이 누구인가"를 증명하는 스포츠 엔터테인먼트.**

## 시작은 이 질문이다

크로스피터, 마라토너, 파워리프터, MMA 파이터, 보디빌더, 러닝크루...
각자 자기 종목에서는 잘한다.

근데 — **"진짜 완성형 운동인"은 누구인가?**

달리기만 잘하면? 아니다.
무거운 거 들기만 잘하면? 아니다.
때리기만 잘하면? 아니다.

**6가지 신체 능력 — 심폐, 근지구력, 근력, 폭발력, 민첩성, 정교함 — 을 다 갖춰야 한다.**

Zone X는 이 질문에 답하는 콘텐츠이자 리그다.

## 왜 봐야 하는데?

> "크로스피터가 파워리프터보다 더 센가?"
> "마라토너가 MMA 파이터보다 빠른가?"

운동하는 사람이면 누구나 한 번쯤 생각한다 — **"내가 이 사람보다 나을까?"**

근데 종목이 다르면 비교할 수가 없었다. 축구선수와 러너가 어떻게 붙어?

Zone X는 그 비교를 가능하게 만든다.
**6가지 능력이라는 공통 잣대**로 모든 종목의 운동인을 같은 판에 세운다.

그리고 그 결과로 — **1:1로 붙는다.**

---

## 3단계 — 하나의 연결된 시리즈

전체가 하나의 연결된 시리즈다. 빌드업 → 컴바인 → 본 리그.
각 단계가 다음 단계를 위한 떡밥이 된다.

```
[4~5월] 빌드업 숏폼     →  [5월말~6월] 컴바인 시리즈  →  [6월 중~] Zone X 본 리그
"이 사람들은 누구인가"       "누가 진짜인가"                "누가 이기는가"
관심 + 호감                 이해 + 기대                    몰입 + 확인
```

### 1단계: 빌드업 숏폼 — "다양한 운동인을 만나서 묻는다"

| | |
|---|---|
| **누가** | 다양한 종목의 운동인 — 크로스피터, 마라토너, 격투기 선수, 보디빌더, 러닝크루, 소방관... 자기 종목에서 인정받는 사람들 |
| **뭘 하는데** | Zone X 촬영팀이 직접 찾아가서, "6가지 능력" 중 하나에 대해 대화하고, 간단한 측정을 해본다 |
| **그래서 뭐** | "이 사람 풀업 50개 하는데 유연성은 0이네?" — 자기 종목 밖에서의 민낯이 드러나는 순간이 재밌다. 그리고 "이 사람이 나중에 다른 운동인이랑 붙으면 어떻게 될까?" 궁금해진다 |

**이게 우리(세한·수민)가 4~5월에 만드는 콘텐츠다.**

### 2단계: 컴바인 시리즈 — "The Proving Ground"

| | |
|---|---|
| **누가** | 빌드업에서 이미 시청자가 얼굴을 아는 운동인 10~16명 |
| **뭘 하는데** | 2박 3일 동안 6개 라운드를 거친다. THE FURNACE에서 심폐를 시험받고, THE WALL에서 근력을 시험받고, THE CHASE에서 서로를 직접 쫓는다 |
| **그래서 뭐** | 누가 진짜 완성형 운동인인지 숫자로 드러난다. 데드리프트 220kg 드는 사람이 핸드스탠드 0초 — 그 장면이 재밌다. 마지막에 공개되는 Hexagon 스탯이 본 리그의 매치업을 결정한다 |

빌드업에서 발굴한 사람들이 여기 모인다. 블랙컴뱃 프로오디션 같은 역할.

### 3단계: Zone X 본 리그 (Season 1) — "Choose Your Attack"

| | |
|---|---|
| **누가** | 컴바인을 통과한 선수들 — 이미 Hexagon 스탯과 캐릭터(The Engine, The Tank 등)가 시청자에게 알려진 상태 |
| **뭘 하는데** | 1:1 매치에서 상대의 약점을 겨냥해 "심폐 공격 / 상체 공격 / 하체 공격" 중 하나를 선택한다. 심폐 92인 Engine형에게 심폐 공격을 걸면 자살행위, 근력 95인 Tank형에게 상체 공격을 걸면 자살행위 — **어떤 공격을 선택하느냐가 곧 전략** |
| **그래서 뭐** | 스탯으로 예측이 가능하니까 시청자가 경기 전부터 분석하고 예측한다. "이 조합이면 하체 공격이 답인데..." 실제로 선수가 뭘 선택하는지 보는 순간이 가장 짜릿하다 |

빌드업 → 컴바인이 다 이 순간을 위한 빌드업이다. 우리가 지금 만드는 콘텐츠가 여기로 연결된다.

## 시청자 여정 — 같은 사람이 세 번 다른 맥락으로 등장한다

```
빌드업에서 본 사람   →  컴바인에서 측정당함  →  본 리그에서 붙음
 "이 사람 재밌네"        "ENGINE 92래!"          "심폐 공격 선택했다!"
 (관심)                 (이해)                   (몰입)

 "풀업 50개?"           "근데 FORCE 45??"        "Tank형이 버틸까?"
 (호감)                 (기대)                   (긴장)

 "컴바인 나오려나"       "Tank형이랑 붙으면..."    "뒤집혔다!!!"
 (궁금)                 (예측)                   (확인 or 서프라이즈)
```

**핵심**: 볼수록 감정이 쌓인다. 빌드업에서 한 인터뷰가 컴바인에서 "예상대로" 또는 "예상과 다른" 결과로 이어지면 그게 드라마가 된다.

## 6가지 능력 — The Hexagon

누가 "6가지 능력이 뭔데?" 물으면 이걸 보여주면 된다.

| # | 능력 | Zone X 이름 | 한 줄 | 예시 |
|---|---|---|---|---|
| 1 | 심폐 지구력 | **ENGINE** | "얼마나 오래 갈 수 있는가?" | 비프테스트, 바이크 올아웃 |
| 2 | 근지구력 | **GRIT** | "얼마나 많이 버틸 수 있는가?" | FRAN (Thruster + Pull-up 21-15-9) |
| 3 | 근력 | **FORCE** | "얼마나 강한가?" | 데드리프트 1RM, 벤치 Max Rep |
| 4 | 폭발력 | **BLAST** | "얼마나 폭발적인가?" | 파워 클린, 박스점프 |
| 5 | 민첩성 | **REFLEX** | "얼마나 빠르게 대응하는가?" | L-Drill, 미러/추격 드릴 |
| 6 | 정교함 | **CONTROL** | "얼마나 정교하게 움직이는가?" | 핸드스탠드, 터키시겟업 |

> **"완성형 운동인 = 6가지 능력의 균형."** 힘만 세면 안 되고, 빠르기만 해도 안 된다. 다 갖춰야 한다.

컴바인에서 이 6개를 측정하면 선수마다 육각형(Hexagon) 스탯이 나온다.
그 스탯이 선수의 캐릭터가 된다 — "The Engine", "The Tank", "The All-Rounder" 같은 별명이 자연스럽게 붙는다.

## 누가 물으면 이렇게 설명하면 된다

> **30초 버전**: "종목 상관없이 가장 완성된 운동인이 누구인지 가리는 리그에요. 심폐, 근력, 폭발력 같은 6가지 능력으로 측정하고, 그걸로 1:1 대결을 해요. 지금은 그 전 단계로 다양한 운동인들 만나서 인터뷰하고 간단한 측정을 하는 숏폼 시리즈를 만들고 있어요."

> **1분 버전**: "크로스피터가 파워리프터보다 센가? 마라토너가 MMA 파이터보다 빠른가? — 이런 질문에 답하는 콘텐츠에요. 6가지 신체 능력을 다 갖춰야 '완성형 운동인'이라 할 수 있거든요. 먼저 다양한 종목의 운동인들을 만나서 인터뷰하고 간단한 측정을 해보는 빌드업 시리즈를 만들고, 여기서 발굴된 사람들이 컴바인에 모여서 진짜 실력을 증명해요. 그리고 그 결과로 1:1 대결 — 상대 약점을 공격하는 전략적 매치를 해요."

# PART 2. 빌드업 콘텐츠 — 우리가 만드는 것

## 기본 정보

| 항목 | 내용 |
|---|---|
| 채널 | Zone X (신규 유튜브 + 인스타) |
| 포맷 | 숏폼 60~90초 |
| 빈도 | 주 2~3회 |
| 기간 | 4월 초 ~ 5월 말 (약 8주) |
| 예상 편수 | 16~24편 |
| 톤앤매너 | 가볍고 친근. 운동하는 사람끼리의 대화. 인스타 릴스 감성 |

## 에피소드 구조 (60~90초)

```
[5초]  후킹 — 측정 결과 또는 인상적인 장면 미리보기
[20초] 소개 — 이 사람은 누구인가 (종목, 배경, 한 줄)
[30초] 대화 — 6개 역량 중 하나에 대한 생각/경험
[20초] 간단 측정 — 해당 역량 관련 간단한 테스트
[10초] 결과 + 리액션 + 다음 예고
```

## 대화 주제 — 6개 역량 기반 로테이션

각 에피소드에서 6개 능력 중 하나를 중심으로 대화한다:

| 능력 | 대화 키워드 | 질문 예시 |
|---|---|---|
| ENGINE | "끝까지 가는 엔진" | "운동하다 가장 한계에 다다른 순간은?" |
| GRIT | "포기하지 않는 반복" | "마지막 1렙에서 뭘 생각해?" |
| FORCE | "절대적인 힘" | "내 한계 무게를 들어올린 순간은?" |
| BLAST | "터뜨리는 순간" | "가장 폭발적이라고 느낀 순간은?" |
| REFLEX | "반응하는 몸" | "순간 판단이 승패를 갈랐던 경험은?" |
| CONTROL | "완벽하게 다루는 몸" | "운동하면서 몸의 정교함이 중요했던 순간은?" |

## 섭외 대상 유형

| 유형 | 섭외 대상 예시 | 에피소드 톤 |
|---|---|---|
| 크로스피터 | 박스 코치/선수 | "WOD를 갈아넣는 사람들" |
| 러너 | 마라톤/트레일러닝 | "42.195km의 엔진" |
| MMA/격투기 | 주짓수/복싱/무에타이 | "링 위의 전사" |
| 구기 종목 | 축구/농구 사회인리그 | "필드의 운동인" |
| 하이록스 | 참가자/코치 | "레이스의 철인" |
| 바디빌딩/피지크 | 보디빌더/피지크 선수 | "근력의 조각가" |
| 서프라이즈 | 소방관/군인/댄서 | "예상 밖의 완성형" |

## 빌드업이 하는 4가지 역할

1. **선수 수급** — 인터뷰하면서 자연스럽게 "컴바인 참가할 생각 있어?"
2. **6능력 교육** — 시청자에게 "완성형 운동인 = 6가지 능력의 균형" 개념 심기
3. **채널 빌딩** — Zone X 채널에 콘텐츠가 쌓여야 컴바인 시리즈가 의미 있음
4. **떡밥 설치** — "이 사람 FORCE는 확실한데 ENGINE은?" → 컴바인에서 검증

# PART 3. 촬영 실전 가이드

## 현장에서 확보해야 할 것

### 필수 정보 (매 촬영마다)

| 항목 | 예시 | 왜 필요한가 |
|---|---|---|
| 종목/배경 | "크로스핏 3년, 전직 축구선수" | 빌드업 에피소드 소개 |
| 소속 | "CrossFit Sentinel", "한강 러닝크루" | 커뮤니티 태그 → 확산 |
| 자타공인 강점 | "풀업 50개", "FRAN 2분대" | 컴바인에서 검증할 떡밥 |
| 본인이 말하는 약점 | "유연성은 자신 없다" | 예상 vs 실제의 드라마 |
| 참가 의향 | "컴바인 나갈 생각 있어?" | 선수 수급 핵심 |

### 촬영 소스

| 소스 | 용도 |
|---|---|
| 인터뷰 영상 (30~60초) | 에피소드 메인 |
| 간단 측정 장면 (20~30초) | 에피소드 후반 + 하이라이트 |
| 운동하는 모습 B-roll | 소개 패키지, 인스타 카드 |
| 현장 분위기 (체육관/필드/코스) | 배경 + 시각적 다양성 |
| 리액션 (측정 결과에 대한 반응) | 후킹 + 엔딩 |

### 간단 측정 예시 (역량별)

빌드업에서는 "정밀 측정"이 아니라 "맛보기 측정"이다. 재밌으면 된다.

| 역량 | 간단 측정 | 도구 |
|---|---|---|
| ENGINE | "지금 여기서 1분 최대 버피" | 없음 |
| GRIT | "풀업 맥스 도전" | 풀업바 (현장) |
| FORCE | "여기 있는 가장 무거운 거 들어봐" | 현장 장비 |
| BLAST | "멀리뛰기 or 수직점프 측정" | 줄자 |
| REFLEX | "반응 테스트" (간단 게임) | 스마트폰 앱 or 손반응 |
| CONTROL | "핸드스탠드 몇 초?" or "피스톨 스쿼트 돼?" | 없음 |

> **포인트**: 간단하게 해도 "이 사람이 뭘 잘하고 뭘 못하는지" 힌트가 나와야 한다. "풀업 50개 하는데 핸드스탠드 0초" — 이게 콘텐츠가 된다.

## 촬영 톤앤매너

- **가볍고 친근하게**. 인터뷰이가 자연스럽게 이야기하도록.
- **질문은 짧게, 반응은 크게**. "아 진짜요?" "대박" — 진짜 놀라는 거 다 OK.
- **측정은 가벼운 도전**. "자신 있다면서? 한번 해볼래?" 식의 톤.
- **결과에 대한 리액션이 핵심**. 잘 되든 안 되든, 그 순간의 표정이 콘텐츠.

## 촬영 체크리스트 (현장용)

```
□ 인터뷰이 기본 정보 확인 (종목/소속/경력)
□ 6개 능력 중 이번 에피소드 주제 설정
□ 인터뷰 녹화 (30~60초 분량 확보)
□ 간단 측정 장면 녹화
□ 측정 결과 리액션 녹화
□ B-roll (운동 장면, 현장 분위기)
□ 참가 의향 확인 ("컴바인 나올 생각 있어?")
□ 추가 섭외 가능성 확인 ("이 주변에 잘하는 사람 또 있어?")
□ 촬영 소스 → 에디터 전달
```

# PART 4. 빌드업 → 컴바인 연결

## 빌드업이 컴바인의 떡밥이 되는 구조

빌드업에서 한 말이 컴바인에서 검증된다. 이 연결이 핵심이다.

| 빌드업에서 일어나는 일 | 컴바인에서 일어나는 일 | 시청자 경험 |
|---|---|---|
| "풀업 50개 합니다" | GRIT 측정에서 FRAN 2:30 기록 | "역시 진짜였네" |
| "유연성은 자신 없어요" | CONTROL 체크리스트 2/8 통과 | "ㅋㅋㅋ 진짜 못하네" |
| "달리기는 좀..." | ENGINE 비프테스트 4번째 탈락 | "예상대로" or "예상 외로 잘했다" |
| "저 사람보다는 제가 잘할걸요" | 같은 라운드에서 직접 비교됨 | 빌드업 때 한 말 회수 |

**촬영할 때 이 연결을 의식해야 한다:**
- 인터뷰할 때 **자기 강점/약점에 대해 구체적으로** 말하게 유도하기
- "6가지 능력 중에 자신 있는 건?" "반대로 자신 없는 건?" — 이 질문을 꼭 하기
- 그래야 나중에 컴바인에서 "이 사람이 빌드업에서 뭐라고 했는지" 편집할 수 있다

## 섭외 → 컴바인 참가자 확정 흐름

```
빌드업 촬영하면서 섭외     →  참가 의향 확인    →  수현에게 보고    →  컴바인 참가자 확정
(매 촬영마다 자연스럽게)     (인터뷰 끝에)        (리스트 업데이트)    (5월 초 최종 확정)
```

- 모든 인터뷰이가 컴바인에 올 필요는 없다
- 하지만 **"이 사람은 올만하다"** 라는 판단이 서야 한다
- 기준: 종목 다양성 (같은 종목만 가득 차면 안 됨), 캐릭터, 실력, 컨디션

# PART 5. 타임라인 + 액션아이템 (Track B)

## 전체 타임라인에서 Track B 위치

```
3월 중순     3월 말      4월 초      4월~5월      5월 초     5월 말
   │           │           │           │           │          │
   ▼           ▼           ▼           ▼           ▼          ▼
일정 확정   섭외 리스트   촬영 시작   주 2~3회    참가자     컴바인
세한/수민   1차 완성     빌드업     현장 방문     확정      촬영
일정 확인               숏폼 촬영   + 촬영      (10~16명)  (2박3일)
섭외 루트                          + 섭외
확보
```

## 즉시 해야 할 것 (3월)

- [ ] 4~5월 촬영 가능 일정 확정 (주중/주말 몇 회 가능한지)
- [ ] 섭외 루트 확보
  - 세한: 크로스핏/하이록스 커뮤니티, 체육관 네트워크
  - 수민: 러닝크루, 구기종목 사회인리그, SNS 기반 접촉
- [ ] 섭외 1차 리스트 작성 (종목별 후보 3~5명)

## 4~5월 메인 업무

- [ ] 주 2~3회 섭외 대상 방문 + 인터뷰 + 간단 측정 촬영
  - 각 방문 시 확보: 종목/배경, 소속, 강점, 약점, 참가 의향
  - 촬영 소스 → Zone X 에디터에게 전달 → 숏폼 편집
- [ ] 촬영하면서 동시에 섭외 — "이 사람 알아?" 네트워킹 확장
- [ ] 공개 모집 폼 홍보 (체육관, 커뮤니티에 직접 전달)

## 병행: 스폰서십 서포트

- [ ] 수현과 협력하여 브랜드 스폰서십 소개서 준비
- [ ] 섭외 현장에서 협찬/파트너십 가능 브랜드·장소 발굴

## 5월 말: 컴바인 촬영

- [ ] 2박 3일 컴바인 촬영 현장 운영 지원
- [ ] 빌드업에서 만났던 선수들의 현장 케어

# PART 6. 이번 주 미팅 어젠다

> 목표: 빌드업 콘텐츠의 구체적인 실행 플래닝

### 어젠다

**1. 일정 확정 (10분)**
- 4~5월 각자 촬영 가능 일정 (주중/주말)
- 주 몇 회 촬영 가능한지 확인
- 첫 촬영 일자 확정

**2. 섭외 루트 구체화 (15분)**
- 세한이 접근 가능한 커뮤니티/체육관 리스트
- 수민이 접근 가능한 커뮤니티/크루 리스트
- 1차 섭외 타겟 종목별 3~5명 선정
- 첫 번째 촬영 대상 확정

**3. 촬영 포맷 얼라인 (10분)**
- 에피소드 구조 (후킹 → 소개 → 대화 → 측정 → 리액션) 확인
- 역할 분담: 누가 인터뷰하고, 누가 촬영하는지
- 촬영 장비 확인 (카메라, 마이크 등)
- 촬영 소스 전달 프로세스 (에디터 연계)

**4. 1화 파일럿 논의 (10분)**
- 첫 에피소드 대상/장소 확정
- 주제 (6개 능력 중 어느 것으로 시작?)
- 톤앤매너 얼라인

**5. 다음 액션아이템 정리 (5분)**
- 각자 이번 주에 해야 할 것 확정
