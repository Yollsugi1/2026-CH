## 2. 브랜드 아이덴티티 재정립

### 2.1 브랜드 언어 위계 확립

260318에서 여러 표현이 동시에 등장한다. 상황별로 쓸 언어의 위계를 정리해야 한다.

**1차 메시지 (Primary — 외부 커뮤니케이션의 첫 접점)**

> **"종목 상관없이 — 가장 운동을 잘하는 사람이 누구인가."**

이것이 훅이다. 모든 대외 커뮤니케이션의 첫 문장. 포스터, 배너, 채널 소개, 썸네일 텍스트에 사용.

**2차 메시지 (Secondary — 세계관을 설명할 때)**

> **"진짜 완성형 운동인을 가린다."**

"완성형 운동인"은 Zone X의 세계관 안에서만 작동하는 용어. 빌드업을 통해 6능력의 맥락이 쌓인 후에야 의미가 전달된다. 처음 접하는 사람에게는 1차 메시지가 먼저.

**3차 메시지 (Tertiary — 리그 차별화를 설명할 때)**

> **"쇼가 아니라 리그다. 랭킹이 남고, 챔피언이 있고, 방어전이 있다."**

P100 비교 질문에 대응할 때. 또는 Zone X의 장기 비전을 설명할 때.

**사용 가이드**:

| 상황 | 사용 메시지 | 예시 |
|---|---|---|
| 처음 만나는 사람에게 설명 | 1차 | "가장 운동 잘하는 사람이 누구인지 가리는 리그예요" |
| 채널 배너/프로필 | 1차 | "WHO IS THE MOST COMPLETE ATHLETE?" |
| 빌드업 에피소드 자막 | 1차 → 2차 전환 | "잘하는가?" → "완성형이란 무엇인가?" |
| 컴바인 시리즈 내레이션 | 2차 | "진짜 완성형 운동인을 가리기 위해 여기 모였다" |
| P100 비교 대응 | 3차 | "그건 쇼, 이건 리그. 다음 시즌이 있다" |
| 스폰서십 자료 | 1차 + 3차 | 포지셔닝 + 리그 구조의 차별점 |

### 2.2 "더 운동을 잘하는가" 비주얼 언어 변환

훅 질문 "크로스피터가 파워리프터보다 더 운동을 잘하는가?"를 비주얼로 변환하는 방법.

**VS 그래픽 시스템**:
- 두 선수/종목의 실루엣이 마주 보는 구도
- 중앙에 "VS" 대신 **"더 잘하는가?"** 텍스트 배치
- 각 선수 옆에 종목 아이콘 (크로스핏 로고, 러닝 실루엣 등)
- 배경: Zone X 다크 톤. 분리선은 액센트 레드.

**훅 숏폼 오프닝 템플릿**:
```
[화면 분할] 좌: A종목 운동 장면 / 우: B종목 운동 장면
[중앙 텍스트] "크로스피터가 마라토너보다"
[텍스트 전환] "더 운동을 잘하는가?"
[Zone X 로고 플래시]
```

이 템플릿을 빌드업 전 에피소드에 반복 적용하면, "잘하는가?" 질문 자체가 Zone X의 비주얼 시그니처가 된다.

**핵심 감정**: "누가 진짜인가?" + "자존심 싸움에서 진 사람이 정식에서는?"

| **편집 톤** | 긴 호흡, 슬로모, 서스펜스 빌딩. 인터뷰 삽입(과거 발언 플래시백). 사운드: 드럼 빌드업, 서스펜스 오케스트라, 하트비트 |
| **그래픽 밀도** | 중간~높음. 라운드 타이틀 카드, 순위표, 개인 스탯 리빌 애니메이션. EP 10에서 Hexagon 첫 공개 시 풀 애니메이션 |
| **SNS 톤** | "ENGINE 92 실화?" / "자기 강점이라던 FORCE에서 졌다" / "중간 순위 뒤집혔다" — 서프라이즈 + 데이터 기반 감탄 |

**ACT 1 자존심 배틀의 톤 (특별 가이드)**:

자존심 배틀은 빌드업(가벼움)과 컴바인(긴장) 사이의 **전환점 톤**이다.

| 요소 | EP 1 (집결+선전포고) | EP 2 (자존심 배틀) |
|---|---|---|
| 감정 키워드 | 자신감, 도발, 신경전 | 충돌, 당혹, 자존심 |
| 카메라 | 시네마틱 입장 + 핸드헬드 신경전 | 멀티캠, 반응 클로즈업 집중 |
| 편집 | 빌드업 영상 플래시백 삽입 + 선언 순간 슬로모 | 미니 대결 실시간 + 결과 리액션 교차편집 |
| 사운드 | 입장 음악(각 선수별) + 선언 시 정적 | 대결 시 서스펜스 빌드업 + 결과 시 임팩트 사운드 |
| 비주얼 톤 | 빌드업의 밝은 톤에서 컴바인의 다크 톤으로 **점진 전환** | 완전한 다크 톤 — 이제 컴바인이다 |

### 4.2 Hexagon 시각화 디자인 가이드

**기본 규격**:

```
         ENGINE
          /\
  CONTROL/  \GRIT
        /    \
  REFLEX\    /FORCE
         \  /
         BLAST
```

- 꼭짓점 순서: 상단부터 시계 방향 — ENGINE → GRIT → FORCE → BLAST → REFLEX → CONTROL
- 이 순서는 모든 매체에서 고정. 변경 불가
- 외곽선: 흰색 또는 밝은 그레이 (#CCCCCC), 1px
- 채움: 반투명 액센트 레드 (#E94560, opacity 30~40%)
- 각 꼭짓점에 능력 카테고리명(CAPS) + 점수(0~100)
- 다크 배경(#0A0A0A) 위에 배치

**비교 모드 (2인)**:
- 좌: A선수 Hexagon (반투명 레드), 우: B선수 Hexagon (반투명 블루 #4A90D9)
- 또는 하나의 Hexagon에 두 선수를 겹쳐 표시 (레이어)
- 겹침 표시 시: A선수 = 레드 채움, B선수 = 블루 채움, 겹치는 영역 = 보라색
- 아래에 각 능력별 수치 비교 바 차트 보조 배치

**사이즈 변형**:
| 용도 | 비율 | 크기 |
|---|---|---|
| 인스타 피드 | 1:1 | 1080x1080 |
| 인스타 스토리/릴스 | 9:16 | 1080x1920 |
| 유튜브 오버레이 | 16:9 | 1920x1080 |
| 유튜브 썸네일 | 16:9 | 1280x720 |

### 4.3 CYA 3공격 유형 컬러 확정안

프레임워크 Q3에서 제기된 "메인 레드와 상체 공격 레드의 충돌" 문제를 해결한다.

**원칙**: 3공격 유형 컬러는 Zone X 메인 팔레트(#0A0A0A + #E94560)와 **계층이 다른 보조 컬러**로 작동한다. CYA 선택 UI 안에서만 사용하며, 브랜드 전체에 적용하지 않는다.

| 공격 유형 | 컬러 | HEX | 근거 |
|---|---|---|---|
| 심폐 공격 | **시안(Cyan)** | #00BCD4 | 산소, 호흡, 지구력. 블루 계열이지만 레드와 대비 명확 |
| 상체 공격 | **앰버(Amber)** | #FFC107 | 힘, 무게, 경고. 레드를 피해 옐로-오렌지 계열로 이동 |
| 하체 공격 | **라임(Lime)** | #8BC34A | 스피드, 폭발, 에너지. 그린 계열 |

**충돌 방지**: 상체 공격에 레드를 쓰지 않음으로써 Zone X 메인 레드(#E94560)와의 혼동을 제거. 메인 레드는 Zone X 브랜드 전체의 액센트로만 사용.

**CYA UI에서의 적용**:
- 선택 순간: 해당 공격 유형의 컬러가 화면 전체를 플래시
- 선택 카드: 공격 유형명 + 아이콘 + 해당 컬러 배경
- 다크 배경 위에서 3색 모두 명확히 구분되는지 확인 필요

### 4.4 티어 시스템 비주얼 (Open / Challenger / Elite)

Season 1에서는 미사용이지만, 컴바인 3일을 티어별로 운영할 경우 비주얼이 필요하다.

| 티어 | 컬러 | 심볼 | 시각적 성격 |
|---|---|---|---|
| **Elite** | 골드 (#FFD700) + 다크 | 왕관 또는 방패 | 고급스러움, 위엄 |
| **Challenger** | 실버 (#C0C0C0) + 다크 | 칼 또는 상승 화살표 | 도전, 날카로움 |
| **Open** | 브론즈 (#CD7F32) + 다크 | 문 또는 입장 | 진입, 시작 |

티어 컬러는 Zone X 메인 팔레트와 독립적으로 운영한다. 메인 컬러(다크+레드)는 Zone X 전체, 티어 컬러(골드/실버/브론즈)는 티어 뱃지와 랭킹 표시에만 사용.

## 5. 크로스 이슈 해결 제안

### 5.1 QA 합의점에 대한 브랜딩 해결안

**합의 1.1 — 콘텐츠 볼륨 비현실적 (4/4 합의)**
- 브랜딩 관점: 그래픽 자산 10종 + Phase별 변형을 우석 1인이 감당 불가
- **해결안**: 자산을 "템플릿 자산"과 "유니크 자산"으로 분리
  - 템플릿 자산 (데이터만 바꾸면 됨): Hexagon 카드, 랭킹 카드, 매치 프리뷰 → 반자동화
  - 유니크 자산 (매번 새로 만들어야 함): 라운드 타이틀, 이벤트 포스터, 특별 뱃지 → 우석 집중
  - 이렇게 분리하면 우석의 유니크 자산 작업량이 50% 이하로 감소

**합의 1.5 — 빌드업→컴바인 인물 연결 (4/4 합의)**
- 브랜딩 관점: 빌드업에서 소개된 선수가 컴바인에 등장할 때 "같은 사람"으로 즉시 인식되어야 함
- **해결안**: 선수 소개 카드의 디자인 코드를 빌드업과 컴바인에서 **동일하게 유지**하되, Phase에 따라 정보량만 변화
  - 빌드업: 이름 + 종목 + 한줄 (스탯 없음)
  - 컴바인: 이름 + 종목 + 아키타입 + Hexagon 스탯 추가
  - 같은 카드 프레임에 정보가 쌓이는 구조 → 시각적 연속성

**합의 1.6 — CYA 시청자 이해 장벽 (3/4 합의)**
- 브랜딩 관점: "심폐 공격"이라는 용어가 직관적이지 않음
- **해결안**: CYA 3공격 유형에 **비주얼 아이콘 + 컬러**를 부여하여 텍스트 없이도 인식 가능하게
  - 심폐 공격: 폐/심장 아이콘 + 시안
  - 상체 공격: 바벨/팔 아이콘 + 앰버
  - 하체 공격: 다리/스프린트 아이콘 + 라임
  - 이 아이콘+컬러 조합을 빌드업 단계부터 노출하여(6능력 교육과 함께) 시청자가 리그 시작 전에 이미 인식하도록

### 5.2 QA 충돌점에 대한 브랜딩 입장

**충돌 2.5 — 6능력 Hexagon 복잡도**
- 브랜딩 에이전트 입장: **6개 유지하되 커뮤니케이션 레이어를 분리**
  - Layer 1 (첫 접점): "체력 vs 힘 vs 스피드" — 3분류로 단순화. 이것이 CYA 3공격 유형과 직결됨
  - Layer 2 (빌드업 이후): 각 분류가 2개씩 세분화됨을 교육. 체력=ENGINE+GRIT, 힘=FORCE+CONTROL, 스피드=BLAST+REFLEX
  - Layer 3 (컴바인): 6개 전체 능력을 Hexagon으로 시각화
  - 이 3-Layer 접근은 "6능력을 줄이는 것"이 아니라 "진입 장벽을 낮추는 것"

**충돌 2.4 — 해설 시스템**
- 브랜딩 입장: Season 1에서 수현이 내레이션/현장 해설을 겸하는 것을 **적극 지지**
  - 이유: 초기 브랜드 보이스는 창업자의 보이스에서 나와야 자연스러움
  - UFC의 다나 화이트가 프로모션의 얼굴이듯, 수현이 Zone X의 초기 보이스가 되는 것은 브랜드 정체성 형성에 유리
  - Season 2+에서 전문 캐스터를 추가하면 "원래 보이스 + 전문 보이스" 2인 체제가 자연스럽게 완성

### 5.3 세계관 일관성 유지 방안 (빌드업 → 컴바인 → 리그)

**원칙: "같은 세계, 다른 챕터"**

세 Phase는 톤이 다르지만, 아래 요소는 반드시 일관되게 유지한다:

| 불변 요소 | 구체적 기준 |
|---|---|
| Zone X 로고 | 모든 콘텐츠 인트로/아웃트로에 동일 로고 |
| 컬러 팔레트 기조 | 다크 베이스 + 레드 액센트 (밝기만 Phase별 조절) |
| 6능력 체계 | 능력명/카테고리명 표기 규칙 통일 |
| 선수 프로필 프레임 | 동일 카드 프레임, 정보량만 Phase별 증가 |
| "잘하는가?" 질문 | 매 Phase에서 다른 맥락으로 반복 |

**전환 연결 장치**:
- 빌드업 마지막 에피소드에서 컴바인 티저 (톤 전환 예고)
- 컴바인 EP 11에서 리그 매치 카드 첫 공개 (CYA UI 첫 노출)
- 각 전환점에서 "이전 Phase 하이라이트 몽타주" → "새 Phase 타이틀 카드" 구조

### 5.4 Creatorhood <-> Zone X 브랜드 관계 정립

**현재 구조**: Creatorhood(모회사/스튜디오) → Zone X(프로젝트/리그)

**정리 원칙**:

| 항목 | Creatorhood | Zone X |
|---|---|---|
| 성격 | 스포츠 라이프스타일 콘텐츠 스튜디오 | 피트니스 스포츠 리그 |
| 채널 | @creatorhood.mag (인스타), CH MAG (유튜브) | Zone X (별도 유튜브/인스타) |
| 로고 | Creatorhood 로고 | Zone X 로고 |
| 연결 | "Presented by Creatorhood" | Zone X 콘텐츠 말미에 Creatorhood 크레딧 |
| 톤 | 친근, 매거진, 라이프스타일 | 전투적, 데이터, 스포츠 |

**금지 사항**:
- Zone X 콘텐츠에 Creatorhood의 다른 시리즈(Hoodies, Playbook 등) 광고를 직접 삽입하지 않음
- Zone X 채널에서 Creatorhood 톤(친근/매거진)을 사용하지 않음
- 단, 크로스 프로모션은 가능: "Zone X 선수가 Playbook에 게스트 출연" 같은 자연스러운 연결

**연결의 이점**:
- Creatorhood의 기존 팔로워에게 Zone X를 노출할 수 있는 채널
- Zone X의 성장이 Creatorhood 전체 브랜드의 입체성을 높임
- 그러나 두 브랜드는 시각적/톤적으로 **명확히 분리**되어야 함
