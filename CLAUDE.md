# Creatorhood (크리에이터후드) 2026 Workspace

## 회사 개요

**크리에이터후드 (Creatorhood)**는 스포츠 라이프스타일 콘텐츠 스튜디오다.
- 포지셔닝: "스포츠판 메티즌" — 운동하는 사람들의 입체적인 삶을 다루는 미디어
- 타겟: "운동에서 배운 몰입의 에너지를 삶 전반에 적용하는 사람들"
- 2025년까지 인스타그램 매거진 → 2026년부터 유튜브+인스타+웹매거진 영상 중심 콘텐츠 스튜디오로 전환
- 대표: 정수현 (@yoll_sugi)
- 인스타그램: @creatorhood.mag

## 법인 구조

- 신설법인: 정수현 51% × 한상필 49%
- 역할: 수현(경영·콘텐츠) / 한상필(영업 전담)
- 기존 클라이언트(국순당 등)는 개인사업자에 귀속

## 사업 4대 축

| 축 | 프로젝트 | 성격 |
|---|---|---|
| A | CH MAG | 자체 콘텐츠 미디어 (핵심 사업) |
| B | Zone X | 크로스핏/기능성 피트니스 기반 스포츠 리그 |
| C | 클라이언트 사업 | 대행 매출 (국순당, 카스라이트, 인헤이즈 등) |
| D | 신설법인 운영 | 법인 설립, 인턴, 재무 안정화 |

## 3레이어 프레임워크

- **Layer 1 (핵심 사업)**: Zone X 스폰서십 + CH MAG 시리즈 브랜드 딜 — 핵심팀 Full 투입
- **Layer 2 (서브채널)**: 서브 매거진 채널 2개 — 인턴+대학생 에디터 운영, 광고 구좌 판매
- **Layer 3 (추가 매출)**: 한상필 대표 외부 수주 건 — 외주 중심 실행, 내부 리소스 최소화

## 작업 컨벤션

### 언어
- 문서 작성: 한국어 기본 (시리즈명, 브랜드명 등 고유명사는 영문 유지)
- 코드/기술 문서: 한영 혼용

### 파일 네이밍
- 세션 백업: `SESSION_BACKUP_[주제]_[YYMMDD].md`
- 내부 문서: 한글 파일명 사용 가능
- 영문 문서: kebab-case

### 워크 디렉토리별 역할
각 `claude-*` 디렉토리는 특정 업무 영역의 Claude 작업 공간이다. 디렉토리 안에서 작업할 때 해당 영역의 맥락을 우선한다.

### 주의사항
- 이 워크스페이스는 git 리포지토리가 아님
- PDF 파일이 많음 — 소개서, 포트폴리오, 제안서 등 참고용
- `.skill`, `.plugin` 파일은 Claude 커스텀 스킬/플러그인 정의 파일
- 재무 데이터(거래내역, 금액 등)는 민감 정보 — 외부 공유 주의
- 한상필 대표 공유용 문서와 내부용 문서는 명확히 구분할 것
