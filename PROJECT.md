# Creatorhood (크리에이터후드)

"스포츠판 메티즌" — 스포츠 라이프스타일 영상 중심 콘텐츠 스튜디오

## What it does

- **CH MAG**: 7개 시리즈로 구성된 스포츠 라이프스타일 자체 미디어
- **Zone X**: 크로스핏 기반 스포츠 리그 (2026년 6월 시즌 1)
- **클라이언트 대행**: 브랜드 콘텐츠 제작 및 SNS 운영

## How to use

이 저장소는 크리에이터후드 2026년 프로젝트 관리 및 콘텐츠 기획 문서를 포함합니다.

## Technologies used

- Instagram / YouTube / 웹매거진 (플랫폼)
- 롱폼 + 숏폼 영상 포맷
- 에디토리얼 큐레이션 콘텐츠
