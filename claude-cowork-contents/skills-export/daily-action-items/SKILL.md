# 데일리 액션아이템 + 스케줄 캘린더 DB 입력 스킬

## Objective

매일 아침 마스터플랜에서 오늘 날짜 기준 액션을 추출하여:
1. **오늘의 체크 항목 요약** — 수현님이 빠르게 훑어볼 수 있는 형태
2. **노션 DB 복붙용 테이블** — '2026 스케줄 캘린더 (CEO Staff)' DB에 바로 입력 가능한 테이블+CSV

## 실행 순서

### Step 1: 오늘 날짜 확인

```bash
date +%Y-%m-%d  # 오늘 날짜 확인
```

### Step 2: 마스터플랜 읽기

- 워크스페이스 폴더에서 `크리에이터후드_2026_마스터플랜*.md` 검색
- 핵심 섹션: "2. 이번 주 ~ 다음 주 상세 스케줄"에서 오늘 날짜 행 추출
- 보조 섹션: "1. 시리즈별 현황"에서 오늘 관련 context 파악

### Step 3: 오늘 항목 추출

마스터플랜 스케줄 테이블에서 **오늘 날짜에 해당하는 모든 행**을 추출하고, 각 항목을 아래 DB 스키마에 매핑:

**노션 DB 스키마 (2026 스케줄 캘린더 CEO Staff):**

| 컬럼 | 타입 | 선택지 |
|------|------|--------|
| 이벤트명 | 제목 | — |
| 날짜 | 날짜 | 시작→종료 |
| 프로젝트 | 선택 | CH MAG / Zone X / 서브채널 / 법인·경영 / 클라이언트 / 전체 |
| 유형 | 선택 | 촬영 / 미팅 / 기획 / 마감 / 업로드 / 기타 |
| 시리즈 | 선택 | Playbook / Pit Stop / 폭력적인 식단 / Shirtless / Hoodies / 후드의 주민들 / False 9 / Zone X / 서브채널 / 기타 |
| 상태 | 선택 | 확정 / 제안 / 조율중 / 완료 |
| 담당 | 텍스트 | — |
| 비고 | 텍스트 | — |

**매핑 규칙:**
- 마스터플랜의 "액션" → 이벤트명
- 마스터플랜의 "프로젝트" → 프로젝트 (선택지에 맞게 변환)
- "촬영", "미팅", "기획" 등 키워드로 유형 자동 분류
- 시리즈명은 마스터플랜에서 직접 매칭
- 캘린더에 이미 있으면 비고에 "캘린더 등록 완료" 표기
- 시간이 TBD이면 비고에 "시간 TBD" 표기

### Step 4: 주간 단위 항목도 함께 제공

오늘뿐 아니라 **이번 주 전체 항목**을 함께 테이블로 제공하되, 오늘 항목은 상단에 별도 분리하여 강조.

### Step 5: 출력 생성

아래 템플릿에 맞춰 MD 파일 작성:

```markdown
# 데일리 액션아이템 — {YYYY/MM/DD (요일)}

> 마스터플랜 v{버전} 기준 오늘 체크 항목

---

## ☀️ 오늘 ({M/D 요일}) 핵심

{오늘 가장 중요한 1~2가지를 한 문장으로}

---

## 📅 오늘 입력 항목

| 이벤트명 | 날짜 | 프로젝트 | 유형 | 시리즈 | 상태 | 담당 | 비고 |
|---------|------|---------|------|--------|------|------|------|
| {항목1} | {날짜} | {프로젝트} | {유형} | {시리즈} | {상태} | {담당} | {비고} |

---

## 📅 이번 주 전체 (W{주차})

| 이벤트명 | 날짜 | 프로젝트 | 유형 | 시리즈 | 상태 | 담당 | 비고 |
|---------|------|---------|------|--------|------|------|------|
| {전체 항목...} |

---

## 📋 CSV (노션 Import용)

\```csv
이벤트명,날짜,프로젝트,유형,시리즈,상태,담당,비고
{오늘 항목 CSV}
\```

---

## ⏰ 내일 미리보기

{내일 예정 항목 간단 요약 (1~2줄)}

---

*마스터플랜 v{버전} 기준 | DB: 2026 스케줄 캘린더 (CEO Staff)*
```

### Step 6: 파일 저장

- 파일명: `데일리_액션아이템_{YYYYMMDD}.md`
- 저장 위치: 워크스페이스 폴더 (`/mnt/claude-cowork-contents/`)

## 상태 판단 기준

| 마스터플랜 표현 | 상태 값 |
|---------------|--------|
| ✅, "확정", "캘린더 등록" | 확정 |
| "제안", "후보", "(제안 중)" | 제안 |
| "TBD", "조율", "가일정" | 조율중 |
| 완료 표시 | 완료 |

## 유형 자동 분류 키워드

| 키워드 | 유형 |
|--------|------|
| 촬영, 티저+ep, 러닝크루 | 촬영 |
| 미팅, 팔로업, 킥오프 | 미팅 |
| 기획, 구상, 방향 | 기획 |
| 마감, 데드라인, 마무리 | 마감 |
| 업로드, 릴리즈, 발행 | 업로드 |
| 기타 | 기타 |

## 성공 기준

- 오늘 날짜 항목 누락 없이 추출
- DB 스키마와 정확히 매칭 (선택지 오타 없음)
- 테이블 형식 + CSV 형식 동시 제공
- 내일 미리보기 포함
- 한국어, 간결

## 제약사항

- 마스터플랜 파일이 없으면 사용자에게 요청
- DB 선택지와 정확히 일치하는 값만 사용 (오타 주의)
- 항상 한국어
- 오늘 항목이 없어도 "오늘 스케줄 캘린더 항목 없음" 명시 + 내일 미리보기는 제공
- CSV에서 쉼표가 포함된 값은 큰따옴표로 감싸기
