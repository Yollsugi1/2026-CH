## 영상 핵심 요약

- 르클레르의 아이코닉한 심플 헬멧 디자인은 **실수로 절반을 삭제**한 데서 탄생했다
- 디자이너 아드리앙 파비오는 2012년부터 7명의 F1 드라이버 헬멧을 만들어온 전문가
- 르클레르 본인은 직선 하나 못 그릴 정도로, 헬멧 디자인은 100% 파비오에게 위임

## 영상에서 못 다룬 이야기

### 1. 아드리앙 파비오는 누구인가 — 저널리스트 출신의 이색 경력

파비오는 원래 **모터스포츠 저널리스트이자 레이서**다. 프랑스 투르 출신으로, 유로스포츠에서 포뮬러 E 해설을 맡고 있고, 2004년에는 FFSA 프랑스 컵에서 프랑스 챔피언에 오른 경력도 있다. 직접 레이스를 뛰어본 사람이 헬멧을 디자인하니까, 드라이버의 니즈를 본능적으로 이해하는 거다.

2012년 **로맹 그로장**과의 작업이 시작이었다. 원래 친구 사이였는데, 그로장의 헬멧 디자인을 맡으면서 본격적인 F1 헬멧 디자이너 커리어가 시작됐다.

### 2. 파비오가 작업한 7명의 F1 드라이버 — 각각의 스타일

| 드라이버 | 특징 |
|---|---|
| **로맹 그로장** | 파비오의 첫 F1 클라이언트, 친구에서 시작 |
| **막스 페르스타펜** | 현 월드 챔피언의 헬멧도 파비오 작품 |
| **피에르 가슬리** | 프랑스 동료 드라이버 |
| **유키 츠노다** | 일본 드라이버 |
| **케빈 매그누센** | 덴마크 드라이버 |
| **이삭 하자르** | 최근 합류한 신예 |
| **샤를 르클레르** | 2018년 소버 데뷔부터 지금까지 가장 긴 파트너십 |

페르스타펜의 헬멧도 파비오가 만들었다는 사실 — 즉 F1 그리드에서 가장 유명한 헬멧 두 개를 같은 사람이 디자인한 셈이다.

### 3. 르클레르 헬멧의 연대기 — 복잡함에서 심플함으로

**2018 (소버 데뷔):** 처음엔 복잡한 패턴과 빨강+검정+흰색의 다양한 색상 조합. 르클레르 자신도 뭘 원하는지 정확히 몰랐던 시기.

**전환점 — "실수의 순간":** 파비오가 작업 중 디자인 파일의 절반을 실수로 삭제. 남은 심플한 형태가 오히려 완벽해서 바로 스크린샷을 찍어 르클레르에게 전송. 채택.

**2022 모나코 GP:** 모나코 국기를 모티프로 한 디자인(위쪽 빨강, 아래쪽 흰색). 파비오는 "아이 같은 단순함의 미학"이라 표현. 세나 헬멧에 비견되는 아이코닉한 가치.

**2024:** 모나코 GP용이었던 디자인이 **공식 시즌 헬멧**으로 승격.

**2025:** 빨강+흰색 2분할 구조 유지, 7시즌째 페라리와 함께.

**2026 (현재):** 기존 빨강+흰색 베이스에 **베이비 블루** 터치 추가. 르클레르 본인 왈 "제가 가장 좋아하는 색이에요. 심플하게 유지하되, 모나코 국기와 제 컬러를 담고 싶었어요."

### 4. F1 헬멧은 어떻게 만들어지나 — 제작 과정과 규정

**FIA Standard 8860** — F1 헬멧의 기술 기준:
- 외부 셸: **카본 파이버** 또는 동등 경도 소재 필수
- 바이저: **렉산(Lexan)** 소재
- 무게: 액세서리 제외 **1.8kg 이하** (드라이버 목 부담 감소)
- 테스트: 제조사가 FIA에 7개 헬멧을 제출, -20°C~50°C 환경에서 낙하+충격 테스트

**파비오 ↔ 르클레르 연간 프로세스:**
1. **11월**: 파비오와 르클레르가 만나 아이디어 교환
2. 페라리가 스폰서 가이드라인 제공 (패치 크기, 색상, 위치)
3. 드라이버가 바이저, 나사, 안감 색상 선택
4. 정치적·성적 논란 요소 배제
5. 완성 → 매니저 올리비에에게 전달 → 페라리 승인
6. 첫 경기에서 페라리 대표가 현장 검수

**디자인 자유도:** 2015~2019년 사이 FIA가 시즌 중 헬멧 디자인 변경을 제한했었지만, 2020년부터 **"레이스 간 무제한 변경 가능"**으로 규정이 완화됐다. 덕분에 특별 GP용 원오프 디자인이 다시 활성화.

### 5. F1 헬멧 = 예술인가?

파비오는 자신의 작업을 **"확실히 예술"**이라고 말한다. 비공식 레플리카 시장이 존재할 정도로 팬덤이 강하다:
- 피사 본사의 라이트닝 맥퀸 헬멧
- 모나코 1000번째 경기 특별판
- 몬차 2023 옐로우 헬멧

모터스포츠에서 헬멧은 드라이버의 **유일한 개인 표현 수단**이다. 차는 팀 것이고, 레이싱 슈트도 스폰서 로고로 뒤덮여 있지만, 헬멧만큼은 드라이버의 정체성을 담는 캔버스인 셈.
