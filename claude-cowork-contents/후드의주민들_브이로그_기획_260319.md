## 1. 대화에서 나온 핵심 문제의식

### 브이로그의 본질은 '스포츠'가 아닐 수 있다
- 임용 제기: CH 네 명의 공통 관심사가 스포츠이긴 하지만, **브이로그에서 더 의미 있는 건 '일'일 수 있다**
- 각 멤버의 실제 Core가 다름:
  | 멤버 | Core |
  |---|---|
  | 수현 (대표) | 스포츠 < productivity 기반 entrepreneur |
  | 우석 (디자이너) | 맛있는거 먹으면서 실없는 농담 |
  | 임용 (에디터) | 비주류 (킥킥) |
  | 지운 (PD) | 아직 파악 못함 — 대부분 무반응인데 좋아하는거 하면 해피해피 |

### 수현의 병목
- 혼자서 6개 시리즈 기획을 짜는 게 아무리 Claude 활용해도 무리
- 지운님이 기획단에서 많은 기여가 어려운 상황 (영상 전문, 브이로그 경험/감이 적음)
- 브이로그 기획 + 피드백에 지금까지 시간이 꽤 많이 들어감

### 지운님 관련 이슈 (촬영 측면)
1. **풋티지 자체가 많이 없음** — 다른거 찍을게 많아서 챙기지 못함
2. **브이로그를 많이 안 봐서** 기획 감이 조금 없음 → 피드백 단에서는 반영해가며 편집하면 되지만, 첫 기획 단계에서 수현이 스크립트 수준으로 상세하게 흐름을 잡고 있음
3. 지운/우석 둘이 서로 찍게 하려 했는데 **그렇게 하면 말을 서로 안 함**

### 임용의 고민
- 재택이라 에피소드의 **전체 방향성을 잡고 가는 사람이 되기 어려움**
- 하지만 **카메라 들고 말걸고 찍는 건 잘할 수 있음** (해보니까)
- 풋티지 쌓고 → 거기서 맥락 찾는 방식은 **장기적으로 길을 잃을 수 있음**
- 월 1회 해인칭 스타일의 프로젝트는 이 면에서 편하게 접근 가능할 듯

## 3. 제작 프로세스 방향

### 기획 → 촬영 → 편집 흐름
```
[수현] 내용 기획 (어떤 내용으로 브이로그 만들건지)
  ↓
[수현] 큰 틀에서 뭘 찍을지 정함
  ↓
[임용] 카메라 들고 닥치는대로 풋티지 많이 남김
  ↓  ※ 연출조 없이, 자연스럽게
[지운] 편집 — 피드백 반영하며 완성
```

### 핵심 원칙
- **지운님에게 기획을 맡기면 → 연출조가 됨** → 원하는 게 아님
- 내용 먼저 잡고 → 큰 틀에서 뭘 찍을지 정하고 → **풋티지 많이 찍어놓고** → 그 기반으로 편집 (자연스럽게 / 연출조 없이)
- 임용: "막 던지고 널널대고 이런 식으로 하는게 더 재밌어서" → **캐릭터성 살리기**
- 우석님도 단순 개그보다 **취향이나 이런 얘기**가 재밌다는 의견 → 포인트로 활용

### 콘텐츠 다양성 원칙
- 메인이 AI 얘기더라도 밥먹는거, 다른 일하는것도 중간중간 보여주기
- **볼거리가 다양한 상태에서 그 안에 메인이 있어야** 드라이하지 않음
- "늘 제일 근본적인 건 풋티지 부족" → 풋티지 확보가 최우선
