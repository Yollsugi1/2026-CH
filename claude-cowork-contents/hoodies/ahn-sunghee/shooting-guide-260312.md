# Hoodies 안성희 — 3/12 촬영 가이드

> **날짜**: 2026-03-12
> **목적**: SET 2 "리와인드" EP.1~4에 들어갈 코멘터리 컷 모아찍기
> **형식**: 크후 스태프와 자연스럽게 대화. 인터뷰 아님.
> **원칙**: 한 블록 10~15분. 흐름이 좋으면 더 가도 됨. 막히면 바로 끊고 다음으로.
> **참고**: EP.4는 독립 에피소드로 남지 않을 수 있음. EP.1~3에 편입될 소재로도 활용.

---

## 이 촬영의 역할 — 성희님께 설명할 때

오늘 찍는 대화들은 **성희님이 직접 찍어오는 브이로그/셀프캠 사이사이에 들어가는 코멘터리**.
셀프캠이 "이 사람의 일상"을 보여준다면, 오늘 대화는 **"이 사람이 무슨 생각을 하며 사는지"**를 들려주는 역할.

SET 2 "리와인드"는 호주에 간 성희님을 본 시청자가 **"이 사람이 한국에서 어떤 사람이었나?"**를 보는 세트. 오늘 이야기들이 그 퍼즐 조각이 된다.

---

## 대화 운영 원칙

- **질문하지 않는다. 대화한다.** "~어떠세요?"보다 "나는 ~인데" 또는 "아 진짜? 그게 뭔데?"
- **성희님이 한 말에서 다음을 꺼낸다.** 준비한 다음 토픽으로 점프하지 않기.
- **리액션이 대화를 만든다.** "대박", "진짜?", "그게 되나?" — 이런 반응이 성희님을 풀리게 한다.
- **신나는 이야기가 나오면 거기서 더 판다.** 플랜에 없는 이야기여도.
- **침묵도 OK.** 생각하는 표정도 좋은 컷.

## Part 1. 대화 갈래 지도

하나의 시작점에서 어디로 흘러갈 수 있는지. 현장에서 흐름 보고 갈래를 선택.

---

### 갈래 A. "인왕산" → 루틴 → 운동 중심의 삶

> **시작**: 이틀 전 같이 다녀온 인왕산 이야기. 가장 자연스러운 출발점.

**흐름:**

"그저께 인왕산 진짜 좋았는데 — 성희님 원래 새벽에 저렇게 뛰는 거예요?"
→ 새벽 운동 습관에 대한 이야기가 나온다
→ "매일 그 시간에 일어나요? 알람 없이?" / "운동 안 하는 날은 없어요?"
→ 이 사람의 하루가 운동 중심으로 돌아간다는 게 드러난다
→ "은행 다닐 때도 이랬어요? 아침에 운동하고 출근하고?"
→ 은행 시절과 지금의 하루가 어떻게 다른지 — 자연스럽게 과거 이야기 진입 가능

**더 갈 수 있는 방향:**
- "트레일이 좋아요 로드가 좋아요?" → 트레일러닝/다이나핏 이야기
- "혼자 뛰는 게 좋아요 같이 뛰는 게 좋아요?" → 크루/사람 이야기 (갈래 D로)
- "호주에서도 이런 코스 있을까요?" → 호주 기대 이야기 (갈래 F로)
- "인왕산 말고 요즘 어디서 뛰어요?" → 최근 러닝/운동 동선 (갈래 D로)

**콘텐츠 연결:**
- → **EP.1 "체질"**: 이 사람의 하루, 운동이 체질인 사람이라는 것. 인왕산 트레일 씬(3/10 촬영분) 뒤에 붙는 코멘터리.
- → **EP.1~3 어디든**: 은행 시절 이야기가 나오면 "왜 떠났는지"의 복선으로 어느 EP에나 쓸 수 있다.

---

### 갈래 B. "은행" → 성취감 → 리셋

> **시작**: 갈래 A에서 자연스럽게 넘어오거나, 독립적으로 시작 가능.

**흐름:**

"은행 다닐 때 어땠어요? 좋았어요?" (가볍게)
→ 성희님이 은행 생활에 대해 이야기한다 — 좋았던 것, 안 맞았던 것
→ "4년이면 꽤 다녔는데, 어느 순간에 '아 이건 아니다' 한 거예요?"
→ **핵심 맥락**: "성장하고 있다는 성취감이 없으면 못 견딘다" — 이 말이 성희님 입에서 나오면 가장 좋다. 안 나오면 안 나오는 대로.
→ "그만둔다고 했을 때 주변에서 뭐라 했어요?"
→ "부모님은요?" (중국에 계시니까 더 복잡했을 수 있다)

**더 갈 수 있는 방향:**
- "그래서 바로 호주를 생각한 거예요?" → 호주 결심 이야기 (갈래 E로)
- "은행에서 배운 게 지금도 남아있는 거 있어요?" → 꼼꼼함, 숫자 습관
- "그때 같이 일하던 사람들이랑 아직 연락해요?" → 사람 이야기 (갈래 D로)

**콘텐츠 연결:**
- → **EP.1~3 어디든**: "이 사람은 원래 이렇게 살아왔다"를 관통하는 맥락. 어느 EP에서든 보이스오버나 코멘터리로 쓸 수 있다.
- → **EP.1 "체질"**: "은행원 때의 아침과 지금의 아침"이라는 대비.

### 갈래 C. "집" → 고정 거처 없는 삶 → 제주도라는 꿈

> **시작**: 현재 어디서 지내는지에 대한 가벼운 대화에서.

**흐름:**

"요즘 어디서 지내요?" (진짜 궁금해서 묻는 톤)
→ 친구 집, 지인 집 이야기가 나온다
→ "그게 불편하지 않아요? 나는 내 방 없으면 좀 힘들 것 같은데"
→ 성희님의 반응 — 원래 그런 스타일인지, 상황이 그런 건지
→ "그러면 서울에서 '여기가 내 공간이다' 하는 데가 있어요? 짐? 카페?"
→ 여러 곳이 조금씩 집이라는 이야기 — 프사오, 단골 카페, 자주 가는 골목

여기서 **제주도 이야기로 가는 갈래가 열린다:**

→ "그러면 언젠가 정착하고 싶은 데가 있어요? 아니면 계속 이렇게?"
→ 성희님이 **제주도** 이야기를 꺼낼 수 있다
→ "아 제주도요? 거기가 좋아요?" — 제주도에 대한 감정을 더 끌어내기
→ "제주도에서 뭐 하고 싶은데요?"
→ **핵심**: 호주에서 전문성(인테리어)을 가져와서 제주도에서 '내 일'을 하고 싶다는 그림
→ "아... 그러면 호주가 그냥 여행이 아니라 제주도를 위한 준비인 거네요?"

이 흐름이 나오면 **EP.3 "집"의 서사가 한 층 깊어진다**:
- 현재: 서울에 고정 거처 없이 떠다니는 삶
- 미래: 제주도라는 꿈 — 거기에 '내 집'과 '내 일'을 만들고 싶다
- 호주: 그 미래를 위해 전문성을 가져오는 여정

**제주도 이야기를 안 꺼내면** 직접 유도할 수 있다:
→ "인왕산 다녀올 때 제주도 좋다고 했잖아요. 그냥 여행으로 좋은 거예요, 아니면 살고 싶은 거예요?"

**더 갈 수 있는 방향:**
- "제주도에서 인테리어 하면 어떤 느낌일 것 같아요?" → 구체적인 미래 이미지
- "그래서 호주에서 인테리어를 배우려는 거구나" → 갈래 E(왜 호주인지)와 합류
- "짐 싸는 건 금방이겠네요" → 출국 준비, 가벼움
- "호주에서 셰어하우스 구했어요?" → 호주 현실 (갈래 G로)

**콘텐츠 연결:**
- → **EP.3 "집"**: 이 갈래 전체가 EP.3의 핵심. "어디도 집이 아닌 사람"에서 시작해서 "제주도라는 미래의 집"까지 — 이 아크가 EP.3을 단순한 '떠돌이' 이야기가 아니라 **방향이 있는 이야기**로 만든다.
- → **EP.1~3 어디든**: "어디서든 편한 사람이라서 호주도 자연스럽다"는 어느 EP에서든 쓸 수 있다.
- → **SET 1 EP.3 "새 손"**: 호주에서 인테리어를 시작하는 장면과 직결. "왜 인테리어인지"의 답이 여기 있다.

**성희님께**: "이 이야기가 나중에 EP.3 '집' 에피소드의 뼈대가 돼요. 성희님이 서울에서 이렇게 지내는 장면들 — 친구 집 아침, 카페, 짐, 골목 — 그 위에 오늘 대화가 깔리는 거예요. 그리고 제주도 이야기가 나오면, 호주가 왜 중요한지까지 자연스럽게 연결돼요."

### 갈래 D. "피트니스 씬" → 여기저기 돌아다니는 사람 → 구체적인 사람들과 에피소드

> **시작**: 최근 경험에서 시작. ON 파티, 박스페이스, 하이록스 짐, 클라이밍, 러닝.
> 이 갈래가 EP.2 "단골"의 핵심 소재 풀.

**흐름 1 — ON 오프닝파티에서 시작:**

"ON 한남 오프닝파티 갔다왔다면서요? 어땠어요?"
→ 파티 자체보다 **거기서 누구를 만났는지, 어떤 분위기였는지**가 중요
→ "거기 어떤 사람들이 왔어요? 운동하는 사람들?"
→ 피트니스 씬의 커뮤니티성이 드러난다 — 브랜드 이벤트에 운동하는 사람들이 모인다
→ "이런 데 자주 가요? 브랜드 행사나 이벤트?"
→ 성희님이 피트니스 씬 안에서 어떤 위치에 있는지

**흐름 2 — 여러 센터 돌아다니는 것에서 시작:**

"요즘 박스페이스도 가고 하이록스 짐도 가고 여기저기 많이 다니던데 — 다 어떻게 알고 가는 거예요?"
→ 각 센터를 어떻게 발견하는지, 누가 추천하는지
→ "가는 데마다 분위기가 다르지 않아요? 어디가 제일 좋아요?"
→ 각 공간의 성격이 나온다. 이 사람은 하나의 짐에 정착하는 게 아니라 **돌아다니면서 각 공간의 에너지를 흡수하는 사람**
→ "거기서도 사람들이랑 바로 친해져요?"
→ 구체적인 사람 이야기가 나오면 더 파기 — 이름, 어떻게 만났는지, 뭘 같이 하는지

**흐름 3 — 클라이밍에서 시작:**

"클라이밍도 한다면서요? 그것도 꽤 오래 했어요?"
→ 성희님에게 클라이밍이 큰 취미였다는 이야기
→ "크로스핏이랑 클라이밍이랑 느낌이 많이 달라요? 쓰는 근육도 다를 것 같은데"
→ "클라이밍은 누구랑 해요? 거기도 크루가 있어요?"
→ 크로스핏 크루, 러닝 크루, 클라이밍 크루 — 운동마다 다른 사람들
→ "성희님은 진짜 운동별로 사람들이 다 있네요. 서울 운동 씬을 다 아는 것 같은데"

**흐름 4 — 러닝에서 시작:**

"요즘 뛰는 데가 인왕산 말고도 있어요? 여기저기 많이 다니잖아요"
→ 어디서 뛰는지, 혼자인지 같이인지
→ "러닝 크루 같은 데 소속되어 있어요?"
→ "뛸 때 아는 사람 만나면 어때요? 서울이 좁은 거예요 아니면 성희님 인맥이 넓은 거예요?"
→ 웃으면서 자연스럽게 인맥 이야기로

**프사오 크루 깊이 파기:**

위 어느 흐름에서든 프사오 이야기가 나오면:
→ "프사오 사람들이랑은 운동 말고도 놀아요?"
→ "그중에 제일 친한 사람이 누구예요?"
→ "걔가 호주 간다니까 뭐라 했어요?"
→ 구체적인 반응, 구체적인 에피소드 — "'나도 데려가' 한 사람 있어요?"
→ "창수코치님이랑은 어떤 사이예요? 코치님이 호주 짐 추천해준 거 있어요?"

**더 갈 수 있는 방향:**
- "발리 갔을 때도 이렇게 짐 찾아다녔어요?" → 갈래 H(발리)로
- "호주 가서도 이렇게 여기저기 드랍인할 거예요?" → 갈래 F(호주 기대)로
- "은행 동기들이랑도 아직 만나요? 운동 안 하는 친구들은?" → 운동 밖 관계
- "이 사람들 두고 가는 게 아쉽지 않아요?" → 두고 가는 관계 감정

**콘텐츠 연결:**
- → **EP.2 "단골"**: 이 갈래 전체가 EP.2의 핵심. 성희님 셀프캠(여기저기서 운동하는 장면)과 오늘 코멘터리가 합쳐서 "이 사람의 운동 생태계"를 보여준다. 프사오만이 아니라 박스페이스, 하이록스 짐, 클라이밍장, 러닝 코스 — 이 모든 곳에 성희님의 사람들이 있다.
- → **EP.1 "체질"**: 여기저기 돌아다니는 것 자체가 "체질"의 일부. 하루 동선 시퀀스에 "왜 한 곳에 안 있고 돌아다니는지"가 깔릴 수 있다.
- → **EP.3 "집"**: 짐이 집이라면, 여러 짐을 돌아다닌다는 건 집이 여러 개라는 것. EP.3의 "어디도 집이고 어디도 집이 아닌" 감각과 연결.

**성희님께**: "이 이야기는 성희님이 직접 찍어올 운동 장면들 사이에 들어가요. 프사오, 박스페이스, 하이록스 짐, 클라이밍, 러닝 — 여기저기서 운동하는 장면 + 오늘 대화 = EP.2 '단골' 에피소드가 됩니다. 성희님이 돌아다니는 곳마다 사람들이 있다는 게 이 에피소드의 핵심이에요."

### 갈래 E. "왜 호주인지" → 완전히 다른 것 → 인테리어/목수

> **시작**: 갈래 B(은행)이나 갈래 C(제주도)에서 이어지거나, 독립적으로.

**흐름:**

"호주는 언제부터 생각한 거예요?"
→ 결심의 과정 — 갑자기? 오래 고민?
→ "같이 가려던 친구가 있었다며요? 걔는 결국 필라테스 쪽으로 간 거예요?"
→ "사무직할 거면 왜 호주까지 가나" — 이 말이 성희님 입에서 나오면 최고. 안 나오면:
"호주 가서도 사무직 할 수 있었을 텐데, 그건 생각 안 했어요?"
→ "완전히 다른 걸 해야 한다"는 논리
→ "그래서 인테리어? 은행원이 목수 된다는 게 좀 극단적이잖아요"
→ 인테리어에 대한 기대 — 손으로 만드는 것, 눈에 보이는 결과물

**갈래 C(제주도)와 합류하는 순간:**

→ 인테리어 이야기가 나오면: "그 기술 배워서 한국 돌아오면 어디서 써먹을 거예요?"
→ 제주도 이야기가 나올 수 있다 — "제주도에서 내 일을 하고 싶다"
→ "아... 그러면 호주가 그냥 경험이 아니라, 제주도를 위한 준비인 거예요?"
→ 호주 → 전문성 → 제주도라는 **장기 플랜**이 드러난다

**더 갈 수 있는 방향:**
- "손으로 뭔가 만들어본 적 있어요? 이케아 조립이라도?" → 웃기는 방향
- "호주에서 인테리어 일 구하는 건 어렵지 않아요?" → 현실 (갈래 G로)

**콘텐츠 연결:**
- → **EP.3 "집"**: 인테리어 + 제주도 이야기가 합쳐지면 EP.3의 가장 깊은 층이 된다. "집이 없는 사람"이 "미래의 집을 직접 지으려 한다"는 아크.
- → **EP.1~3 어디든**: "왜 떠나는지"는 어느 EP에서든 깔릴 수 있는 맥락.
- → **SET 1 EP.3 "새 손"**: 호주에서 인테리어 시작하는 장면의 복선.

**성희님께**: "이 이야기는 나중에 호주에서 인테리어 시작하는 장면과 이어져요. 그리고 제주도 이야기까지 나오면, 호주가 왜 중요한지가 훨씬 깊어집니다."

### 갈래 F. "호주에서 하고 싶은 것들" → 버킷리스트 → 설렘과 두려움

> **시작**: 어떤 갈래에서든 자연스럽게 넘어올 수 있는 토픽.

**흐름:**

"호주 가면 제일 먼저 뭐 할 거예요?"

"하이록스 브리즈번이요? 한국이랑 다를 것 같은데"
"아 거기 Parkrun이라는 게 있어요? 매주 무료로?"
→ UTMB 50K 이야기가 나오면:
"50K를 산에서요? 그건 좀... 무섭지 않아요?"
→ 솔직한 두려움이 나올 수 있다 — "이건 좀 걱정돼요" 같은 말
→ "연습은 어떻게 해요? 한국에서 트레일 뛸 데가 있어요?"

**더 갈 수 있는 방향:**
- "거기 짐은 어떻게 찾아요?" → 호주 크루 만들기
- "호주 피트니스 문화가 한국이랑 많이 다를 것 같아요?" → 야외 짐, 비치 부트캠프
- "마라톤은 호주에서도 뛸 거예요?" → 동아마라톤(3/15) 연결
- "발리에서도 짐 찾아다녔다며요? 그때 어땠어요?" → 갈래 H(발리)로

**콘텐츠 연결:**
- → **EP.1~3 어디든**: 버킷리스트 = 앞으로 펼쳐질 SET 3~5의 메뉴판. 어느 EP에서든 "다음에 뭘 할 건지"로 쓸 수 있다.
- → **EP.1 "체질"**: 인왕산/트레일 이야기로 돌아가면 EP.1 소재.

**성희님께**: "호주에서 하고 싶은 것들을 이야기해주시면, 나중에 호주 에피소드들의 예고편이 돼요. 시청자가 '어 저거 진짜 하나?' 하면서 따라오게 되는 구조예요."

### 갈래 G. "현실" → 비자, 돈, 집 → 꼼꼼함과 불안

> **시작**: 호주 이야기 중 현실적 측면이 나올 때.

**흐름:**

"근데 현실적으로 — 비자, 돈, 집은 좀 준비했어요?"
→ 워홀 비자 1년 → 2년 연장하려면 88일 농장이라는 구조
→ "농장 88일이요? 그건 어때요?" → 농장에 대한 기대 or 걱정
→ "은행원이라 이런 거 계획 잘 세울 것 같은데" → 꼼꼼함 발동
→ "이걸 원으로 치면..." 하는 은행원 습관 — 호주 물가 이야기

**더 갈 수 있는 방향:**
- "혼자 가는 거잖아요. 그게 괜찮아요?" → 혼자에 대한 감정
- "최악의 시나리오는 뭐예요?" → 솔직한 불안
- "1년 후에 어떤 사람이 되어 있을 것 같아요?" → 미래 이미지

**콘텐츠 연결:**
- → **EP.1~3 어디든**: 꿈과 현실 사이의 균형. 어느 EP에서든 현실감 있는 코멘터리로 쓸 수 있다.
- → **SET 6 "88일"**: 나중에 농장 에피소드가 나올 때, "한국에서 이미 이런 이야기를 했었다"는 연결.

### 갈래 H. "발리" → 여행지에서 운동하기 → 어디서든 연결되는 사람

> **시작**: 갈래 D(피트니스 씬)이나 갈래 F(호주 기대)에서 자연스럽게 넘어올 수 있다. 또는 독립적으로.

**흐름:**

"발리 다녀왔다면서요? 거기서도 운동했어요?"
→ 발리에서 크로스핏 박스 갔던 이야기가 나온다
→ "여행 가서 짐부터 찾은 거예요? 원래 그래요?"
→ "거기 분위기가 한국 짐이랑 많이 달랐어요?"
→ 발리 짐에서 만난 사람들, 문화 차이, 재밌었던 순간
→ "그러면 호주 가서도 똑같이 할 거네요. 도착하자마자 짐부터 찾고"

**더 갈 수 있는 방향:**
- "발리에서 같이 운동한 사람들이랑 아직 연락해요?" → 운동이 만드는 글로벌 인맥
- "발리 말고 여행 가서 운동했던 데 또 있어요?" → 이 사람의 여행 패턴
- "거기서 짐 찾을 때 어떻게 찾아요? 인스타? 구글?" → 실용적이면서 재밌는 디테일
- "친구들이랑 같이 갔는데 성희님만 짐 갔어요?" → 웃기는 에피소드 가능

**콘텐츠 연결:**
- → **EP.2 "단골"**: 서울 밖에서도 운동으로 사람을 만드는 패턴. 한국 크루 셀프캠 + 발리 썰 = "이 사람은 어디서든 운동으로 연결된다."
- → **EP.1 "체질"**: 여행 가서도 짐부터 찾는 건 "체질" 그 자체.
- → **SET 1 EP.2 "첫 번째 WOD"**: 호주에서 드랍인하는 장면의 복선. "이 사람은 발리에서도 이랬다 — 호주에서도 이럴 것이다."

**성희님께**: "발리에서 짐 갔던 이야기가 나중에 호주에서 처음 짐 가는 장면이랑 이어져요. '이 사람은 어딜 가든 이렇게 한다'는 걸 보여주는 소재예요."

## Part 2. 독립 대화 시작점

갈래 지도와 별개로, 흐름이 끊겼을 때 **새로 꺼낼 수 있는 시작점들**.

---

### EP.2 "단골" 소재 — 사람들과 공간들

**최근 경험에서 꺼내기:**

- "ON 한남 오프닝파티 어땠어요? 거기 어떤 사람들이 왔어요?"
  → EP.2: 피트니스 씬의 커뮤니티성. 브랜드 이벤트에 모이는 운동하는 사람들.

- "요즘 박스페이스 다녀봤다면서요? 거기 분위기 어때요?"
  → EP.2: 여러 센터의 각기 다른 분위기. 한 곳에 안 머무르는 사람.

- "하이록스 짐도 갔어요? 거기서 뭐 했어요?"
  → EP.2: 하이록스 준비도 되고, 공간 탐방도 되는 이야기.

- "클라이밍 요즘도 자주 해요? 원래 엄청 좋아했다면서요"
  → EP.2: 크로스핏 외의 운동. 이 사람의 운동 스펙트럼이 넓다.

- "클라이밍은 어디서 해요? 거기서도 단골이에요?"
  → EP.2: 클라이밍장에서의 사람들. 운동마다 다른 커뮤니티.

- "러닝은 요즘 어디서 뛰어요? 혼자예요?"
  → EP.2: 러닝 동선과 러닝 크루.

**구체적인 사람 끌어내기:**

- "요즘 제일 자주 보는 사람이 누구예요?"
  → EP.2: 핵심 인물. 이름, 관계, 에피소드.

- "프사오 말고 다른 데서 만난 사람 중에 기억나는 사람 있어요?"
  → EP.2: 프사오 밖의 인맥. 운동 생태계의 넓이.

- "호주 간다고 하니까 제일 아쉬워하는 사람이 누구예요?"
  → EP.2/EP.3: 관계의 깊이. 두고 가는 사람들.

- "같이 운동하다가 친해진 사람 중에 재밌는 케이스 있어요? 예상 밖으로?"
  → EP.2: 에피소드. 운동이 만드는 예상 밖의 만남.

**관계의 패턴 끌어내기:**

- "새로운 짐 가면 사람들이랑 금방 친해져요? 아니면 좀 걸려요?"
  → EP.2: 이 사람이 관계를 만드는 방식. 호주에서도 통할지.

- "운동 크루끼리는 뭐가 달라요? 크로스핏 사람들, 러닝 사람들, 클라이밍 사람들?"
  → EP.2: 각 커뮤니티의 문화 차이. 성희님은 그 사이를 넘나드는 사람.

- "운동 안 하는 친구들이 성희님 보면 뭐라 해요?"
  → EP.2: 외부 시선. "너 매일 운동만 하냐" 같은 반응.

### EP.3 "집" 소재 — 공간과 소속감

**현재의 삶에서 꺼내기:**

- "요즘 어디서 지내요? 매일 같은 데예요?"
  → EP.3: 고정 거처 없는 삶의 실제 모습.

- "아침에 일어나면 '여기 누구 집이지?' 하는 순간 있어요?"
  → EP.3: 남의 공간에서 사는 감각. 유머러스하게 갈 수도 있고, 진지하게 갈 수도 있다.

- "짐이 여러 개잖아요. 그중에 '여기가 내 공간이다' 하는 데가 있어요?"
  → EP.3: 짐 = 집. 프사오인지, 박스페이스인지, 아니면 그때그때 다른지.

- "서울에서 제일 편한 장소가 어디예요? 카페? 짐? 산?"
  → EP.3: "집" 대신 "편한 곳"으로 접근하면 더 자연스러울 수 있다.

- "부모님이 중국에 계시잖아요. 명절 같은 때는 어떻게 해요?"
  → EP.3: 가족과의 거리. 물리적으로 떨어져 사는 게 익숙한 사람.

**제주도와 미래에서 꺼내기:**

- "제주도 자주 갔어요? 뭐가 그렇게 좋아요?"
  → EP.3: 제주도에 대한 구체적인 감정. 어떤 제주도를 좋아하는지.

- "제주도에서 살면 매일 뭐 할 것 같아요?"
  → EP.3: 구체적인 미래 이미지. 러닝? 서핑? 인테리어 작업? 카페?

- "제주도에서 '내 일'을 한다는 게 구체적으로 어떤 거예요?"
  → EP.3: 호주에서 가져올 전문성 → 제주도에서의 삶. 이 연결이 EP.3의 깊이.

- "서울은 결국 떠날 곳이에요? 아니면 돌아올 곳이에요?"
  → EP.3: "집"에 대한 가장 본질적인 질문.

### 일상/루틴 계열 (EP.1 "체질")

- "아침에 일어나면 제일 먼저 뭐 해요? 운동? 커피?"
  → EP.1: 하루의 시작. 인왕산/프사오 시퀀스와 연결.

- "하루에 운동 두 번 하는 날도 있어요? 그게 힘들지 않아요?"
  → EP.1: 하루에 운동 두 번 = 강력한 캐릭터 포인트.

- "쉬는 날은 뭐 해요? 진짜 아무것도 안 하는 날?"
  → EP.1/EP.3: 이 사람의 "오프"가 어떤 모습인지.

- "요즘 뭐 보고 있어요? 유튜브든 넷플릭스든. 포테이토터틀 좋아한다면서요?"
  → EP.1: 운동 외의 관심사.

### 과거/전환 계열 (EP.1~3 어디든)

- "은행에서 제일 좋았던 게 뭐였어요? 하나도 없었어요?"
  → 좋았던 게 있었는데 떠났다면 더 강한 서사.

- "그만두기로 한 날이 기억나요? 그날 뭐 했어요?"
  → 구체적인 순간. "결정의 날"이 있었는지, 점진적이었는지.

- "주변에서 말렸어요? 아니면 '너답다' 했어요?"
  → 주변 반응 = 이 사람의 캐릭터를 보여주는 거울.

- "서른이라는 나이가 좀 의미가 있었어요? 아니면 상관없었어요?"
  → 나이와 전환의 관계.

### 호주/미래 계열 (EP.1~3 어디든)

- "호주에서 제일 기대되는 거 하나만 꼽으면?"
  → 가장 본능적인 대답이 나온다.

- "겁나는 건 없어요? 솔직하게"

- "1년 후에 어떤 사람이 되어 있을 것 같아요?"
  → 미래 이미지. SET 7 "역환율"(귀국편)과의 대비 소재.

- "호주에서 짐 찾으면 거기가 바로 '내 동네'가 되는 거예요?"
  → EP.3 "집": "짐 = 집"이라는 이 사람만의 공간 감각.

### 운동 깊이 계열 (EP.1~2)

- "크로스핏이랑 트레일러닝이랑 클라이밍이랑 — 다 다르잖아요. 뭐가 제일 좋아요?"
  → EP.1/EP.2: 운동 종목별 매력. 다이나핏 앰배서더 이야기.

- "대회 나갈 때 준비 어떻게 해요? 계획 세우는 편이에요?"
  → EP.1~3: 은행원의 꼼꼼함 — 페이스, 보급, 스프레드시트. 동아마라톤(3/15) 대비.

- "여행 가면 꼭 짐이나 러닝 코스부터 찾아요?"
  → EP.1/EP.2: 발리, 제주도 등에서의 운동. 이 사람이 도시를 탐색하는 방식.

- "기록에 집착하는 편이에요? 아니면 완주가 중요한 편?"
  → EP.1~3: "성취감 중독자"의 구체적인 모습.
