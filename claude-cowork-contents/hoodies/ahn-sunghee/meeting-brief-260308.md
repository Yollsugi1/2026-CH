## 1. Hoodies 시리즈 소개

### 한 줄

> "운동이 삶의 중심인 입체적인 사람들의 세계를, 크후 크루가 함께 경험하며 풀어내는 브이로그"

### 포맷

- **롱폼 브이로그**: 에피소드당 7~12분
- **21:9 시네마틱 비율**: 화면은 영화처럼, 내용은 가볍게
- **3EP 세트 구조**: 하나의 이야기/시기를 3개 에피소드로 나눔. 마치 여행 유튜브에서 한 나라를 EP.1~3으로 담는 것과 같은 구조

### 톤 레퍼런스

**노홍철 브이로그** + **포테이토터틀** 스타일

- 껍데기는 시네마틱하고 멋있게 (21:9, 좋은 카메라)
- 속은 자연스럽고 가볍게 — 스태프들과 함께 움직이면서 대화, 썰풀기
- 인터뷰 X, 연출 X — 그냥 같이 가서 같이 경험하면서 찍는 방식

### 찍히는 방식

- **크후 스태프가 동행해서 찍어주는 장면** — 운동, 일상, 이동 등
- **본인 셀프캠 장면** — 혼자 있는 시간, 호주에서의 일상 (특히 호주 세트들)
- **대화 코멘터리** — 제작진과 자연스럽게 나누는 이야기가 영상 흐름을 만든다
- 10~15분 텀으로 자연스럽게 끊어가며 진행. 억지로 이야기 만들 필요 없음

## 2. 안성희 편 전체 큰 그림

> **중요: 촬영 순서 ≠ 업로드 순서**
> SET 2(한국 라이프스타일) 내용을 먼저 촬영하지만, 시청자에게는 SET 1(새출발)이 먼저 공개된다.
> 즉, 지금 3/10~3/17에 찍는 것들은 SET 2로 나중에 나오는 것들이다.

| 세트 | 타이틀 | EP | 업로드 순서 | 촬영지 | 핵심 |
|------|--------|----|------------|--------|------|
| SET 1 | "제2의 삶" | 3EP | 먼저 공개 | 한국→호주 | 출국 + 호주 도착 + 첫 주 (새출발) |
| SET 2 | 리와인드 | 4EP | 두 번째 공개 | 한국 (먼저 촬영) | 안성희 한국 라이프스타일 + 동아마라톤 |
| SET 3 | "주파수" | 3EP | — | 호주 전역 | 피트니스 문화 탐험 + UTMB 준비 |
| SET 4 | "50K" | 3EP | — | 호주 산악 | UTMB 울트라트레일 |
| SET 5 | "88일" | 3EP | 미정 | 호주 시골 | 농장 서바이벌 (비자 연장 시) |
| SET 6 | "역환율" | 3EP | 미정 | 한국 | 귀국 |

> SET 5, 6은 호주 상황 보고 유동적으로 결정. 지금은 SET 1, 2 기획이 집중 포인트.

## 3. SET 1 — "제2의 삶" 상세 설명

> **한국→호주 새출발 편 | 3EP | 촬영: 3월 말 (출국 전후)**
> 업로드 순서상 가장 먼저 공개되는 세트. 시청자가 처음 만나는 안성희는 "호주에 막 도착한 안성희"다.

### 컨셉

안성희가 한국을 떠나 호주에서 새로운 삶을 시작하는 과정을 담는 세트.
"왜 갔나", "어떻게 갔나", "가서 뭐할건가"를 비행기 안에서부터 호주 첫 주까지 자연스럽게 풀어낸다.

비행기 안에서 시작하는 "72시간"처럼 — 공항으로 가는 길, 이륙, 브리즈번 도착까지를 하나의 플로우로.
무겁지 않게. 새 출발의 설렘이 중심.

---

### EP.1 "출발" — 동아마라톤 그리고 공항

**촬영**: 3/15 (동아마라톤) + 3월 말 출국 당일

한국에서의 마지막 큰 이벤트 → 출국으로 이어지는 흐름.

**담아야 할 순간들:**
- 동아마라톤 완주 (한국에서의 마지막 큰 도전) — 결승선 표정, 기록 확인
- 마라톤 후 며칠: 짐 싸기, 마지막 인사
- 공항 출국장 → 게이트 → 비행기 탑승
- 비행기 안: 이 선택에 대해 혼자 정리하는 시간 (나레이션 or 셀프캠)

**감정**: 마무리와 출발이 동시에. "끝이 아니라 시작"의 느낌.

---

### EP.2 "도착" — 브리즈번, 뭐할건가

**촬영**: 호주 도착 첫 며칠 (셀프캠 중심)

브리즈번에 내리는 순간부터. 핵심은 "이 사람은 도착하자마자 짐부터 찾는다."

**담아야 할 순간들:**
- 공항 → 숙소 셋업 → 러닝화 꺼내는 순간 (이 사람의 우선순위)
- 앞으로의 계획 이야기: 하이록스(4월), UTMB(5월), 풀마라톤, 인테리어 일, 가보고 싶은 짐들
  - "이건 좀 무서워"처럼 솔직한 반응이 섞이면서
  - 시청자가 다음 세트들을 기대하게 되는 자연스러운 예고편
- 크후에게 첫 화상통화: "여기 어때요?"

**감정**: 설렘. 새 나라에 착륙한 첫날의 에너지.

---

### EP.3 "첫 번째 WOD" — 호주 첫 주

**촬영**: 호주 도착 첫 1주 (셀프캠 + 원격 디렉팅)

브리즈번에서 첫 번째 드랍인. 언어도 다르고 짐도 다르지만 — 바벨 앞에선 그냥 몸이 움직인다.

**담아야 할 순간들:**
- 첫 드랍인: "Hi, I'm here for drop-in" — 영어로 WOD 설명 듣는데 몸은 알아서 움직이는 순간
- 첫 러닝 코스: 구글맵 보며 뛰기 좋은 데 탐색. 한강과 완전히 다른 풍경
- 인테리어 일 첫 발: 면접 or 첫 출근. 금융 보고서 쓰던 손이 목재를 만지는 순간
- "이걸 원으로 치면..." 하는 은행원 습관이 호주 물가 앞에서

**감정**: 어색하지만, 바벨을 잡는 순간 — "여기서도 나는 나다"라는 안도감. SET 1 마무리.

## 4. SET 2 — 리와인드 상세 설명

> **한국 라이프스타일 편 | 4EP | 촬영: 3/10~3/17 (먼저 찍음)**
> 업로드 순서상 두 번째. SET 1에서 호주에 있는 안성희를 본 시청자가, 리와인드로 "그래서 이 사람이 원래 어떤 사람이었나?"를 보게 된다.

### 컨셉

SET 1에서 "이 사람, 호주 갔구나"를 보여줬다면, SET 2는 리와인드 — "그래서 한국에서 어떤 사람이었나?"
소개하지 않는다. 설명하지 않는다. 이 사람의 한국 일상을 따라가면 시청자가 알게 된다.

핵심: **"출발 전의 마지막 시간"이 아니라 "이 사람이 원래 이렇게 산다"**를 보여주는 것.
호주를 가든 안 가든 이렇게 살았을 사람이라는 게 더 강하다.

촬영은 먼저 하지만, 시청자 입장에선 "아, 저 사람이 한국에서는 이런 사람이었구나" 하면서 보게 되는 세트.

---

### EP.1 "아침 6시" — 안성희의 하루

**촬영일**: 3/10 또는 3/12 (평일)

안성희의 평범한 하루를 처음부터 끝까지. "평범"이 핵심 — 이 사람에게 평범한 하루가 보통 사람에겐 전혀 평범하지 않다.

**담아야 할 순간들:**
- 아침 루틴: 일어나서 커피, 스트레칭, 핸드폰 — 은행원 때와 지금이 어떻게 다른지
- 오전 운동: 프사오 수유 WOD. 운동 자체보다 **운동하는 공간에서 성희님이 어떻게 존재하는지** — 도착할 때 인사, 크루들과의 케미, WOD 끝나고 바닥에 누워서 하는 대화
- 점심: 운동 후 식사, 크후 크루와 가볍게 대화
- 오후: 호주 준비(비자, 짐) 또는 다른 만남
- 저녁: 러닝 또는 두 번째 운동 (하루에 운동 두 번이면 강력한 캐릭터 포인트)

**감정**: 밝고 에너지 넘침. "이 사람, 좋다" — 이 한마디가 나오면 성공.

---

### EP.2 "단골" — 성희의 사람들

**촬영일**: 3/12 또는 3/17

안성희가 매일 만나는 사람들을 통해 이 사람을 읽는 에피소드.
사람이 어떤 사람인지는 그 사람의 말이 아니라 **주변 사람들의 반응**에서 나온다.

**담아야 할 순간들:**
- 프사오 크루와의 시간: "성희 진짜 호주 가?" "나도 데려가" 같은 자연스러운 반응
- 짐 코치/멤버와의 대화: "호주 가서 짐 어디 다닐 거예요?" → 호주 피트니스 이야기로 자연스럽게
- 운동 외 만남: 대학 친구, 은행 동기, 저녁 식사 — 과거 이야기가 자연스럽게
- 혼자인 순간: 모든 사람 만나고 집에 가는 길

**감정**: 따뜻함. 이 사람은 혼자 떠나지만 혼자가 아니라는 것.

---

### EP.3 "리스트" — 호주 버킷리스트

**촬영일**: 3/12 또는 3/17

안성희와 크후 크루가 앉아서, 호주에서 하고 싶은 것들을 이야기하는 시간.
리와인드의 맥락에서 이 에피소드를 보면 — "아, 한국에 있을 때부터 이미 이런 계획이 있었구나." 더 선명해진다.

**담아야 할 순간들:**
- 버킷리스트 하나씩 꺼내기 (하이록스, UTMB, 풀마라톤, 인테리어, 피트니스 탐방)
- 크루 리액션: "50K는 진짜예요?" — 시청자의 궁금증을 대신하는 반응
- "그러고 보니 마라톤이 2주 뒤네" — 자연스럽게 EP.4로 연결

**감정**: 설렘 + 약간의 긴장. 리와인드로 보면 "다 이뤘잖아"라는 뿌듯함도 겹친다.

---

### EP.4 "42.195" — 동아마라톤

**촬영일**: 3/15 (대회 당일)

SET 2 리와인드의 클라이맥스. 한국에서의 마지막 큰 이벤트.
시청자는 SET 1에서 이미 "그래서 호주에서 더 잘 뛰었나?"를 알고 있는 상태에서 이 마라톤을 본다 — 그 레이어가 감정을 더 깊게 만든다.

**담아야 할 순간들:**
- 전날 준비: 은행원의 꼼꼼함 — "이 사람은 달리기도 스프레드시트로 한다"
- 새벽 출발 + 스타트라인
- 레이스 중간 변화: 초반 여유 → 중반 고통 → 후반 멘탈
- 크후 크루 응원 포인트 장면
- 결승선 완주 순간 + 직후 표정
- 마라톤 후 마지막 장면: "이제 진짜 가는구나" — SET 2(리와인드)의 마무리이자 SET 1(새출발)로 자연스러운 연결

**감정**: SET 2 마무리. "이걸 마치고 떠났구나" — 시청자가 이 사람의 한국 시절을 정리하는 순간.

## 6. 촬영 관련 안내사항

**현장에서 꼭 전달할 것들:**

- **자연스럽게 끊어가기**: 10~15분 텀으로 자연스럽게 멈춰서 쉬거나 방향 전환. 무리하게 이어갈 필요 없음
- **인터뷰 형식 아님**: 카메라 보고 말하는 방식 X. 그냥 같이 움직이고, 같이 대화하면서 찍는 것
- **셀프캠 호주편**: 호주에 크후 스태프가 매번 갈 수 없어서, 핵심 이벤트(UTMB, 하이록스)는 현장 투입, 나머지는 성희님 셀프캠 + 원격 디렉팅으로 진행 예정
- **티저 먼저**: 첫 번째로 올라갈 것은 나레이션 방식의 숏폼 티저. 이건 별도로 기획 예정
- **성희님 직접 유튜브 하고 싶었던 분** 이니까, 셀프캠 촬영에 대해 편하게 이야기해보기
