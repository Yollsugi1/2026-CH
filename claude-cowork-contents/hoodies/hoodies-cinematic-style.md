# Hoodies 시네마틱 스타일 가이드

> 이 문서는 정답이 아니라, 레퍼런스와 원칙을 누적하는 살아있는 문서다.
> 촬영과 편집을 거듭하면서 계속 업데이트된다.

---

## 핵심 원칙

### "껍데기는 영화, 속은 썰풀기"

화면은 시네마틱하고 멋있게 — 내용은 자연스럽고 가볍게.
이 이중성이 Hoodies 연출의 핵심이다.

- **화면**: 21:9 비율, 정제된 색감, 멈추지 않는 컷
- **내용**: 연출된 인터뷰 X. 같이 움직이고 대화하면서 나오는 날것의 반응

### 하나의 에피소드 = 다양한 공간·컷의 직조

하나의 에피소드가 하나의 이벤트를 담는 구조가 아니다.
여러 공간과 상황이 자연스럽게 전환되며 하나의 무드·주제를 만드는 구조.

> 예: "아침 6시" 에피소드 = 불암산 새벽 → 짐 WOD → 카페 점심 → 저녁러닝
> 이 4개의 공간이 편집으로 연결되며 "이 사람의 하루"를 보여준다.

이 원칙은 어떤 에피소드에나 적용된다.
불암산에 갔다고 해서 불암산 에피소드가 되는 게 아니다.
불암산은 그날 에피소드의 한 컷이다.

## 샷 원칙

### 비율 & 카메라

- **21:9 기본**: 모든 주요 장면은 21:9 시네마틱 비율
- **핸드헬드**: 이동, 대화, 현장 반응 — 흔들림이 오히려 현장감
- **짐벌/고정**: B-roll, 풍경, 공간 전경 — 안정적이고 압도적으로
- **클로즈업 vs 와이드**: 감정적 순간 → 클로즈업 / 공간 선언 → 와이드 (21:9 진가)

### 컷 원칙

- **한 공간에서 오래 머물지 않는다**: 3~5분 이상 같은 공간이면 공간 전환 컷 삽입
- **전환은 자연스럽게**: 암전, 자막 타이틀 없이 — 소리나 움직임으로 이어지게
- **대화 없는 시퀀스 계획**: 각 에피소드에 숨소리, 발소리, 도시 소음만으로 1~2분 흐르는 구간 필수
  - 이 구간이 시청자가 숨을 고르고 공간을 느끼는 시간
  - 음악이 들어오는 순간

### 색감 & 빛

- **시간 흐름 = 색온도 흐름**: 새벽 블루→흰빛 → 낮 선명 → 저녁 오렌지→황금
- **골든 아워 적극 활용**: 새벽(일출 전후), 저녁(일몰 전후) — 21:9로 극대화
- **짐 내부 vs 야외**: 짐 인공조명(강하고 날카로운) vs 야외 자연광(부드럽고 넓은) 대비

---

## 사운드 원칙

- **다이제틱 사운드 우선**: 현장음 먼저 — 바벨 소리, 숨소리, 도시 소음, 카페 BGM
- **음악은 감정 전환점에**: 공간이 바뀌거나 감정이 전환될 때 음악 삽입
- **나레이션은 최소**: 화면이 설명하도록. 나레이션이 필요하면 인터뷰 아닌 독백 스타일

## 에피소드 구성 패턴

Hoodies 각 에피소드가 따르는 기본 리듬:

```
[오프닝 컷 — 공간 선언]
→ [첫 번째 공간 / 상황]
→ [공간 전환 — 이동 컷]
→ [두 번째 공간 / 상황]
→ [대화 없는 시퀀스 — 음악]
→ [세 번째 공간 / 상황]
→ [클로징 — 감정 마무리]
```

각 에피소드의 "렌즈"(어떤 관점으로 피사체를 읽는가)를 먼저 정하고,
그 렌즈 아래 여러 공간이 연결되도록 구성.
