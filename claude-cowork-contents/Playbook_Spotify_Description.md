# Playbook — Spotify Podcast Description 후보

## 옵션 A (553자)
스포츠를 좋아하는 사람들이 모여서 스포츠 '말고' 다른 이야기를 합니다.

스포츠와 음악, 스포츠와 팬덤, 스포츠와 돈, 스포츠와 미디어 — 경기 결과나 선수 분석 대신, 스포츠가 우리 삶과 문화에 닿아 있는 지점들을 꺼내봅니다. 풀경기 대신 하이라이트만 보는 시대에 '진짜 팬'이란 뭔지, 왜 NBA에는 힙합이고 축구에는 록인지, 직관 한 번에 15만원을 쓰면서도 또 가게 되는 이유는 뭔지.

매 에피소드마다 하나의 큰 주제를 잡고, 각자의 경험과 생각을 나누며, 가끔은 쓸데없이 진지해지기도 합니다.

CH MAG이 만드는 스포츠 라이프스타일 토크. Playbook.

## 옵션 B (498자)
경기 결과 말고, 스포츠 이야기.

Playbook은 스포츠를 좋아하는 사람들이 모여 스포츠가 우리 삶에 닿아 있는 순간들을 이야기하는 토크쇼입니다. 오디션 프로그램은 왜 스포츠 리그와 닮았는지, 넷플릭스 한 편이 어떻게 36만 명의 새 팬을 만드는지, 경기장에서 터지는 떼창은 왜 그렇게 소름 돋는지.

매회 하나의 주제를 깊게 파고, 각자의 경험을 나누고, 결론 없이 끝나기도 합니다.

스포츠 라이프스타일 매거진 CH MAG이 만듭니다.

## 옵션 C (434자)
스포츠 좋아하는 사람들의 수다.

Playbook은 스포츠를 다른 영역과 엮어서 이야기하는 토크쇼입니다. 팬덤, 음악, 미디어, 돈 — 매 에피소드 하나의 큰 주제를 잡고, 세부 꼭지를 나눠서 이야기합니다. 정답을 찾기보다 각자의 시선을 나누는 대화.

하이라이트만 보는 시대에 풀경기를 틀어놓고 맥주 마시며 떠드는 느낌. 그런 자리입니다.

CH MAG이 만드는 스포츠 라이프스타일 토크.
