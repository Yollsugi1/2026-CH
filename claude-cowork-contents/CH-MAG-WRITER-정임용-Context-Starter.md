## 세팅 현황

### ch-mag-writer 스킬이 설치되어 있다

나는 CH MAG 에디토리얼 워크플로우 전체를 담당하는 `ch-mag-writer` 스킬이 내 Claude에 설치되어 있어.
이 스킬은 아래 5단계 워크플로우를 수행해:

```

Phase 3: Draft 작성
```

스킬 안에는 다음 참고 파일들이 포함돼 있어:

## CH MAG 스타일 핵심 요약

Claude가 내 스타일을 빠르게 이해할 수 있도록:

**톤:** "아는 사람이 쿨하게 해석하는" — 팬심도 아니고 중립 저널리즘도 아닌

**인트로:** 선언형 훅 OR 맥락→각도 전환. 구조 예고 금지, 각도만 던지고 끝냄

**섹션 헤더:** 설명적 라벨이 아닌 시적 표현
- "축구 유니폼, 예술이 되다" (O) / "21/22 시즌 홈 킷" (X)
- "드레이크가 구해낸 베네치아의 영혼" (O) / "베네치아 FC 재정 위기" (X)

**문장 리듬:** 긴 묘사 문장 → 짧고 단호한 마무리 문장 반복

**아웃트로:** 인트로 각도를 완성하는 한 문장. 새 정보 없음

**금지:** "여러분", "앞으로도 기대해봅시다", 독자 직접 호명

## 피드백 누적 현황 (v1.0)

### 확인된 스타일 원칙 ✓
- 선언형 훅으로 시작 (발렌시아가 글의 "패션은 문화를 빨아들이는 블랙홀이다." 스타일)
- 인트로는 각도만 던지고 끝내기 — 구조 예고 금지
- 섹션 헤더는 시적 표현
- 긴 묘사 문장 → 짧고 단호한 마무리 문장의 반복 리듬
- 아웃트로는 인트로 각도를 완성하는 한 문장
- 구체적 수치와 이름으로 글의 무게 잡기
- 스포츠 × 문화 교차점 — 단순 소개 글이 아닌 것

### 구조적 원칙 (참고, 절대 법칙 아님)
- 단순하고 가벼운 소재 앞 → 서사적으로 의미 있는 소재 뒤
- 소재마다 어울리는 구조가 다를 수 있음
