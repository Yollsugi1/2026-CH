## KEY_EXCHANGES

### Playbook 기획의도 (핵심 DNA)
> Playbook 컨셉: "본업은 다르지만 스포츠가 삶의 중심인 세 사람이, 세상 모든 것을 스포츠의 렌즈로 보는 토크쇼"
> 대화 스타일: "흑백요리사를 보면서 '이건 완전 플레이오프 토너먼트 구조잖아'", "연애 프로그램을 보면서 '저건 완전 FA 시장이야'"
→ implication: 순수 스포츠 토크 아님. 스포츠를 렌즈로 세상을 읽는 것이 핵심.

### "뻔한 주제" 거절
> user: 스포츠 명장면 월드컵 이런건 너무 뻔하고, 우리가 해야할 이유가 없어. 재미도 없을거야.
→ implication: 아무 스포츠 팟캐스트나 할 수 있는 주제 전부 탈락. "크리에이터후드만의 이유"가 필요.

### 1차 일반 토크 주제 거절
> user: 이거로 뭔가 주제가 40분 정도 떠들만큼 잘 나올 것 같지 않아. 레퍼런스로 우리가 'B주류초대석' 같은걸 잡고 있는데, 여기서 하나의 주제를 어떻게 확장하는지를 참고해봐도 좋을 것 같아.
→ implication: 1차 제안(중독/소비/정체성, 운동이 당신을 말해준다, 운동장에서 배운 것 등)은 넓고 얇았다. B주류초대석처럼 하나의 문화 현상을 깊이 파서 40분을 채우는 구조가 필요. 여러 축으로 분산 X → 하나를 디깅 O.

### B주류초대석 분석 결과
> 핵심 패턴: (1) 도발적 명제를 던진다 (2) 하나의 문화 현상을 깊이 판다 (3) 출연자의 진짜 경험에서 나온다 (4) 대결/토론 구도
> 에피소드 예: "요즘 연프에는 로맨스가 없다", "게임 안 하는 시대", "재무제표로 읽는 패션계 결산"
→ implication: 제목 자체가 도발적 명제. 하나를 파되, 개인→사회→경제→문화로 깊어지는 구조.

### 주제 제시 방식
> user: 하는 스포츠 방향 잡았더라도 여러가지 주제를 제안해줘
→ implication: 하나를 고정하지 말고 선택지를 줘라.

## 현재 유력 후보

### 일반 토크 주제 (2차 — B주류초대석 구조 적용)

**1. "운동 뭐 하세요? — 이 질문이 당신에 대해 말해주는 것들"**
명제: 운동 선택이 성격/계급/정체성을 드러내는 시대.
흐름: "운동 뭐 하세요?"의 사회적 의미 → 수현 크로스핏 vs 임용 클라이밍 비교 → 운동별 스테레오타입 토론 → 운동이 패션이 된 시대(고프코어, 아크테릭스, 살로몬) → 운동의 계급화(필라테스/골프/크로스핏) → 음악 취향 정체성과 비교 → 10년 뒤에도 이 질문이 중요할까

**2. "지금 가장 핫한 운동은 6개월 뒤에 없어진다"**
명제: 운동에도 패션처럼 트렌드가 있고 유통기한이 있다.
흐름: 에어로빅→헬스→요가→크로스핏→러닝→클라이밍→하이록스 역사 → 각 운동이 뜬 사회적 맥락 → "10년 뒤에도 남을 운동"은 뭔가 → 음악 장르 트렌드와 비교(인디→힙합→시티팝) → 다음에 뜰 운동 예측 → "크로스핏은 이미 트렌드 아닌가?" 도발 → 트렌드를 따라가는 게 맞나

**3. "크로스핏이 세상에서 가장 욕을 많이 먹는 운동인 이유"**
명제: "크로스핏 하는 사람인 걸 어떻게 알 수 있냐? 본인이 말해줌" — 왜 특정 운동은 밈이 되는가.
흐름: 크로스핏 밈 → 비건/애플유저/테슬라오너와 같은 구조("말하고 싶은 충동"의 심리학) → 수현의 크로스핏 내부자 시점 vs 임용의 외부자 시점 → 부상 논란의 데이터 → 과몰입이 문화를 만든다(크로스핏 게임즈→Zone X) → 음악씬의 "인디 듣는 사람" 밈 비교 → "빠가 까를 만든다 vs 빠가 문화를 만든다"

### 월드컵/랭킹 포맷 (1차)

**A. "운동 종목 티어 랭킹"** (추천) — 기준을 바꿔가며 S~D 매기기 (소개팅 반응, 가성비, 인스타 반응, 부상위험 대비 재미). 숏폼: 기준별 1-2분 클립. 시리즈화 가능.
**B. "운동인 공감 이상형 월드컵"** — 운동 후 보상/최악의 핑계/짜증나는 순간/같이 운동하고 싶은 셀럽 멀티 카테고리.
**C. "스포츠 취향 속사포"** — 이분법 논쟁 타임어택 (새벽vs밤, 유산소vs근력 + 비스포츠 섞기). 숏폼: 논쟁당 1개.
**D. "이건 스포츠야? 아니야?"** — e스포츠/체스/당구/댄스 등 경계 활동 토너먼트.
