## 🎯 A-B-C 프레임워크

### A (주체 설정)
- **주체:** CH MAG 큐레이터
- **맥락:** 스포츠웨어 혁신 트렌드를 빠르게 큐레이팅하는 사람
- **포지션:** "이거 봤어?"라고 질문하는 친구

### B (행위 설정)
- **행위:** On Running의 LightSpray 기술을 소개한다
- **특별함:** 로봇이 3분 만에 운동화를 '분사'해서 만드는 기술. 기존 제조 방식을 완전히 뒤집는 혁신.
- **핵심 앵글:** "신발을 만드는 게 아니라 뿌린다" — 제조의 미래

### C (감상/이유)
- **핵심 감정:** 경이로움, 미래 기술에 대한 호기심
- **기대 반응:** "이건 진짜 몰랐다", "이게 가능해?", "On이 이 정도였어?"

## 📝 스크립트

### [Ver.1 — 풀 버전]

```
[0–3초 | 훅]
여러분 이거 봤어요?
(On Running LightSpray 로봇 제조 영상 빠른 컷)

[3–8초]
On Running이 운동화를 만드는 방식을 완전히 바꿨어요.
'LightSpray'라는 기술인데요,
로봇 팔이 3분 만에 신발 한 켤레의 어퍼를 뿌려서 만들어요.

[8–15초]
기존 운동화 공장이랑 비교하면요,
보통은 200명의 작업자가 필요한 공정을
로봇 한 대가 대체하는 거예요.
봉제선도 없고, 끈도 없고, 겨우 8개 부품으로 완성되죠.

[15–22초]
그리고 이 기술로 만든 신발은 이미 올림픽에서 검증됐어요.
파리 올림픽 마라톤에서 헬렌 오비리가 LightSpray 어퍼를 달고 뛰어서
올림픽 메달리스트가 됐고, 이후 보스턴 마라톤까지 우승했거든요.

[22–30초]
On은 최근에 부산에 로봇 32대 규모의 공장을 열었어요.
취리히 본사 공장 4대에서 시작해서,
2026년 생산량을 30배로 늘리겠다는 계획이에요.

[30–38초]
이번에 나오는 Cloudmonster 3 Hyper가
LightSpray를 일반 러너에게 처음 선보이는 모델인데요.
무게가 기존 대비 25% 가벼운 205g,
가격은 280달러, 3월 5일 출시예요.

[38–45초 | 엔딩 훅]
On의 공동 창업자가 이런 말을 했어요.
"아직 LightSpray 잠재력의 3%만 보여줬다."
로봇이 만드는 운동화, 2028 LA 올림픽까지 어디로 갈지—
여러분 생각이 궁금합니다. 댓글로 남겨주세요!
```

## 🔍 키포인트 정리
1. **기술 혁신:** 로봇이 3분 만에 어퍼를 분사 제조 (LightSpray)
2. **제조 혁명:** 200명 작업자 → 로봇 1대, 8개 부품
3. **올림픽 검증:** 파리 올림픽 마라톤 메달리스트 착용
4. **글로벌 확장:** 부산 공장 (로봇 32대), 30배 증산
5. **소비자 접점:** Cloudmonster 3 Hyper — 205g, $280, 3/5 출시
6. **미래 비전:** "잠재력의 3%", 2028 LA 올림픽 마케팅 계획

## 📐 영상 구성 참고
- **썸네일:** "이거 봤어?" + LightSpray 로봇 제조 장면 or 완성 신발
- **B-roll 필요:** On LightSpray 로봇 제조 영상, Cloudmonster 3 제품 샷, 올림픽 경기 장면
- **BGM 톤:** 테크/혁신 느낌, 깔끔하고 모던한 비트
