공통 규칙
  • 실제 사이즈: 1080×1440px (Instagram 세로형 3:4) — 정방형 아님
  • "DORM" 배지 — 흑백 박스 타입, 모든 썸네일 헤더 왼쪽 상단 고정
  • 시리즈별 포인트 컬러 1개 + 흑백 베이스 — 색상 혼용 없음
  • 타이틀 텍스트만 교체하는 Figma 마스터 컴포넌트로 제작
  • 에디터 소재 확정 → 타이틀 입력 → 자동 썸네일 완성 구조 목표
Fan Archive — 세부
  • 크림 배경 + Figma 노이즈 레이어 (opacity 6~8%)
  • 상단 1/3: 큰 회색 "001" (opacity 6%) + 파일링 메타데이터
  • "ARCHIVED" 빨간 스탬프 — -10도 회전, opacity 60%, 3px 외곽선
  • Noto Serif KR Black 타이틀 — 40pt, 최대 4줄, keep-all
  • 빨간 수평선 (32×2.5px) + 서브텍스트 구분
Fan's POV — 세부
  • 뷰파인더 브래킷 — 노란색 2px, 4모서리 내측 12px, 길이 16px
  • 스코어보드 위젯 — 노란 테두리, 팀명/점수/회 표시, 상단 고정
  • LIVE 적색 점 — 정지 버전 + GIF 깜빡임 버전 2종
  • Black Han Sans 타이틀 — 54pt, 노란 악센트 라인 1줄
  • Scanline 오버레이 — 수평선 반복, opacity 7%, 최상단 레이어
팬 소비 리포트 — 세부
  • 배경: 열지 그레이(#EEEEE9) 위에 흰 영수증 카드 float
  • 영수증 상단 3px 검정 솔리드 + 하단 zigzag 절취선 SVG
  • 품목 5행: Space Mono, 이름 + 점선 + 금액 레이아웃
  • 합계 + 결제수단 행 + 바코드 + 채널명 순서
  • "FAN DATA" 파란 스탬프 — +13도 회전, opacity 48%
