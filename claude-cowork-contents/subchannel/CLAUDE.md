# 서브채널 TF — 작업 공간

CH MAG 서브채널 4종의 기획·운영 문서 모음. 각 채널은 독립 프로젝트로 운영.

## 채널 구조

| 폴더 | 채널 | 리더 | 런칭 |
|---|---|---|---|
| `subchannel-dorm/` | DORM — 스포츠 라이프스타일 팬 저널리즘 | 은서 (이선 초기 세팅) | 4월 중 |
| `subchannel-if/` | IF — 20대 대화 소재 제안 채널 | 이선 | 3월 중 |
| `subchannel-senior/` | CH SENIOR — 50-60대 웰니스 채널 | 임용 (추후) | 5월+ |
| `subchannel-unit0/` | UNIT-0 — AI 스니커즈 수집 SF 채널 | 수현+임용 (직접 운영) | 4월 초 |

## 공통 담당

- **예영**: DORM + IF 대학생 에디터 풀 모집·운영·관리 (cross-channel)
- **임용**: 모든 채널 카피 교정·사실 확인 (주 1-2h/채널)
- **우석**: 채널별 비주얼 가이드라인 초기 셋업 (1회)
- **수현**: 월 1회 방향 점검

- 채널별 문서는 해당 `subchannel-[채널명]/` 폴더 안에서 작업
- 파일명: `[채널명]_[주제]_YYMMDD.md`
- 예: `DORM_콘텐츠_캘린더_260401.md`, `IF_질문DB_260315.md`
