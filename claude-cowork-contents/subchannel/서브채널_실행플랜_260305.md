## DORM — 실행 플랜

### DORM 실행 포인트

- **콘텐츠 소재**: 스포츠·라이프스타일 전반 (KBO·NBA 등 리그 소식, 패션·아이템, 피트니스 트렌드)
- **앵글**: 소식 뒤에 잠들어 있던 사람의 이야기를 꺼낸다 (Dormant Stories)
- **에디터**: 20대 대학생 풀로 구성 → 자연스럽게 20대 타겟 콘텐츠가 됨
- **제작 공수**: 편당 2-3시간 이내. CH MAG의 1/3 이하.

### 콘텐츠 파이프라인

```
[수현] 소재 방향·주간 스포츠·라이프스타일 씬 훑기 (월 1회 방향 + 주간 소재 힌트 공유)
    ↓
[예영 매니저] 주간 소재 확정 + 에디터 배정 (주 10-15h)
    ↓
[대학생 에디터 2-3명] 취재·촬영·편집 (스마트폰 OK, 현장감 우선)
    ↓
[임용] 카피 교정 + 사실 확인 (발행 전, 주 1-2h)
    ↓
[우석] 피드 비주얼 가이드라인 적용 (초기 셋업 후 인턴에 위임)
    ↓
예약 발행
```

### 대학생 에디터 풀 모집 시스템

**에디터 역할**: 취재자·스토리텔러. 소식을 사람을 통해 다시 풀어내는 사람. 촬영 스킬보다 커뮤니케이션 능력과 스포츠·라이프스타일에 대한 관심이 더 중요.

**모집 채널**:
1. 에브리타임 광고 게시 (서울·수도권 주요 대학 타겟)
2. DORM (A) 인스타그램 계정에 "에디터 모집" 포스트 (채널명 확정 후 계정 오픈 시)
3. CH MAG 본 계정 스토리 리포스팅 1회

**모집 헤드라인**: "DORM 입주자 모집 — 스포츠·라이프스타일 씬의 숨은 이야기를 꺼낼 사람"

**모집 포지셔닝**: "스포츠 좋아하고, 사람 이야기 듣는 거 좋아하는 사람. 카메라 없어도 됨. 스마트폰으로 시작. 우리 방에 들어와."

**선발 기준**:
- "요즘 팔로우하는 스포츠·라이프스타일 계정 3개와 이유" 제출
- 왜 지원했는지 50자 이내
- 스포츠·라이프스타일에 대한 진짜 관심 우선 (전공·스펙 무관)

**운영 구조**:
- Notion 기반 콘텐츠 캘린더 공유 (DORM 룸메이트 채팅방 운영)
- 주간 소재 힌트 → 에디터가 사람 찾고 취재
- 월 1회 온라인 미팅 (피드백 + 다음달 소재 방향)
- 인원: 2-3명 유지

## 채널 C: IF — 실행 플랜

### 콘텐츠 파이프라인

```
[수현 + 임용] IF 질문 아이템 30개 생성 — Claude 활용 (월 1회, 2시간)
    ↓
[이선 매니저] 주간 콘텐츠 큐 구성 + 발행 일정 관리 (주 10-15h)
    ↓
[대학생 에디터 2-3명] IF 질문 시각화 + 썰 공유 영상 제작
    ↓
[임용] 카피 최종 확인 + 스토리 인터랙션 기획 (주 1h)
    ↓
발행 + 스토리 투표·퀴즈 병행 업로드
    ↓
[이선] 댓글·DM 반응 수집 → 다음 콘텐츠 소재 순환
```

### 대학생 풀 모집 시스템

**모집 특성**: IF 채널 에디터는 제작자보다 "기획자" 역할에 가깝다.

**모집 채널**:
1. IF 계정 자체에서 모집 포스트 발행 (채널 자체가 홍보 수단)
2. 에브리타임 홍보 (DORM (A)와 동시 진행, 공고 내용 차별화)
3. 카카오 오픈채팅방 (대학 커뮤니티)

**모집 포지셔닝**: "인스타 콘텐츠 만들어본 적 없어도 됨, 대화 좋아하는 사람 환영, 기획하는 게 좋은 사람"

**운영 특성**:
- 매주 "IF 질문을 친구들한테 실제로 써봤는지" 간단 리포트 공유
- 반응 좋은 질문·답변을 다음 콘텐츠 소재로 순환

### 자동화 방안

**IF 질문 생성 자동화 (핵심)**:
```
Claude 월 1회 프롬프트:
"20대 대학생이 처음 만난 사람에게 할 수 있는 IF 질문 30개.
조건:
- 특정 지식 없어도 답할 수 있음
- 답변이 그 사람을 알 수 있게 함
- 너무 무겁지 않고, 1-2문장으로 답 가능
카테고리: 관계/직업·미래/소비/여행/스포츠/음식 각 5개씩"
```

**멀티포맷 자동 변환**:
```
IF 캐러셀 1개 소재
    → 릴스 스크립트 (Claude 변환)
    → 틱톡 텍스트 오버레이 형식 (Claude 변환)
    → 스토리 투표 문항 (Claude 변환)
= 하나의 소재로 4종 포맷 동시 제작
```

**반응 분석 자동화**:
- 스토리 투표·퀴즈 결과 데이터 → Claude 인사이트 요약
- 다음 달 IF 질문 기획에 반영

### 인스타그램 인터랙션 설계

발행할 때마다 3종 세트 운영:
1. 캐러셀/릴스 — 메인 콘텐츠
2. 스토리 — 해당 IF 질문으로 투표·퀴즈
3. 마지막 슬라이드 CTA — "여러분이면? 댓글로 답해주세요"
