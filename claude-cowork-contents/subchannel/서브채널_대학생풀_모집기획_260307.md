## 지원 프로세스

### 서류 (간단하게)

복잡한 서류 없이 3가지만:

1. 요즘 팔로우하는 스포츠·라이프스타일 인스타 계정 3개와 팔로우 이유
2. 왜 지원했는지 50자 이내
3. 관심 채널 선택: DORM / IF / 둘 다 가능

> **평가 기준**: 스펙·전공 무관. 스포츠·라이프스타일에 대한 진짜 관심 + 사람 이야기에 대한 관심 우선.

### 인터뷰 (예영 주도)

- 형식: 30분 내외 화상 또는 대면
- 핵심 질문:
  - 최근에 가장 인상깊었던 스포츠 관련 소식은?
  - 직접 취재하고 싶은 "사람"이 있다면?
  - DORM / IF 중 어느 채널에서 어떤 콘텐츠를 만들고 싶은지?

### 합격 → 온보딩

**온보딩 패키지 (이선 세팅)**:
- Notion 콘텐츠 캘린더 접근 권한 부여
- DORM 채널 가이드라인 문서 (톤·비주얼·캡션 가이드)
- "DORM 룸메이트 채팅방" 입장
- 첫 소재 배정 + 파일럿 제작 시작

## DORM vs IF 배분 기준

같은 에디터 풀에서 채널 특성에 맞게 배분. 지원서의 "관심 채널" + 인터뷰 결과로 결정.

| 기준 | DORM 에디터 | IF 에디터 |
|---|---|---|
| 성향 | 현장형, 직접 만나서 이야기 듣는 게 좋은 사람 | 아이디어·기획형, 질문 설계하고 반응 보는 게 좋은 사람 |
| 스포츠 관심 | 높음 (직접 팬이거나 씬에 관심) | 상대적으로 낮아도 OK |
| 제작 방식 | 현장 취재·촬영·편집 중심 | 질문 기획·시각화·인터랙션 설계 중심 |
| 양 | 2-3명 (소수 정예) | 2-3명 |

**운영 방침**: 모든 에디터는 채널 1개 전담 + 반대 채널 콘텐츠 주 1개 병행 가능.
→ DORM 에디터도 IF 질문 아이디어 제안 가능, 역으로도 가능.
