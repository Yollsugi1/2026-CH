**정수현 | 크리에이터후드**

## 1. 사업 방향

크리에이터후드는 2026년, 인스타그램 매거진 중심의 운영에서 **유튜브·인스타그램·웹매거진을 아우르는 영상 중심 콘텐츠 스튜디오**로 전환합니다.

핵심 포지셔닝은 **"스포츠판 메티즌"**입니다. 메티즌이 식문화 콘텐츠에서 자리를 잡은 것처럼, 크리에이터후드는 스포츠 라이프스타일 분야에서 독보적인 콘텐츠 IP를 구축하는 것을 목표로 합니다.

신설법인은 이 전환의 기반이 되며, 크게 **세 가지 축**으로 사업을 전개합니다.

| 축 | 프로젝트 | 성격 |
|---|---|---|
| **CH MAG** | 오리지널 콘텐츠 스튜디오 | 정규 시리즈 7종 + 웹매거진, 멀티 플랫폼 운영 |
| **서브 매거진 채널** | 바이럴 콘텐츠 채널 (2개) | 시의성·속도 중심, 대학생 에디터 기반 고볼륨 운영 |
| **Zone X** | 스포츠 리그 사업 | 크로스핏/기능성 피트니스 기반 리그, 시즌 1 (6월~) |

## 2. CH MAG — 오리지널 콘텐츠 스튜디오

### 2-1. 전략적 전환

2025년까지 인스타그램 매거진으로 운영하면서 축적한 브랜드 자산과 콘텐츠 제작 역량을 기반으로, 2026년에는 **7개 정규 시리즈**를 중심으로 한 멀티 플랫폼 콘텐츠 스튜디오를 구축합니다.

타겟 오디언스는 **"운동에서 배운 몰입의 에너지를 삶 전반에 적용하는 사람들"**이며, 단순 운동 정보나 스포츠 뉴스가 아닌 입체적인 라이프스타일 콘텐츠를 지향합니다.

### 2-2. 정규 시리즈 라인업

| 시리즈 | 장르 | 포맷 | 발행 주기 |
|---|---|---|---|
| **Hoodies (후디스)** | 크리에이터 브이로그 | 롱폼 5~15분 | 주 1회 |
| **Playbook (플레이북)** | 토크쇼 | 롱폼 20~30분 + 숏폼 클립 | 주 1회 |
| **폭력적인 식단** | 맛집 숏폼 | 숏폼 60~90초 | 주 1회 |
| **Pit Stop** | 카페 큐레이션 숏폼 | 숏폼 60초 | 주 1회 |
| **Shirtless** | 스타일링 숏폼 | 숏폼 60~90초 | 격주 1회 |
| **False 9** | 시네마틱 필름 | 롱폼 5~10분 | 월 1~2회 |

이 외에 기존 [봤어?] 릴스, 큐레이션 캐러셀 등 상시 콘텐츠를 병행합니다.

### 2-3. 플랫폼별 발행량 (월간 목표)

| 플랫폼 | 발행량 | 주요 콘텐츠 |
|---|---|---|
| 인스타그램 | 30~40개/월 | 릴스 20 + 매거진형 포스트 10~15 |
| 유튜브 | 6~10개/월 | 롱폼 시리즈 + 숏폼 미러링 |
| 웹매거진 | 10~15개/월 | 에디토리얼·큐레이션 상세 버전 |

### 2-4. 시리즈 론칭 타임라인

| 시기 | 마일스톤 |
|---|---|
| 2월 | 4개 시리즈 티저 촬영·업로드, Hoodies 세부기획·촬영 시작 |

### 2-5. Hoodies 브랜드 스폰서십

Hoodies 시리즈는 크리에이터후드의 핵심 IP로서 **브랜드 스폰서십의 주요 대상**입니다.

- 크리에이터별 개별 스폰서 or 시리즈 전체를 특정 브랜드와 협업하는 형태
- 브랜드가 취할 수 있는 가치(브랜디드 콘텐츠, 제품 노출, 공동 기획 등)를 정리한 **브랜드 소개서를 별도 제작**하여 영업에 활용할 수 있도록 준비할 예정입니다.

## 3. 서브 매거진 채널 — 바이럴 콘텐츠 운영

### 3-1. 왜 서브채널인가

CH MAG은 오리지널 시리즈 등 공수가 높은 콘텐츠 위주로 운영됩니다. 이와 별도로, **시의성 있는 콘텐츠를 빠르게·많이 생산하며 바이럴시킬 채널**이 필요합니다.

벤치마크 대상은 1%클럽, 패션상사, 굿테이스트, 캐릿 등 **빠른 속도와 포맷 중심으로 성장한 매거진 채널**들이며, 크리에이터후드의 대학생 네트워크를 활용하여 이를 실현합니다.

### 3-2. 운영 구조

- **채널 수**: 2개 (각각 별도의 콘텐츠 방향성)
- **운영 모델**: 포맷 중심, 대학생 주도, 고볼륨, 빠른 속도
- **CH MAG과의 역할 분담**:

| | CH MAG | 서브 매거진 채널 |
|---|---|---|
| 속도 | 주 단위 정규 발행 | 일 단위 빠른 발행 |
| 방향 | 오리지널 IP, 퀄리티 중심 | 트렌드·시의성, 바이럴 중심 |
| 제작 주체 | 내부 핵심팀 | 매니저 4명 + 대학생 에디터 |

### 3-3. 인력 구조

서브채널은 **매니저 4명**이 운영하며, 이들이 신규 대학생 에디터를 모집·관리하는 **중간관리자** 역할을 수행합니다.

```
크리에이터후드 (수현 디렉션)
  └─ 서브채널 매니저 4명
       ├─ 운영 총괄 PM (1명)
       ├─ 커뮤니케이션 & 섭외 리드 (1명)
       ├─ 채널 A 콘텐츠 디렉터 (1명)
       └─ 채널 B 콘텐츠 디렉터 (1명)
            └─ 각 채널별 대학생 에디터 (추가 모집)
```

- 매니저 4명은 2025년 크리에이터후드 에디터로 활동한 경험이 있으며, 올해 매니저로 역할을 전환합니다.
- 이들이 대학생 에디터를 직접 모집·온보딩·관리하는 구조로, **단순 아르바이트가 아닌 조직을 운영하는 경험을 제공**하면서도 크리에이터후드의 인력 레버리지를 만드는 핵심 구조입니다.

### 3-4. 서브채널의 사업적 가치

1. **볼륨 생산 능력 확보**: CH MAG만으로는 채울 수 없는 발행량과 시의성을 대학생 네트워크로 해결
2. **바이럴 채널 → 광고·스폰서십**: 팔로워 성장 시 독립적인 매출원으로 발전 가능
3. **인력 파이프라인**: 서브채널 에디터 중 역량 있는 인재를 CH MAG이나 Zone X로 영입
4. **조직 확장의 기반**: 매니저 4명이 각각 5~10명 에디터를 운영 → 실질적인 조직 운영 구조

## 4. Zone X — 스포츠 리그 사업

### 4-1. 개요

Zone X는 크로스핏/기능성 피트니스 기반 리그 사업으로, **시즌 1을 6월에 시작**하는 것을 목표로 합니다.

- **예산**: ~1.5억원 (스폰서십으로 조달)
- **시즌 구조**: 리그라운드 4주 (격주 운영 → 최대 8주) + 챔피언십 2주
- **기간**: 최소 6~7월, 최대 8월까지

### 4-2. 타임라인

| 시기 | 마일스톤 |
|---|---|
| 2월 (설 연휴 후) | 본격 준비 시작, 기획 구체화 |
| 2~3월 | **스폰서십 유치 (영업)**, 참가팀 모집 |
| 3~4월 | 디자인·브랜딩, 트레일러 제작, 콘텐츠 사전 제작 |
| 5월 | 리허설·시뮬레이션, SNS 사전 마케팅 |
| **6월~** | **시즌 1 시작** |
| 시즌 후 | 콘텐츠 릴리즈, 시즌 2 방향 검토 |

### 4-3. 외부 파트너

| 파트너 | 역할 |
|---|---|
| PIP | 영상 제작 |
| Paul | 트레일러 촬영 |
| HOONTAEK | 디자인 |
| COV | 현장 운영 |

### 4-4. Zone X의 사업적 가치

- **콘텐츠 자산**: 리그 운영 과정과 결과물이 CH MAG 콘텐츠로 연계
- **브랜드 파워**: 스포츠 리그 운영사로서의 입지 → 스폰서십·파트너십 확대
- **커뮤니티**: 참가팀·관람객 네트워크 → 콘텐츠 소비자 풀 확대
- **반복 매출 구조**: 시즌제 운영으로 매년 스폰서십 매출 발생

```
정수현 (대표)
  ├─ 핵심팀 4명 (지운, 우석, 임용 + 한상필 대표)
  ├─ 인턴/매니저 8명
  │    ├─ 서브채널 매니저 4명 → 대학생 에디터 모집·관리
  │    ├─ Zone X 운영 2명

  └─ 크리에이터 5명 (Hoodies)
```

## 8. 투자 근거 요약

신설법인 초기 투자의 핵심은 **인턴/매니저급 인력 8명의 상반기 운영비**입니다.

### 왜 이 투자가 필요한가

1. **서브 매거진 채널 운영**: 4명의 매니저가 2개 채널을 운영하며 CH MAG만으로는 불가능한 볼륨과 바이럴을 실현
2. **Zone X 운영 지원**: 2명이 리그 현장 운영을 담당하여 시즌 1의 안정적 진행을 보장
3. **조직 확장의 레버리지**: 매니저 8명이 아래에 대학생 에디터 풀을 관리 → **8명으로 20~30명 규모의 인력을 운영**하는 효과
4. **인력 파이프라인**: 서브채널 에디터 → CH MAG 기여 → 핵심팀 합류의 성장 경로
