# 주간미팅 브리핑 자동 생성 스킬

## Objective

매주 주간미팅 전에 마스터플랜 + 노션 미팅 문서를 크로스체크하여, 콘텐츠별 논의 필요 액션을 정리한 브리핑 MD 파일을 자동 생성한다.

## 실행 순서

### Step 1: 소스 데이터 수집

**1. 마스터플랜 읽기**
- 파일 경로: 워크스페이스 폴더에서 `크리에이터후드_2026_마스터플랜*.md` 검색
- 핵심 섹션: "1. 시리즈별 현황 & 당장 필요한 액션", "2. 이번 주 ~ 다음 주 상세 스케줄"

**2. 노션 주간미팅 문서 Fetch**
- 사용자가 미팅 문서 링크를 제공하면 `notion-fetch`로 읽기
- 링크가 없으면: `notion-search`로 "주간미팅" 키워드로 최신 문서 검색
- 미팅 문서 구조 참고:
  - 사전작성내용 → 지난주 액션아이템 점검 → 금주 액션아이템 설정 → 주요 일정 공유 (table) → 금주 콘텐츠 플랜 → 기타 공유내용 → Meeting Notes → Action Items

**3. (선택) 이전 주간미팅 문서 참조**
- 지난주 미팅 문서가 있으면 함께 읽어서 지난 액션아이템 대비 진행 상황 체크

### Step 2: 크로스체크 분석

마스터플랜과 미팅 문서를 대조하여 다음을 파악:

| 체크 항목 | 설명 |
|----------|------|
| **마스터플랜에 있지만 미팅 문서에 없는 것** | 논의에서 빠질 수 있는 중요 액션 |
| **미팅 문서에 있지만 마스터플랜에 없는 것** | 새로 추가된 사항 또는 임시 항목 |
| **스케줄 충돌** | 같은 날짜에 겹치는 촬영/미팅 |
| **리소스 병목** | 특정 인원(특히 지운PD)의 일정 과밀 |
| **데드라인 임박** | 이번 주 내 반드시 완료해야 할 것 |

### Step 3: 브리핑 MD 생성

아래 템플릿에 맞춰 MD 파일 작성:

```markdown
# 주간미팅 액션아이템 브리핑 — {날짜}
### {미팅 제목} | @{날짜}

> 마스터플랜 v{버전} 기준으로 콘텐츠별 논의 필요사항을 정리했습니다.

---

## 이번 주 핵심 — "{주간 테마}"

{이번 주 가장 중요한 2~3가지 포인트 요약}

---

## 콘텐츠별 논의 필요 액션

### {N}. {시리즈명} ({긴급도})

**현황**: {현재 상태}
**이번 주 액션**:
- {구체적 액션 1}
- {구체적 액션 2}

**미팅에서 논의할 것**:
- {논의 포인트 1}
- {논의 포인트 2}

---
(시리즈별 반복)

## 리소스 병목 체크

{지운PD 등 핵심 리소스의 이번 주 스케줄 테이블}

---

## 미팅 후 확정해야 할 것 (체크리스트)

- [ ] {확정 필요 사항 1}
- [ ] {확정 필요 사항 2}

---

*마스터플랜 v{버전} 기준 작성*
```

### Step 4: 파일 저장

- 파일명: `주간미팅_액션아이템_W{주차}_{날짜}.md`
- 저장 위치: `claude-cowork-contents/`

## 긴급도 분류 기준

| 레벨 | 기준 |
|------|------|
| 시급 | 이번 주 내 촬영/마감 있음 |
| 기획 필요 | 기획 미완, 다음 주 촬영 예정 |
| 안정 | 운영 중 또는 보류 |
| 해당 없음 | 이번 주 액션 없음 |

## 리소스 병목 체크 대상

1. **김지운PD** — 촬영/편집 핵심 인력, 연속 촬영 체크
2. **수현** — 동시 디렉션 과부하 (기획+출연+총괄+경영)
3. **외부 게스트** — 스케줄 조율 필요 인원

## 제약사항

- 마스터플랜 파일이 없으면 사용자에게 요청
- 미팅 문서가 없으면 마스터플랜 기반으로만 작성
- 항상 한국어
- MD 파일은 `claude-cowork-contents/`에 저장
