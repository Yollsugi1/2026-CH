# 콘텐츠 기획 템플릿 구조

## 완성 기획안의 구조

콘텐츠 기획은 다음의 주요 섹션으로 구성됩니다:

### 1. 기본 정보
- **제목**: 콘텐츠 제목
- **길이**: 숏폼(1분 이하) / 미드폼(1-10분) / 롱폼(10분+)
- **제작일정**: 시작일 ~ 종료일 (예상 기간)

### 2. 아이디어 레벨
사용자가 제공한 raw 아이디어의 출처와 영감을 정리:
- **일상 관찰**: 날짜, 상황, 느낀점, 콘텐츠화 가능성
- **트렌드 분석**: 플랫폼, 인기 콘텐츠, 후킹 포인트, 응용 아이디어
- **레퍼런스 분석**: 제목, 채널, 성공 요인, 벤치마킹 포인트, 차별화 전략
- **아이디어 메모**: 핵심 아이디어 bullet points

### 3. A-B-C 프레임워크 (핵심)

#### A (주체 설정)
- 기본 주체
- 맥락 부여 (특별함을 만드는 배경/상황)
- 체크포인트:
  - [ ] 주체가 특별한가?
  - [ ] 맥락이 명확한가?
  - [ ] 시청자가 관심 가질만한가?

#### B (행위 설정)
- 기본 행위
- 특별함 부여 (제약, 방식, 규모 등)
- 체크포인트:
  - [ ] 일상적이면서도 특별한가?
  - [ ] 시각적으로 흥미로운가?
  - [ ] A와 충돌 또는 강화되는가?

#### C (감상/이유)
- 핵심 감정
- 시청 이유
- 기대 반응
- 체크포인트:
  - [ ] "그래서 뭐?"에 답이 되는가?
  - [ ] 감정이 명확한가?
  - [ ] 공유하고 싶은가?

### 4. 추가내용 - 기획안 레벨 보충

#### 타당성 검증
- **현실 연결성 체크**:
  - 일상 속 연결점
  - 사회적 맥락
  - 시의성
  
- **타겟 분석**:
  - 1차 타겟 (직접 관심층)
  - 2차 타겟 (호기심층)
  - 3차 타겟 (우연 유입층)
  - 타겟 확장 전략

#### 구성안
- **핵심 상황 설정** (3-4개):
  - 각 상황별 내용과 기대 반응
  - 롱폼은 숏폼 관점으로 접근 - 계속 '킥'을 넣어야 함
  
- **클라이맥스 설정**:
  - 핵심 장면 (가장 강조할 부분)
  - 위치 (초반/중반/후반)
  - 썸네일 후보

### 5. 시리즈 레벨 (해당시)

#### 시리즈 가능성
- [ ] 포맷 반복 가능
- [ ] 출연자 교체 가능
- [ ] 상황 변주 가능
- [ ] 스핀오프 가능

#### 발전 방향
- 유지할 것
- 바꿀 것
- 새로 시도할 것

### 6. 기획 검증 - 즉시 답해보기

**필수 질문**:
1. 한 줄 설명 (A+B=C)
2. 왜 봐야 해?
3. 누가 볼까?
4. 썸네일은?
5. 제목은?

**체크리스트**:
- [ ] A 또는 B만 특별함 (둘 다 평범하거나 둘 다 특별하면 안 됨)
- [ ] C가 불명확함 (감정이나 메시지가 애매함)
- [ ] 타겟이 너무 좁음 (특정 집단만 이해 가능)
- [ ] 현실과 동떨어짐 (일상과 연결점 없음)
- [ ] 주사위 던지기 (결과를 예측할 수 없음)

### 7. 세부 아이디어 (메모)
기타 추가로 떠오르는 아이디어나 메모

## 출력 형식

기획안은 Markdown 형식으로 작성되며, 다음의 원칙을 따릅니다:
- 헤더로 섹션 구분 (## 메인 섹션, ### 서브 섹션)
- 배경색 강조는 {color="yellow_bg"} 또는 {color="green_bg"} 사용
- 체크리스트는 - [ ] 형식
- 중요 강조는 **볼드** 또는 <span color="red">컬러</span>
- 예시나 부가 설명은 코드블럭이나 > 인용구 사용

## 기획 수준별 작성 가이드

### 간단한 아이디어 단계
- 섹션 1-3까지만 작성
- A-B-C 프레임워크에 집중
- 기본 타당성만 검증

### 구체적 기획 단계
- 섹션 1-6까지 작성
- 타겟 분석과 구성안 포함
- 기획 검증까지 완료

### 시리즈 기획 단계
- 모든 섹션 작성
- 시리즈 확장 가능성 분석
- 에피소드별 변주 방안 제시
