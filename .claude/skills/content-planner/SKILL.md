# Content Planner Skill

Creatorhood 콘텐츠 기획을 위한 전문 스킬입니다. 사용자의 raw한 아이디어를 받아 체계적인 콘텐츠 기획안으로 발전시킵니다.

## 사용 시나리오

이 스킬은 다음 상황에서 사용됩니다:

1. **새로운 콘텐츠 시리즈 기획**: 전체 시리즈의 컨셉과 방향성 수립
2. **시리즈 내 에피소드 기획**: 기존 시리즈의 개별 에피소드 개발
3. **단발성 콘텐츠 기획**: 독립적인 콘텐츠 아이디어 구체화
4. **아이디어 검증 및 구조화**: 막연한 아이디어를 실행 가능한 기획안으로 변환

## 기획 프로세스

### 1단계: 사용자 인풋 파악

사용자로부터 다음 정보를 수집합니다:

**필수 정보**:
- 콘텐츠 아이디어나 컨셉 (raw한 형태도 괜찮음)
- 콘텐츠 유형 (숏폼/미드폼/롱폼, 시리즈/단발)

**선택 정보** (있으면 좋음):
- 타겟 오디언스
- 영감의 출처 (일상 관찰, 트렌드, 레퍼런스)
- 원하는 감정이나 메시지
- 제작 제약사항 (예산, 일정, 리소스)

사용자의 인풋이 매우 raw하거나 불완전해도 괜찮습니다. 이 스킬의 목적은 raw한 아이디어를 구조화하는 것입니다.

### 2단계: 브랜드 컨텍스트 확인

**IMPORTANT**: 기획 전에 반드시 `references/brand-context.md`를 읽어 Creatorhood의:
- 브랜드 정체성과 콘텐츠 철학
- 타겟 오디언스 특성
- 톤 앤 매너
- 최근 콘텐츠 트렌드

이를 이해하고 기획에 반영해야 합니다.

### 3단계: A-B-C 프레임워크 적용

**CRITICAL**: `references/abc-framework.md`를 참고하여 A-B-C 프레임워크를 적용합니다.

**워크플로우**:

1. **A (주체) 정의**:
   - 사용자 아이디어에서 주체 식별
   - 맥락 부여하여 특별하게 만들기
   - 체크포인트 확인

2. **B (행위) 정의**:
   - 주체가 하는 행위 식별
   - 특별함 요소 추가 (제약, 방식, 규모 등)
   - A와의 관계 확인 (충돌 또는 강화)
   - 체크포인트 확인

3. **C (이유/감상) 정의**:
   - 핵심 감정 설정
   - 시청 이유 명확화
   - 기대 반응 정의
   - "그래서 뭐?" 질문에 답하기
   - 체크포인트 확인

**검증**:
- A 또는 B 중 하나만 특별한가? (둘 다 평범하거나 둘 다 특별하면 안 됨)
- C가 명확하고 공감 가능한가?
- 타겟에게 어필하는가?

### 4단계: 기획안 작성

`references/template-structure.md`를 참고하여 완성된 기획안을 작성합니다.

**기획안 수준 결정**:
- **간단한 아이디어**: 기본 정보 + A-B-C + 검증
- **구체적 기획**: 위 항목 + 타당성 검증 + 타겟 분석 + 구성안
- **시리즈 기획**: 위 항목 + 시리즈 가능성 + 발전 방향

**작성 원칙**:
1. **명확성**: 모호한 표현 지양, 구체적으로
2. **실행 가능성**: 제작 가능한 수준으로
3. **Creatorhood 스타일**: 브랜드 톤앤매너 반영
4. **한 줄 요약 가능**: A+B=C로 명쾌하게

### 5단계: 검증 및 피드백

기획안 작성 후 다음 5가지 질문에 즉시 답할 수 있어야 합니다:

1. **한 줄 설명**: A+B=C로 설명
2. **왜 봐야 해?**: 시청 동기
3. **누가 볼까?**: 타겟 정의
4. **썸네일은?**: 시각적 포인트
5. **제목은?**: 매력적인 제목

이 5가지에 명확히 답할 수 없다면 기획을 재검토해야 합니다.

## 출력 형식

기획안은 Markdown 형식으로 작성하며 다음 구조를 따릅니다:

## 중요 원칙

### DO
- 사용자의 raw한 아이디어를 경청하고 존중
- Creatorhood의 브랜드 맥락 항상 고려
- A-B-C 프레임워크를 충실히 적용
- 타겟 오디언스 관점에서 사고
- 실현 가능성 고려
- 시리즈화 가능성 염두

### DON'T
- 사용자 아이디어를 임의로 크게 변형
- 뻔한 스포츠 콘텐츠 패턴 답습
- 과도하게 복잡하거나 추상적인 기획
- 브랜드 정체성과 맞지 않는 톤
- 제작 불가능한 비현실적 기획
- "그래서 뭐?" 질문에 답 못하는 C

## 시리즈 vs 단발 콘텐츠

### 시리즈 기획시 추가 고려사항
- 포맷의 반복 가능성
- 출연자 교체 가능성
- 상황 변주 가능성
- 스핀오프 가능성
- 시리즈 전체의 일관성
- 에피소드별 차별화

**시리즈 기획시 추가 섹션**:
```markdown
## 시리즈 레벨 {color="yellow_bg"}

### 시리즈 가능성
- [ ] 포맷 반복 가능
- [ ] 출연자 교체 가능
- [ ] 상황 변주 가능
- [ ] 스핀오프 가능

### 발전 방향
- **유지할 것**: 
- **바꿀 것**: 
- **새로 시도할 것**: 
```
