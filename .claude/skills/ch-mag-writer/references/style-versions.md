# CH MAG 스타일 버전 기록

편집자(정임용) 피드백을 거치며 축적되는 CH MAG 스타일 정의.
새 글을 쓰기 전에 항상 이 파일부터 읽어서 최신 스타일 방향을 반영한다.

---

## 사용법

각 피드백 엔트리는 다음 구조를 따른다:

```
### [날짜] — [글 제목 또는 피드백 상황]
**맞다 ✓:**
- 무엇이 CH MAG 스타일로 인정받았는지

**CH MAG 스타일 아니다 ✗:**
- 무엇이 스타일에서 벗어났는지

**스타일 업데이트:**
- style-guide.md에 반영해야 할 새 규칙이나 수정 사항
```

---

## v1.0 — 2026-02-19 초기 스타일 정의

> 근거: 기존 게시 글(베네치아 FC 유니폼 아카이브, 발렌시아가 + 스포츠, 슈퍼볼 패션) 분석 + 첫 번째 draft 작성 세션

### 확인된 스타일 원칙

**맞다 ✓:**
- 선언형 훅으로 시작: "패션은 문화를 빨아들이는 블랙홀이다." 스타일
- 인트로는 각도만 던지고 끝내기 — 구조 예고 금지
- 섹션 헤더는 시적 표현 (설명적 라벨 아님)
- 긴 묘사 문장 → 짧고 단호한 마무리 문장의 반복 리듬
- 아웃트로는 인트로 각도를 완성하는 한 문장
- 구체적 수치와 이름으로 글의 무게 잡기
- 스포츠 × 문화 교차점 — 단순 소개 글이 아닌 것

**구조적 원칙 (첫 세션에서 확인):**
- 단순하고 가벼운 소재 → 서사적으로 의미 있는 소재 순서 배치
- 단, 이건 참고 원칙이지 절대 법칙이 아니다
- 소재마다 어울리는 구조가 다를 수 있음
- 가장 좋은 글이 나오는 구조가 정답

**소싱 확장 원칙:**
- 사용자가 제시한 3개 소재에만 갇히지 않기
- 전혀 생각 못 했을 인접 영역까지 열어두기
- 넓게 제안 → 함께 추려나가는 방식

### 첫 draft "금지 기술의 미학" — 채택된 요소들
- 인트로: "모든 스포츠에는 하면 안 되는 것들의 목록이 있다. 그 목록에 이름이 올라간 기술들은, 때로 이름이 없는 것들보다 훨씬 오래 기억된다." → **선언형 훅의 좋은 예시**
- 섹션 헤더 4개: 지워진 혁명들 / 쓸 수 없어서 더 유명해진 / 막힌 곳에서 열린 길 / 그녀는 알고 있었다 → **시적 표현 헤더의 좋은 예시**
- 클라이맥스: 수리아 보날리 1998 나가노 — 실격을 알면서 뛴 인간적 서사로 마무리 → **가장 무거운 서사를 끝에 배치하는 구조**
- 아웃트로: "금지는 스포츠의 문법이다. ... 하지만 그 문법 바깥에서 쓰인 문장들이 때로는 더 오래 남는다." → **인트로 각도를 완성하는 아웃트로**
