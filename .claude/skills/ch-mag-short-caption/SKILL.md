# CH MAG Short Caption Writer

CH MAG 톤을 유지하면서 짧고 임팩트 있는 캡션을 쓰는 스킬.
장문 에디토리얼(ch-mag-writer)과 달리, 한 호흡에 읽히는 짧은 글을 만든다.

---

## 용도

- **포토덤프**: 여러 장의 이미지를 묶어 올릴 때 붙는 캡션
- **단편 포스트**: 이미지 1~3장 + 카피 한 줄 + 캡션 한 문단
- **PR/협찬 콘텐츠**: 브랜드로부터 의뢰받은 콘텐츠 홍보 포스팅
- **이벤트 안내**: 팝업, 파티, 러닝 이벤트 등 행사 소개
- **릴스/영상 캡션**: 짧은 영상에 붙는 설명 텍스트

---

## CH MAG 숏폼 톤

CH MAG의 기본 톤("아는 사람이 쿨하게 해석하는")을 유지하되, 숏폼에 맞게 압축한다.

**핵심 원칙:**

1. **정보를 서사로 포장한다.** 단순 안내문이 아니라, 한두 문장 안에 관점이 들어간다.
   - X: "3월 15일 동아마라톤 애프터파티가 열립니다"
   - O: "42.195km를 달리고 난 뒤의 시간은 결승선 너머에 있다."

2. **독자에게 직접 말 걸지 않는다.** "~하세요", "여러분", "함께해요" 금지. 서술자는 관찰자다.
   - X: "러너 여러분, 함께해요!"
   - O: "달린 사람도, 달리지 않은 사람도, 결국 같은 테이블에 앉는다."

3. **구체적 팩트를 넣는다.** 날짜, 장소, 브랜드명, 숫자 — 짧은 글일수록 구체성이 신뢰를 만든다.

4. **마지막 문장에 여운을 남긴다.** 짧은 글이라도 마무리는 열린 여운이 있어야 한다.

5. **광고 냄새를 지운다.** PR 콘텐츠라도 CH MAG을 통해 나가는 이상, 홍보 문구가 아니라 콘텐츠로 읽혀야 한다.

## 산출물 구조

### 1. 카피 (Cover Copy)
이미지 위에 올라갈 수 있는 짧은 텍스트. 1~2문장, 최대 30자 내외.
선언적이거나 감각적인 표현.

### 2. 캡션 (Caption)
인스타그램 본문에 들어가는 텍스트. 3~6문장, 한 문단.

구조:
- **오프닝 (1문장):** 훅. 관점이나 감각이 담긴 짧은 문장.
- **컨텍스트 (2~3문장):** 무엇이, 언제, 어디서, 왜. 핵심 정보를 서사적으로 전달.
- **클로징 (1문장):** 여운. 짧고 열린 마무리.

### 3. 해시태그
CH MAG 고정 태그 + 소재 관련 태그. 10~15개.
```
고정: #CHMAG #CHMAGazine #스포츠라이프스타일
소재별 추가: 주제, 브랜드, 이벤트, 장소 관련
```

### 4. 카피/캡션 옵션
항상 2~3개 버전을 제시한다. 각 버전은 톤이나 앵글이 다르게.

```
=== CH MAG SHORT CAPTION ===

[버전 A — 앵글: OOO]
카피: ...
캡션: ...
해시태그: ...

[버전 B — 앵글: OOO]
카피: ...
캡션: ...
해시태그: ...
```

---

## PR/협찬 콘텐츠 처리법

1. 의뢰 내용에서 핵심 메시지를 추출한다.
2. CH MAG의 시각으로 재해석한다.
3. 정보는 모두 넣되, 톤은 CH MAG이다.
4. 브랜드명은 자연스럽게 문맥 안에서 한두 번.

---

## 포스트 유형별 가이드

### 이벤트/파티 안내
- 날짜, 장소, 참여 조건은 반드시 포함
- "모집" "신청" 같은 단어보다는 상황 묘사로 전달

### 제품/브랜드 소개
- 제품 스펙 나열 금지
- 이 제품이 어떤 순간/맥락에서 의미 있는지를 중심으로

### 러닝/스포츠 이벤트
- 커뮤니티 느낌을 살리되, "함께 달려요!" 같은 상투적 표현 금지

### 포토덤프 (여러 이미지)
- 각 이미지를 설명하지 않는다
- 전체 이미지가 만드는 무드를 한 문단으로 포착
