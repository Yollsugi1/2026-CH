## Objective

- 시리즈: 1.순당포착, 2.취하지아니한가, 3.1막1장, 4.기타

### Step 2: 3단계 공유 시스템 기반 D-N 분석

| 업로드까지 D-N | 이미 했어야 할 것 | 현재 안 됐다면 |
|---|---|---|
| D-14 이내 | 1차 공유(30%) 완료 | 1차 공유 지연 |
| D-10 이내 | 2차 공유(80%) 완료 | 2차 공유 긴급 |
| D-5 이내 | 최종 공유(100%) 완료 | 최종 공유 위험 |
| D-3 이내 | 업로드 준비 완료 | 업로드 위험 |

**"미팅 논의 필요" 자동 감지 조건:**
- 업로드일 대비 현재 공유 단계가 1단계 이상 지연된 경우
- 클라이언트 피드백이 D+2를 넘겨도 아직 반영 안 된 경우
- 촬영 일정이 확정되지 않은 채 D-21 이내인 경우
- "예정" 상태인데 아무 액션도 없는 항목이 7일 이상인 경우
