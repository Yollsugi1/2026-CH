## A-B-C Framework

### A (Subject + Context)
**Subject**: [Who is the main character?]  
**Context**: [What makes them special/interesting?]

```
Checklist:
□ Subject is special or interesting
□ Context is clear
□ Viewers would care about this person
```

### B (Action + Constraint)
**Action**: [What are they doing?]  
**Constraint**: [What makes this interesting/challenging?]

```
Checklist:
□ Relatable yet special
□ Visually interesting
□ Enhances or contrasts with A
```

### C (Emotion + Reason)
**Core Emotion**: [What should viewers feel?]  
**Reason to Watch**: [Why should they invest time?]  
**Expected Reaction**: [What should they do/think after?]

```
Checklist:
□ Answers "So what?"
□ Emotion is clear
□ Shareable/memorable
```
