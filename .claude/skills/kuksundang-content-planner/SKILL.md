# 국순당 콘텐츠 기획 스킬

## 시리즈 개요

### 1. 순당포착
- **컨셉**: 검색해도 안 나오는 진짜 이야기. 막걸리를 파는 가게와 사장님의 이야기를 포착
- **핵심**: "사람"이 만드는 것 — 술도, 공간도, 사람이 만든다
- **포맷**: 가게 방문 → 사장님 인터뷰 → 음식+막걸리 페어링 → 이야기
- **톤**: 따뜻하고 진정성 있는, 다큐멘터리 감성

### 2. 취하지아니한가
- **컨셉**: 막걸리와 함께하는 문화적 순간들
- **핵심**: 막걸리가 빛나는 상황과 감정을 포착
- **포맷**: 상황 설정 → 막걸리가 어울리는 이유 → 감성적 마무리
- **톤**: 감성적이고 공감 가는

### 3. 1막1장
- **컨셉**: 막걸리가 떠오르는 장면의 수집
- **핵심**: 소리, 풍경, 분위기 — 오감으로 느끼는 막걸리의 순간
- **포맷**: 감각적 영상 (소리 중심) → 짧고 임팩트 있는 구성
- **톤**: 시네마틱, ASMR 감성, 미니멀

## A-B-C 프레임워크 적용

각 에피소드 기획 시 반드시 A-B-C를 정의:

### A (주체 설정)
- **순당포착**: A = 특정 가게/사장님
- **취하지아니한가**: A = 특정 인물/상황
- **1막1장**: A = 특정 장소/풍경

### B (행위 설정)
- **순당포착**: B = 가게에서 음식과 막걸리를 나누는 행위
- **취하지아니한가**: B = 막걸리와 함께 하는 특별한 활동
- **1막1장**: B = 막걸리가 떠오르는 감각적 장면

### C (감상/이유)
- **공통**: "막걸리가 일상 속에서 특별해지는 순간"을 전달
- 각 시리즈가 다른 감정 채널로 접근
