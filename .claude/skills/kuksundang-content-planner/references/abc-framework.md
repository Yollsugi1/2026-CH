# A-B-C 프레임워크 상세 가이드 (국순당 적용)

## 기본 원리

모든 콘텐츠는 A(주체) + B(행위) = C(감상/이유)로 설명 가능해야 한다.

**핵심 규칙**: A 또는 B 중 하나만 특별해야 한다.
- A가 특별하면 B는 평범하게 (특별한 사람이 일상적인 일을 하는)
- B가 특별하면 A는 평범하게 (평범한 사람이 특별한 일을 하는)
- 둘 다 특별하면 → 산만함
- 둘 다 평범하면 → 재미없음

## 국순당 시리즈별 적용

### 순당포착 (A가 특별)
- **A**: 20년째 막걸리집을 지킨 사장님 (특별한 주체)
- **B**: 단골 손님에게 안주를 내어주는 (평범한 행위)
- **C**: "이 동네에 이런 곳이 있었어?" → 발견의 감동

예시:
- A: 3대째 전을 부치는 영동시장 할머니 + B: 막걸리와 전을 내어주는 = C: 세월이 만든 맛의 진정성
- A: 25년 골목 피자전 사장님 + B: 국순당 막걸리와 페어링하는 = C: 의외의 조합이 주는 놀라움

### 취하지아니한가 (B가 특별)
- **A**: 평범한 직장인/친구들 (평범한 주체)
- **B**: 벚꽃 아래서 막걸리 한 잔 (특별한 상황)
- **C**: "이런 순간에 막걸리가 딱이지" → 공감

### 1막1장 (감각이 특별)
- **A**: 특정 장소/시간 (분위기가 특별한 주체)
- **B**: 소리와 풍경을 담는 (감각적 행위)
- **C**: "이 장면에 막걸리가 떠오른다" → 감각적 연상

## 체크포인트

### A 체크포인트
- [ ] 주체가 누구인지 한 문장으로 설명 가능한가?
- [ ] 시청자가 "이 사람/장소를 더 알고 싶다"고 느끼는가?
- [ ] 국순당/막걸리와의 연결이 자연스러운가?

### B 체크포인트
- [ ] 행위가 시각적으로 표현 가능한가?
- [ ] 영상으로 보여줄 때 흥미로운가?
- [ ] A와의 조합이 의외성 또는 진정성을 만드는가?

### C 체크포인트
- [ ] "그래서 뭐?"에 명확히 답할 수 있는가?
- [ ] 시청자가 공유하고 싶어할 감정인가?
- [ ] 막걸리/국순당 브랜드에 긍정적 연상을 만드는가?

## 기획 검증 5가지 질문
1. **한 줄 설명**: A + B = C로 말해보기
2. **왜 봐야 해?**: 시청 동기가 명확한가
3. **누가 볼까?**: 1차/2차/3차 타겟
4. **썸네일은?**: 어떤 장면을 잡을 것인가
5. **제목은?**: 클릭하고 싶은 제목인가
