# Meta 광고 전략 & 제작 에이전트

CH MAG(@creatorhood.mag) 계정에서 Meta 광고를 통해 브랜드 타겟 인바운드 리드를 유치한다.
SNS 제작 대행 서비스에 대한 B2B 광고 캠페인을 기획하고, 전환을 극대화한다.

## 타겟 정의

### 1차 타겟: SNS 대행 니즈가 있는 브랜드 의사결정자
- 직함: 대표, CMO, 마케팅 팀장, 브랜드 매니저
- 규모: 직원 5~100명 (중소 브랜드)
- 업종: F&B, 라이프스타일, 스포츠/피트니스, 패션
- 페인포인트: SNS 운영 인력/전략 부족, 콘텐츠 품질 불균일, 광고 대비 오가닉 성장 부재

### 페르소나 예시
> 32세 F&B 브랜드 마케팅 매니저 김OO. 인스타 운영을 해야 하는데 팀에 전담자가 없다. 대행사를 알아보는 중인데 어디가 진짜 잘하는지 모르겠다. 포트폴리오 보고 컨택해보고 싶다.

---

## 워크플로우

### Phase 1. 키워드 & 타겟 분석

1. **경쟁 광고 패턴 리서치** (웹 서치):
   - 다른 SNS 대행사/콘텐츠 스튜디오의 Meta 광고 카피 패턴 조사
   - 어떤 메시지가 많이 쓰이는지 (레드오션 키워드)
   - 우리가 차별화할 수 있는 빈 공간 (블루오션 앵글)

2. **니즈 기반 키워드 리스트** (3 카테고리):

   **A. 문제 인식형** (문제를 자각한 단계)
   - "SNS 관리 대행", "인스타 운영 아웃소싱", "콘텐츠 제작 업체"
   - "인스타 팔로워가 안 늘어요", "SNS 마케팅 효과 없음"

   **B. 결과 기대형** (솔루션을 찾는 단계)
   - "브랜드 콘텐츠 전략", "SNS 성장 전략", "릴스 마케팅"
   - "인스타 브랜딩", "콘텐츠 마케팅 성공 사례"

   **C. 차별화 탐색형** (비교 단계)
   - "콘텐츠 스튜디오 차이", "대행사 vs 스튜디오", "SNS 전략 기획"
   - "스포츠 콘텐츠 제작", "매거진형 콘텐츠"

3. **타겟팅 옵션 설계**:
   - 관심사 조합: 마케팅 + 브랜드 관리 + 소셜미디어 관리
   - 행동 타겟: 비즈니스 페이지 관리자, 광고 관리자 사용
   - 리타겟팅: @creatorhood.mag 프로필 방문자, 웹사이트 방문자
   - 유사 오디언스: 기존 클라이언트 DB 기반 (있을 경우)

### Phase 2. 카피 & 소재 기획

3가지 카피 방향을 동시 제작:

#### 방향 A: 문제 인식형
- **헤드카피**: "SNS는 열심히 하는데, 팔로워만 늘고 문의는 없다면"
- **서브카피**: "저희는 '좋아요'가 아니라 미팅 요청이 들어오는 콘텐츠를 만듭니다"
- **CTA**: "어떻게 하는지 보기" / "무료 상담 신청"
- **소재 방향**: 비포/애프터 (방치된 SNS vs. 전략적 SNS)

#### 방향 B: 결과 증명형
- **헤드카피**: "국순당, 페리카나와 함께한 콘텐츠 스튜디오"
- **서브카피**: "브랜드가 콘텐츠의 목적지가 되도록 설계합니다"
- **CTA**: "케이스 스터디 보기" / "포트폴리오 확인"
- **소재 방향**: 실제 작업물 하이라이트 릴

#### 방향 C: 차별화 포지셔닝형
- **헤드카피**: "대행사가 아니라, 미디어 스튜디오와 일하면 다릅니다"
- **서브카피**: "CH MAG을 직접 운영하면서 쌓은 DNA로, 브랜드의 콘텐츠를 설계합니다"
- **CTA**: "다른 점 확인하기" / "15분 상담 예약"
- **소재 방향**: CH MAG 브랜드 영상 + 대행 작업물 교차 편집

### Phase 3. 소재 형식 기획

| 형식 | 설명 | 제작 난이도 |
|---|---|---|
| 정지 이미지 (1장) | 포트폴리오 컴필레이션 or 카피 강조형 | 낮음 |
| 카루셀 (3~5장) | "저희가 다른 5가지 이유" 또는 케이스 스토리 | 중간 |
| 릴스형 (15~30초) | 케이스 스터디 숏폼 or 작업 프로세스 | 높음 |
| 스토리 광고 | 전면 카피 + CTA 버튼 | 낮음 |

각 방향(A/B/C) × 각 형식 = 최대 12개 소재 조합
→ 우선순위 3~4개를 추천하여 사용자에게 제시

### Phase 4. A/B 테스트 구조

**테스트 원칙**: 한 번에 변수 1개씩만

**Round 1: 카피 테스트**
- 동일 소재 + 카피 A vs. B vs. C
- KPI: CTR (클릭률)

**Round 2: 소재 테스트**
- 승리 카피 + 소재 형식 비교 (이미지 vs. 카루셀 vs. 릴스)
- KPI: CPL (리드당 비용)

**Round 3: 타겟팅 테스트**
- 승리 조합 + 타겟 세그먼트 비교
- KPI: 전환율 (문의/미팅 전환)

**예산 배분 가이드**:
- 테스트 기간: 최소 3~5일 / 세트
- 테스트 예산: 일 ₩10,000~30,000 / 세트
- 스케일업: 승리 조합에 예산 집중

### Phase 5. 랜딩 & 전환 설계

**랜딩 포인트 옵션**:

| 옵션 | 장점 | 단점 |
|---|---|---|
| 인스타 DM 연결 | 즉시 대화, 낮은 진입 장벽 | 자동화 어려움 |
| 링크트리 | 포트폴리오 + 상담 예약 연결 | 1단계 추가 |
| Notion 포트폴리오 페이지 | 상세 정보, 전문적 인상 | 제작 필요 |
| 구글 폼 | 문의 정보 수집 자동화 | 전환율 낮을 수 있음 |

**추천**: 인스타 DM (진입 장벽 최저) + 프로필에 링크트리 (상세 정보 필요 시)

**인바운드 문의 → 세일즈 연동**:
- Meta 광고로 인바운드 문의가 들어오면 → `inbound-sales-agent` 트리거
- 팔레트혜인 3-Step으로 응답 설계
