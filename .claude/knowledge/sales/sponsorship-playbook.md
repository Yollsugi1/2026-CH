# CH MAG 스폰서십 & 브랜디드 콘텐츠 플레이북

> 용도: `sponsorship-agent`의 핵심 프레임워크
> 관련: brand-knowledge 스킬, quotation-builder 스킬

## 핵심 개념

CH MAG의 7개 콘텐츠 시리즈 + Zone X는 각각 고유한 오디언스와 톤을 가진 "미디어 채널"이다.
브랜드가 광고를 사는 것이 아니라, **콘텐츠 경험의 일부가 되는 것** — 이것이 Creatorhood 스폰서십의 핵심 가치 제안.

**포지셔닝**: "광고가 아니라, 브랜드가 콘텐츠의 '자연스러운 일부'가 되는 경험"

---

## 시리즈별 브랜드 매칭 매트릭스

### 1. Playbook (토크쇼) — 주 1회, 롱폼 20~50분
| 항목 | 내용 |
|---|---|
| 오디언스 | 스포츠/비즈니스에 관심 있는 2030 |
| 적합 브랜드 | 음료, 보충제, 앱/SaaS, 피트니스 장비 |
| 스폰서십 형태 | PPL (음료/보충제 자연스러운 사용), 타이틀 스폰서 ("Playbook, presented by X"), 미드롤 광고 |
| 장점 | 호스트의 신뢰감 있는 멘션, 장시간 노출, 반복 시청 |

### 2. Hoodies (크리에이터 브이로그) — 주 1회, 롱폼 5~15분
| 항목 | 내용 |
|---|---|
| 오디언스 | 라이프스타일에 관심 있는 2030, 수현의 팬층 |
| 적합 브랜드 | 패션, 여행, 라이프스타일, 카페/F&B |
| 스폰서십 형태 | PPL (착용/사용), 브랜디드 에피소드 (특정 브랜드와 일상 촬영), 장소 협찬 |
| 장점 | 가장 자연스러운 PPL 가능, 일상 속 브랜드 노출, 높은 친밀감 |

### 3. 폭력적인 식단 (숏폼 맛집투어) — 주 1회, 60~90초
| 항목 | 내용 |
|---|---|
| 오디언스 | 맛집/F&B에 관심 있는 전 연령 |
| 적합 브랜드 | F&B 브랜드, 배달앱, 키친웨어, 식재료 |
| 스폰서십 형태 | PPL (메뉴/식재료 노출), 콜라보 에피소드 (브랜드 매장 방문), 장소 협찬 |
| 장점 | 높은 도달률 (숏폼), 직접적인 F&B 매칭, 방문/주문 전환 |

### 4. Pit Stop (숏폼 카페 큐레이션) — 주 1회, 60초
| 항목 | 내용 |
|---|---|
| 오디언스 | 카페/디저트 문화에 관심 있는 2030 |
| 적합 브랜드 | 카페, 음료 브랜드, 디저트, 커피 머신 |
| 스폰서십 형태 | PPL (음료 노출), 장소 협찬 (카페 소개), 콜라보 메뉴 |
| 장점 | 직접 방문 유도, 로컬 비즈니스와 높은 매칭도 |

### 5. Shirtless (숏폼 스타일링) — 격주 1회, 60~90초
| 항목 | 내용 |
|---|---|
| 오디언스 | 패션/스포츠웨어에 관심 있는 남성 2030 |
| 적합 브랜드 | 패션, 스포츠웨어, 향수, 그루밍 |
| 스폰서십 형태 | PPL (착용 스타일링), 브랜디드 콘텐츠 (특정 브랜드 전체 룩 구성) |
| 장점 | 직접적인 제품 노출, 구매 전환 유도, 비주얼 임팩트 |

### 6. False 9 (시네마틱 필름) — 월 1~2회, 5~10분
| 항목 | 내용 |
|---|---|
| 오디언스 | 프리미엄 콘텐츠 선호 시청자 |
| 적합 브랜드 | 프리미엄 브랜드, 자동차, 시계, 스포츠웨어 플래그십 |
| 스폰서십 형태 | 브랜디드 필름 (브랜드 스토리를 시네마틱으로), 타이틀 스폰서 |
| 장점 | 높은 제작 퀄리티, 브랜드 이미지 제고, 장기 재생 |

### 7. Zone X (크로스핏 리그) — 시즌제
| 항목 | 내용 |
|---|---|
| 오디언스 | 크로스핏/기능성 피트니스 커뮤니티 + 일반 스포츠 팬 |
| 적합 브랜드 | 스포츠웨어, 피트니스 장비, 보충제, 스포츠 음료 |
| 스폰서십 형태 | 타이틀 스폰서 (이벤트명), 참가자 용품 협찬, 이벤트 부스, 콘텐츠 내 브랜딩 |
| 장점 | 이벤트+콘텐츠 통합 노출, 커뮤니티 직접 접점, 장기 파트너십 |

## 스폰서십 티어 체계

### Tier 1: 타이틀 스폰서
- 시리즈/이벤트 명에 브랜드명 포함
- 모든 에피소드에 브랜드 노출
- 전용 브랜디드 콘텐츠 제작
- 월간 퍼포먼스 리포트
- 독점 카테고리 (동일 업종 다른 브랜드 불가)

### Tier 2: 골드 스폰서
- 에피소드당 PPL + 멘션
- 인트로/아웃트로 브랜드 로고
- 분기 1회 전용 브랜디드 에피소드
- 월간 성과 요약

### Tier 3: 실버 스폰서
- 월 N회 PPL 노출
- SNS 피드 1회 브랜드 태그
- 분기 성과 요약

### Tier 4: PPL / 단발
- 에피소드 1회 PPL
- 제품/장소 제공 + 노출
- 가장 진입 장벽 낮음

## 브랜디드 콘텐츠 전환 프로세스

기존 시리즈 에피소드를 브랜디드 콘텐츠로 전환하는 프로세스:

### 1단계: 매칭
- 브랜드의 타겟 오디언스 × 시리즈의 시청자 겹침 분석
- 브랜드의 톤앤매너 × 시리즈의 톤 적합성 확인
- 자연스러운 통합이 가능한 에피소드 컨셉 도출

### 2단계: 기획
- 에피소드 내 브랜드 등장 시나리오 설계
- "광고 같지 않은" 통합 방식 결정:
  - 자연스러운 사용 (음료를 마시며 대화)
  - 장소 배경 (카페/레스토랑에서 촬영)
  - 스토리라인 일부 (브랜드 관련 주제를 토크에 포함)

### 3단계: 제작
- 브랜드 가이드라인 반영하되 CH MAG 톤 유지
- 브랜드 리뷰 → 수정 → 최종 승인

### 4단계: 배포 + 리포트
- CH MAG 채널 배포 (YouTube + Instagram)
- 성과 리포트: 조회수, 도달, 인게이지먼트, 댓글 반응

## Zone X 스폰서십 패키지

### 이벤트 스폰서십
- 컴바인 (Season 0): 2박 3일 이벤트 전체 브랜딩
- 리그 (Season 1+): 시즌 단위 파트너십
- 참가자 용품: 공식 운동복/장비 제공 + 로고 노출

### 콘텐츠 스폰서십
- The Proving Ground (빌드업 시리즈): 에피소드 내 제품 노출
- 리그 하이라이트: 경기 영상 내 브랜딩
- 선수 프로필 콘텐츠: 개인 스토리 + 브랜드 연결

### 현장 활성화
- 이벤트 부스 운영
- 샘플링/체험존
- 포토존 브랜딩

## 제안 작성 가이드

sponsorship-agent가 제안서를 작성할 때의 구조:

### 이메일 제안 구조
1. **왜 이 브랜드인가** (옥성아 B2B 변형의 세계관 연결)
2. **CH MAG/Zone X 소개** (2~3문장 핵심만)
3. **매칭 시리즈 + 스폰서십 형태** (구체적 시나리오)
4. **예상 노출/효과** (채널 규모, 타겟 오디언스)
5. **다음 단계** (미팅 or 제안서 전달)

### proposal-docx 연동 시 추가 포함
- 시리즈 소개 + 오디언스 데이터
- 스폰서십 티어 옵션 (3단계 앵커링)
- 예상 타임라인
- 성과 측정 방법

## 가격 설정 참고

스폰서십 가격은 아래 요소를 종합하여 결정:
- 시리즈의 월 도달 규모
- 에피소드당 제작 비용
- 브랜드 노출 빈도/기간
- 독점 여부
- 추가 제작물 (전용 콘텐츠) 포함 여부

> 구체적인 가격은 `quotation-builder` 스킬의 서비스 카탈로그와 연동하여 산출
