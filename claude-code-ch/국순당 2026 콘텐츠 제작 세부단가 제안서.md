#### 비용 구조

- 풀세트: 캐러셀 1개 + 숏폼 2개 구성
- 축소세트: 캐러셀 1개 + 숏폼 1개 구성

- **1회 촬영 시 발생하는 고정금액 ₩1,600,000**: 기획(₩200,000) + 답사(₩150,000) + 촬영(₩600,000) + 사진보정(₩150,000) + 캐러셀 디자인(₩200,000) + 영상기획 전체(₩300,000)

| 세트 구성 | 1회 촬영 시 고정금액 | 영상 변동비 | 합계 |
|---|---|---|---|

> 축소세트의 경우, 1회 촬영에 필요한 기획·답사·촬영·캐러셀 편집·영상기획 전체 등 고정 공수가 동일하게 투입되므로, 숏폼 1개를 줄여도 전체 비용이 정비례(1/3)로 감소하지 않고 영상 변동비(₩550,000)만 차감됩니다.

### 1-3. 순당포착 vs 1막1장 단가 비교

두 시리즈는 촬영·편집의 성격이 다르기 때문에 단가가 다르게 책정되었습니다.

| 비교 항목 | 순당포착 | 1막1장 |
|---|---|---|
| 세트당 총 M/H | 54시간 | 31시간 |
| 촬영 특성 | 인물 화보(스틸) + 인터뷰(영상) 동시 촬영 | TPO 컨셉 영상 촬영 |
| 추가 편집 | 사진보정 + 캐러셀 디자인 | 촬영 색보정 |
| 숏폼 1편당 변동비 | ₩550,000 (기획 ₩150,000 + 편집 ₩400,000) | ₩300,000 (편집) |
| 고정금액 비중 | 59.3% (₩1,600,000/₩2,700,000) | 61.3% (₩950,000/₩1,550,000) |

> 순당포착은 사전 답사, 인물 섭외, 화보+인터뷰 동시 촬영, 사진 보정, 캐러셀 디자인까지 풀 프로세스를 거치므로 1막1장 대비 약 1.7배의 공수가 투입됩니다. 또한 각 숏폼에 대한 세부 기획(1명×3h)이 별도로 필요하여 영상 1편당 변동비도 더 높게 책정되어 있습니다.

이러한 단가 구조는 각 콘텐츠에 투입되는 인원과 시간을 기준으로 투명하게 산출하되, 촬영별 고정금액과 영상별 변동비를 명확히 분리하여 "추가 1편의 비용"이 직관적으로 드러나도록 설계되었습니다. 이를 통해 구성을 변경하실 때 비용 변화를 사전에 정확하게 파악하실 수 있으며, 시리즈 간 단가 차이의 근거도 명확하게 확인하실 수 있습니다.

### 2-2. 비용 구조 설계 배경 및 할인 적용 방식

본 단가 체계는 국순당과의 장기적인 파트너십을 전제로, 비용의 투명성과 합리적인 부담 구조를 모두 고려하여 설계하였습니다.

**투명한 산출 기준**
모든 비용은 투입 인원 × 시간 × 시급(₩50,000)으로 산출되며, 촬영별 고정금액과 영상별 변동비를 분리하여 구성 변경 시 비용 변동을 명확하게 예측하실 수 있습니다.

**파트너십 할인**
현재 구성의 정가 합계는 월 ₩8,000,000이나, 지속적인 협업 관계를 반영하여 ₩500,000을 할인 적용하고 있습니다(월 ₩7,500,000).

**볼륨 연동 할인**
콘텐츠를 추가하실 경우, 제작 물량이 증가하는 만큼 할인 금액도 함께 커지는 구조를 적용합니다. 이를 통해 추가 콘텐츠의 실질 부담이 정가 대비 낮아지며, 더 많이 제작하실수록 편당 비용이 유리해집니다.

**만단위 절삭**
모든 최종 금액은 만원 단위로 내림 처리하여 항상 고객에게 유리하게 적용합니다.

**축소세트 및 유연한 구성**
예산 변동이나 콘텐츠 전략 변화에 대응할 수 있도록, 순당포착에 축소세트(캐러셀1+숏폼1)를 별도로 설계하였습니다. 또한 현재 월 계약금액(₩7,500,000) 범위 내에서 구성을 조정하시는 경우에는 산출금액 기준으로 안내드리며, 현재 구성을 초과하는 경우에만 볼륨 할인이 적용되어 확장 시 실질 부담을 낮추는 구조입니다.

### 2-5. 할인 구조 요약

현재 파트너십에서 적용하고 있는 비용 구조를 정리하면 다음과 같습니다.

| 정가(Raw) 구간 | 적용 방식 | 설명 |
|---|---|---|
| ₩7,500,000 미만 | 산출금액 그대로 적용 | 축소 구성 시 정가 기준으로 투명하게 안내 |
| ₩7,500,000 ~ ₩8,000,000 | 최저 ₩7,500,000 적용 | 이 구간에서는 최저 계약금액이 적용되어 현재 구성 유지가 가장 효율적 |
| ₩8,000,000 (현재) | ₩500,000 할인 → ₩7,500,000 | 장기 파트너십 기반 할인 |
| ₩8,000,000 초과 | 볼륨 연동 할인 | 물량 증가에 비례하여 할인 금액 확대 |

**볼륨 연동 할인 상세**

- 정가 ₩9,000,000대 → 할인 ₩700,000 ~ ₩900,000 (실질할인율 7~9%)
- 정가 ₩10,000,000대 → 할인 ₩850,000 ~ ₩1,000,000 (실질할인율 8~9%)
- 정가 ₩11,000,000 이상 → 할인 ₩1,000,000 이상 (실질할인율 9~10%)

**추가 비용 배려 사항**

- **축소세트 옵션**: 예산 변동에 유연하게 대응할 수 있도록 캐러셀1+숏폼1 구성을 별도 설계하였습니다.
- **숏폼 단위 추가**: 대부분의 시나리오에서 1막1장 숏폼 1편 추가 시 실질 추가 비용은 ₩250,000 ~ ₩300,000입니다. 이는 정가 ₩300,000 대비 볼륨 할인이 함께 적용되기 때문입니다.
- **만단위 절삭**: 모든 최종 금액은 만원 단위로 내림 처리하여 항상 고객에게 유리하게 적용합니다.
