## Layer 1: 핵심 사업 영업

### 이것은 무엇인가
크리에이터후드의 **자체 IP이자 핵심 사업**. 이미 기획·제작 체계가 잡혀 있고, 한상필 대표의 영업력으로 **새로운 세일즈(스폰서십, 브랜드 딜)**를 만들어내는 구조.

### 프로젝트

**Zone X (스포츠 리그)**
- 스폰서십 1.5억 목표
- 한상필 대표: 스포츠·피트니스 브랜드 대상 스폰서십 영업
- 영업 도구: Zone X 스폰서십 제안서

**CH MAG 정규 시리즈 (7종)**

## Layer 2: 함께 키우는 뉴미디어 채널

### 이것은 무엇인가
크리에이터후드와 한상필 대표가 **신규로 같이 만드는 프로젝트**. CH MAG과 다른 포지션의 서브 매거진 채널 2개를 운영하며, 핵심 의의는 **한상필 대표의 브랜드 네트워크가 광고로 활용할 수 있는 구좌(인벤토리)를 만드는 것**.

### 왜 Layer 2가 중요한가

1. **광고 구좌 생성**: 한상필 대표가 영업할 수 있는 구체적인 광고 상품이 생김
2. **공동 자산**: Layer 1은 크리에이터후드 IP → 한상필 대표 입장에서 "함께 만든" 채널은 사업 동기 부여
3. **조직 레버리지**: 인턴 4명 → 대학생 에디터 20~30명 관리 → 콘텐츠 볼륨 확보
4. **바이럴 → 팔로워 성장 → 광고 단가 상승**: 초기 투자로 자산이 커지는 구조
5. **인력 파이프라인**: 대학생 에디터 → CH MAG 기여 → 핵심팀 합류의 성장 경로

### CH MAG과의 역할 분담

| | CH MAG (Layer 1) | 서브채널 (Layer 2) |
|---|---|---|
| 콘텐츠 | 오리지널 시리즈, 공수 높은 퀄리티 | 빠르게, 많이, 시의성, 포맷 중심 |
| 속도 | 주 단위 정규 발행 | 일 단위 발행 |
| 제작 주체 | 핵심팀 (수현, 지운, 우석, 임용) | 인턴 매니저 4명 + 대학생 에디터 |
| 광고 모델 | 브랜디드 콘텐츠, PPL, 협업 | **광고 구좌 판매** (한상필 영업) |
| 핵심팀 부하 | Full | **최소** (수현 디렉션만) |

### 인력 구조

```
수현 (디렉션, 주간 1회 체크인)
  └─ 인턴 매니저 4명
       ├─ 운영 총괄 PM (1명)
       ├─ 커뮤니케이션 & 섭외 리드 (1명)
       ├─ 채널 A 콘텐츠 디렉터 (1명)
       └─ 채널 B 콘텐츠 디렉터 (1명)
            └─ 대학생 에디터 (채널별 5~10명 모집)
```

### 비용 구조

| 항목 | 월 비용 | 기간 | 합계 | 비고 |
|---|---|---|---|---|
| 인턴 매니저 4명 | ~100만원 (인당 100~150만) | 3~6월 (4개월) | ~500만원 | 서브채널 전담 |
| 대학생 에디터 | 건바이건 기고료 | 상시 | ~50만원/월 추정 | 대외활동 형태 |
| 콘텐츠 제작비 | 미미 | 상시 | — | 촬영장비·식비 정도 |
| **Layer 2 소계** | **~150만원/월** | | **~600만원/H1** | |

### 예상 매출 (채널 성장에 따라)

| 시나리오 | 팔로워 규모 | 월 광고 매출 가능성 | 시기 |

## Layer 3: 한상필 대표 추가 매출

### 이것은 무엇인가
한상필 대표가 **자신의 기존 네트워크에서 가져오는 건**을 법인이 실행하는 구조. 본질적으로 **대대행/실행 대행**이며, 법인 매출 확보가 목적.

### 핵심 원칙: 리소스 컨트롤

| 원칙 | 설명 |
|---|---|
| **외주 중심 실행** | 내부 핵심팀 리소스를 쓰지 않고, 외주 파트너로 실행 |
| **마진 구조 확보** | 수주액 - 외주비 = 순수익, 내부 공수 최소화 |
| **건바이건 판단** | 내부 리소스가 빨리면 안 됨 → 건별로 수현이 GO/NO-GO 판단 |
| **Layer 1 보호** | Layer 3 때문에 핵심 사업의 퀄리티·일정이 밀리면 안 됨 |
