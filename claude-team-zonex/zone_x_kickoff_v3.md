## 1. Zone X 프로젝트 개요

### 한 줄 요약
**Zone X는 서로 다른 트레이닝을 하는 8개 팀/크루가 카디오 미션으로 10주간(격주 리그 + 연속 챔피언십) 경쟁하는 팀 카디오 스포츠 엔터테인먼트 시리즈(IP)입니다.**

### 핵심 포지셔닝
- **피지컬100의 리얼리티** + **스우파의 팀 팬덤** → 카디오 종목으로 결합한 새로운 포맷
- "피지컬 트레이닝 버전의 스트릿 우먼 파이터"
- 기존 대회는 **"스포츠 이벤트"** → Zone X는 **"스포츠 엔터테인먼트 시리즈"**

### Zone X의 의미
- **Xtreme**: 심폐 Zone 5를 넘어서는 한계 도전
- **미지수 X**: 이전에 없던 새로운 유형의 팀+리그+카디오 컴피티션
- **크로스오버 X**: 크로스핏, 하이록스, MMA, 러닝, 맨몸운동 등이 모여 경쟁

### 기존 대회 vs Zone X

| 구분 | 기존 대회 (하이록스/크로스핏) | Zone X |
|------|--------------------------|--------|
| 성격 | 하는 사람의 기록 경쟁 | **보는 사람의 몰입**에 최적화 |
| 기간 | 1-2일 일회성 이벤트 | **격주 시즌제 리그 (10주)** |
| 진입장벽 | 높은 기술/고중량 (크로스핏) | 저스킬, 고강도 카디오 중심 |
| 콘텐츠 | 하이라이트 몇 편 | **22편+ 시즌 시리즈** |
| 경기시간 | ~1시간 (하이록스) | **1~5분 미션**, 짧고 굵은 승부 |
| 형태 | 개인 참가 | **팀전** (전략적 요소 + 서사) |

## 2. 리그 구조

### 전체 구조: 격주 리그 4R + 연속 챔피언십 2R (총 10주)

```
┌──────────────────────────────────────────────────────┐
│  LEAGUE ROUND (격주 진행, 8주에 걸쳐 4라운드)            │
│  • Round 1 (W1): Head-to-Head Match (1:1 탐색전)      │
│     ── 1주 휴식 ──                                     │
│  • Round 2 (W3): King of the Hill (1:1:1 영토전쟁)     │
│     ── 1주 휴식 ──                                     │
│  • Round 3 (W5): The Vault (2:2 연합매치)              │
│     ── 1주 휴식 ──                                     │
│  • Round 4 (W7): Auction Draft Challenge (경매+실행)    │
│  → 상위 4팀 선발                                        │
└──────────────────────────────────────────────────────┘
                        ↓
┌──────────────────────────────────────────────────────┐
│  CHAMPIONSHIP ROUND (2주 연속 진행)                     │
│  • Week 9: Semi-Finals (3-Stage Battle)              │
│  • Week 10: Finals (Best of 7)                       │
│  → Champion 탄생                                      │
└──────────────────────────────────────────────────────┘
```

## 4. 운영인력 & 역할 분담

### 조직 구조

```
                         정수현 (총괄 디렉터)
          ┌──────┬──────┬──────┬──────┐
       고수민   김세한   석예나   신희준   김나강
```

모든 인원이 정수현 직속으로, 각자의 역할 범위 내에서 오너십을 갖고 운영.

### 역할 배분 근거

**고수민 · 김세한** → 실무 경험이 있어 외부 브랜드·파트너 커뮤니케이션에서 실무적 관점 적용 가능. Zone X에 시간 투입 비중이 높을 것으로 예상되어, 제안서·스폰서십·파트너 관계 등 **대외적 추진력** 담당.

**석예나 · 신희준 · 김나강** → CH MAG 매거진 에디터 연속성. 다른 프로젝트에도 투입되어 Zone X에는 상대적으로 한정된 시간 배분. SNS 운영, 현장 홍보물, 후속 문서 정리 등 **오퍼레이션 중심** 역할.

### 역할별 정의

| 이름 | 포지션 | 핵심 오너십 |
|------|--------|------------|
| **정수현** | 총괄 디렉터 | 프로젝트 비전·방향, 핵심 파트너 초도 미팅, 콘텐츠 디렉션, Go/No-Go |
| **고수민** | 전략·제안서·실무 운영 | 협찬제안서 작성·관리, 스폰서 컨택 리스트, 팀 섭외 실무, 계약 조건 정리 |
| **김세한** | 브랜드·파트너 관계 구축 | 브랜드 미팅·관계 구축, 외부 파트너십 관리, 예산·WBS 트래킹 |
| **석예나** | 후속 커뮤니케이션·문서 정리 | 미팅 후속 문서 정리, 파트너/팀별 커뮤니케이션 이력 관리, 후속 액션 트래킹 |
| **신희준 · 김나강** | SNS 마케팅·현장 홍보물 | SNS 채널 기획·운영, 현장 운영용 홍보물(배너/현수막/사이니지 등) 기획·제작 |

> ※ 게임 설계, 콘텐츠 촬영·편집 담당은 **추후 별도 확정**

### 고수민 ↔ 김세한 협업 흐름

두 사람은 명확한 벽이 아니라 자연스럽게 이어지는 협업 구조.

```
제안서 작성 (고수민)
  → 브랜드 미팅 (김세한 + 고수민)
  → 관계 유지·후속 미팅 (김세한)
  → 계약 조건 정리 (고수민)
  → 최종 합의 (정수현)
```

### 역할 경계 요약

| 업무 영역 | 오너 | 서포트 |
|----------|------|--------|
| 프로젝트 방향·의사결정 | 정수현 | 고수민, 김세한 |
| 협찬제안서·계약 실무 | 고수민 | 정수현(리뷰), 김세한(미팅) |

## 5. 업무 흐름도

### 스폰서십 유치

```
정수현: 타겟 브랜드 방향 결정
  → 고수민: 제안서 작성, 컨택 리스트
  → 김세한 + 고수민: 브랜드 미팅
  → 김세한: 관계 유지, 후속 미팅
  → 석예나: 미팅 내용 정리, 후속 액션 문서화
  → 고수민: 계약 조건 정리
  → 정수현: 최종 합의
```

### 팀 섭외

```
정수현: 핵심 팀 직접 컨택
  → 고수민: 팀 섭외 자료 발송, 일정 조율
  → 석예나: 미팅 후 후속 정리, 팀별 상태 트래킹
  → 정수현: 최종 확정
```

### SNS · 홍보물

```
정수현: 채널 방향성, 브랜딩 톤앤매너 결정
  → 신희준 + 김나강: SNS 콘텐츠 캘린더, 콘텐츠 제작·배포
  → 신희준 + 김나강: 현장 홍보물 디자인·제작
  → 정수현: 주요 콘텐츠 최종 컨펌
```

### 경기일 현장

```
정수현 ─── 현장 총괄, 돌발 의사결정
고수민 ─── 현장 로지스틱스, 스폰서 케어
김세한 ─── 스폰서 현장 응대, 파트너 관리
석예나 ─── 현장 커뮤니케이션 기록, 이슈 정리
신희준 + 김나강 ─── 실시간 SNS, 현장 홍보물 관리
(게임 진행·촬영·편집 ─── 담당자 추후 확정)
```

## 8. WBS 업데이트 참고

기존 WBS(스프레드시트)는 11월 시작 → 3월 대회 기준으로 설계되어 있으므로, 6월 시작에 맞춰 전체 리셋 필요.

### 기존 WBS 5단계 → 6월 기준 재배치

| 기존 단계 | 기존 시기 | 목표 | → 신규 시기 |
|----------|----------|------|-----------| 
| 1단계 | 11~12월 | 기획 완료, 제안서, 콘텐츠 기획 | → **2~3월** |
| 2단계 | 12~1월 | 팀 섭외 완료, 메인 스폰서십, 디자인 시작 | → **3~4월** |
| 3단계 | 1월 | 디자인 완료, 웹/인스타/티저, 게임 자문 | → **4월** |
| 4단계 | 1~2월 | 채널/티저 런칭, 스폰서십 완료, 촬영/시뮬레이션 | → **5월** |
| 5단계 | 2~3월 | SNS 마케팅, 리그 웹 세팅, 모집 오픈 | → **5월 말~6월 초** |

### WBS 주요 작업 항목

| WBS# | 작업 | 담당 | 비고 |
|------|------|------|------|
| 1.1 | 전체 이벤트(리그) 기획 | CH | ✅ 대부분 완료, 격주 반영 업데이트 |
