## 1. 프로젝트 배경

### 목표
- Zone X 스폰서십 제안서를 브랜드(스폰서) 관점에서 전면 재기획

- 고수민에게 세일즈/비즈니스 이해도를 높이는 교육적 맥락도 포함

### 확정 사항

- **산출물 형태**: PDF
- **미팅 활용**: 완성본 뽑아보되, 미팅에서 흐름 중심으로 다시 작업

## 2. 기존 제안서 핵심 문제점 분석

### 문제 1: "우리가 뭘 하는지"에 치중 → "브랜드가 뭘 얻는지"가 약함
- 현재 제안서는 Zone X가 무엇인지, 어떤 게임을 하는지, 콘텐츠를 어떻게 만드는지를 상세히 설명
- 브랜드 의사결정자가 가장 먼저 묻는 질문 "그래서 우리가 여기에 돈을 쓰면 뭘 얻죠?"에 대한 답이 28페이지 중 뒤쪽에 가서야 나옴

### 문제 2: "첫 대회"라는 리스크에 대한 설득이 부족
- "First Mover Advantage"라고 적혀 있지만, "검증 안 된 대회에 왜 투자하냐"는 반론이 더 강함
- 블랙컴뱃, F1 DTS, 스우파 같은 좋은 레퍼런스가 있지만 유기적으로 연결되지 못함

### 문제 3: 차별성이 나열식 → "한 줄로 꽂히는" 포지셔닝 없음
- 팀전, 6주 시즌제, 콘텐츠 시리즈, 다종목 크로스오버 등 훌륭한 차별점들이 많지만 나열되어 있음
- "그래서 Zone X는 한마디로 뭔데?"에 대한 명확한 답이 없음

---

## 3. 재기획 방향: "브랜드의 의사결정 여정"을 따라가는 구조

현재 제안서가 "Zone X 설명서"라면, 새 제안서는 **"브랜드를 설득하는 여정"**으로 바꿔야 함.

| 단계 | 브랜드의 머릿속 질문 | 제안서가 해야 할 일 |
|---|---|---|
| ① Hook | "이게 뭔데?" | 3초 안에 꽂히는 한 줄 정의 |
| ② 시장 기회 | "지금 왜?" | 피지컬100/하이록스/스우파 트렌드 → 빈 자리 제시 |
| ③ 컨셉 | "구체적으로 뭐하는 건데?" | 직관적 비유 + 핵심 차별점 3가지 |
| ④ 브랜드 가치 | "우리가 뭘 얻지?" | 콘텐츠 산출물, 노출량, 타겟 도달 → 숫자로 |
| ⑤ 리스크 해소 | "첫 대회인데 괜찮아?" | 참가팀 파워 + 콘텐츠 스튜디오 역량 + 레퍼런스 |
| ⑥ 패키지 | "얼마에 뭘 주는 건데?" | 명확한 티어별 제공사항 |
| ⑦ 신뢰 | "누가 하는 건데?" | Creatorhood 소개 + 실적 |
