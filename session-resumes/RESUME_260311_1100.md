## 이번 세션 결정 사항

### 네비게이션 구조
- 단일 행: `[CH MAG] DAILY SERIES FEATURED ... About Contributor 🔍 Contact`
- CH MAG = 홈버튼 (가장 큼), 대분류 3개는 약간 작게
- 검색 아이콘 기준으로 좌=콘텐츠 네비, 우=매거진 정보 패널
- 호버 → 하위 카테고리 드롭다운 / 클릭 → 대분류 서브홈

### 대분류 매핑
| 대분류 | 하위 콘텐츠 |
|--------|------------|
| DAILY | 캐러셀 콘텐츠, 봤어, 스낵 콘텐츠 |
| SERIES | Pit Stop, Playbook, Hoodies, False 9, 후드의 주민들, 폭력적인 식단, Shirtless |
| FEATURED | 캐러셀 중 월간 반복 코너 (월간 승부처, 월간 잘산템 등) — 태그 기반 필터링 |

### 홈페이지 스크롤 섹션 순서
1. Hero/Featured — 옆으로 넘기는 캐러셀
2. Latest — 최신 콘텐츠
3. 숏폼 섹션 — Pit Stop, 폭력적인 식단 (9:16)
4. Playbook 섹션 — 토크/팟캐스트 (16:9)
5. Featured Articles — 월간 콘텐츠
6. Vlog 섹션 — Hoodies, 후드의 주민들
7. See More → Footer

### 서브홈 페이지
- DAILY → 캐러셀/봤어/스낵이 섹션으로 나뉘어 보이는 페이지
- SERIES → 각 시리즈가 섹션 단위로 Interactive하게 표현
- FEATURED → 월간 코너별로 묶여서 보이는 페이지

### 기타
- "캐러셀" = 인스타 캐러셀 형식 데일리 콘텐츠 통칭
- FEATURED = 별도 콘텐츠가 아닌 캐러셀의 큐레이션 뷰 (태그 필터)
