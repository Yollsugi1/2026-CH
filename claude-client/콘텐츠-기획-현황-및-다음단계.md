## 🎯 다음 세션에서 이어갈 콘텐츠 기획 작업

### 즉시 진행 가능 (우선순위 높음)

#### 1. 순당포착 EP.4+ 기획
- 다음 촬영 대상 가게 선정 필요
- A-B-C: A(새 가게/사장님) + B(음식+막걸리 페어링) = C(발견의 감동)
- `/plan 순당포착 - [가게명/지역]` 으로 바로 기획 시작 가능

#### 2. 1막1장 3월호 "신학기 (대학가)" 세부 기획
- 컨셉: 대학가의 봄, 신학기 분위기 + 막걸리
- 필요한 결정: 촬영 장소 (어느 대학가?), 시간대, 소리 포인트
- `/plan 1막1장 3월 - 신학기 대학가` 로 시작

#### 3. 취하지아니한가 시리즈 본격 기획
- 시리즈 전체 방향성 확정 필요
- EP.1 아이디어 브레인스토밍
- `/plan 취하지아니한가 EP.1` 로 시작

### 중기 기획 (3~4월 업로드 대비)

#### 4. 1막1장 4~6월 에피소드 세부 기획
- 월별 계절감 반영한 장소/분위기 확정
- 촬영 스케쥴 역산 필요 (업로드 D-14 기준 촬영 완료)

#### 5. 순당포착 시즌 로드맵
- 2026년 상반기 촬영 대상 가게 리스트업

## 기획할 때 이렇게 사용하세요

### Plugin 사용 (임우석 Cowork에서)
```
/plan 순당포착 - 영등포 어묵집 30년 된 곳
```
→ A-B-C 프레임워크 적용한 기획안 자동 생성

### 기획 프로세스
1. 아이디어/소재 제시 (가게, 상황, 장소 등)
2. 시리즈 매칭 (순당포착? 1막1장? 취하지아니한가?)
3. A-B-C 정의
4. 기획안 작성 (촬영 포인트, 구성, 캡션 방향)
5. 5가지 검증 (한 줄 설명, 왜 봐야해?, 누가 볼까?, 썸네일, 제목)
6. Notion DB 등록
