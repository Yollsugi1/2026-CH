# 크리에이터후드 데일리 대시보드

국순당 + CH MAG 업무를 한눈에 확인하는 로컬 웹 대시보드.

## 구성

| 파일 | 역할 |
|------|------|
| config.py | Notion 토큰, DB ID, 서버 설정 |
| notion_client.py | Notion API 클라이언트 |
| app.py | 대시보드 메인 서버 (다크 테마) |
| requirements.txt | Python 의존성 |
| setup.sh | 원클릭 설치 + macOS 자동 실행 설정 |

## 설치 (수현 STEP)

1. Notion → 설정 → 내 연결 → 새 API 통합 → **내부 통합** 생성
   - 이름: `크리에이터후드-대시보드`
   - 워크스페이스: Suhyun
2. 생성된 토큰 (`ntn_...`) 복사
3. 국순당 관련 DB 3개에 각각 **연결 추가**:
   - 내부 업무관리/제작 캘린더
   - 2026 국순당 콘텐츠 마스터
   - 클라이언트 공유 보드
4. 토큰을 우석에게 Slack DM으로 전달

## 설치 (우석 STEP)

`우석-설치가이드-완전초보용.md` 참고

## 기능

- **오늘/이번주/롱텀** 업무 자동 분류 (D-minus 기준)
- **국순당 콘텐츠 파이프라인** 시리즈별 현황
- **3단계 공유 시스템** 알림 (1차 D-14, 2차 D-10, 최종 D-5)
- **모닝 체크리스트** 인터랙티브 (로컬 저장)
- **CH MAG 메모** 자동 저장
- **macOS launchd** 매일 오전 10시 자동 실행
