## 업데이트 방법 (2분 소요)

### 1단계: 새 파일 받기
수현이 보내준 최신 `wooseok-dashboard` 폴더를 받아서, 기존 폴더를 **통째로 교체**해줘.

> 기존 위치: 아까 넣어둔 곳 (데스크탑이나 홈 폴더)

### 2단계: 터미널에서 실행
```bash
cd ~/Desktop/wooseok-dashboard
```
(폴더 위치가 다르면 그에 맞게 변경)

```bash
bash setup.sh
```

- "매일 오전 10시 자동 실행 설정을 진행할까요?" → **y 입력**

### 3단계: 테스트
```bash
python3 test.py
```

이렇게 나오면 성공:
```
=== 내부 업무관리 테스트 ===
N개 업무 발견
  [촬영] 순당포착 EP.XX - 03/15
  ...

=== 콘텐츠 마스터 테스트 ===
N개 콘텐츠 발견
  [순당포착] 에피소드 제목 - 세부기획
  ...
```

**0개 업무 발견** 이라고 나오면 수현에게 연락!

### 4단계: 대시보드 열기
```bash
python3 app.py
```

브라우저에 대시보드가 열리고, 실제 업무 데이터가 보이면 완료!

## 자동 실행 동작 방식
- **매일 오전 10시**: 맥이 켜져있으면 자동으로 대시보드 열림
- **10시에 맥이 꺼져있었던 경우**: 맥을 켜면 바로 대시보드 열림
- **수동 실행**: 언제든 `python3 app.py` 로 직접 열 수 있음
