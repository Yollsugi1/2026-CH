# 우석 데일리 대시보드 설치 가이드

> 개발 경험 없어도 OK! 하나씩 따라하면 됩니다.

---

## STEP 1: 수현님에게 토큰 받기

수현님이 Notion Internal Integration을 만들고 토큰을 줄 거예요.
토큰은 `ntn_` 으로 시작하는 긴 문자열입니다.

**Slack DM으로 받으면 됩니다.**

---

## STEP 2: 폴더를 바탕화면에 놓기

이 `wooseok-dashboard` 폴더 전체를 **바탕화면**에 놓으세요.
(다른 곳이어도 괜찮지만, 바탕화면이 가장 찾기 쉬워요)

---

## STEP 3: config.py에 토큰 입력

1. `wooseok-dashboard` 폴더 안에서 `config.py` 파일을 찾으세요
2. **마우스 오른쪽 클릭** → **텍스트 편집기로 열기** (또는 VS Code)
3. 맨 위쪽에 이 부분을 찾으세요:

```
NOTION_TOKEN = "여기에_노션_인테그레이션_토큰_입력"
```

4. `여기에_노션_인테그레이션_토큰_입력` 부분만 지우고, 수현님에게 받은 토큰을 붙여넣으세요

```
NOTION_TOKEN = "ntn_XXXXXXXXXXXXXXXXXXXXX"
```

⚠️ **주의사항**:
- 양쪽 따옴표(`"..."`)는 **절대 지우지 마세요!**
- 토큰을 따옴표 안에 넣으세요
- 앞뒤 공백 없이 딱 붙여넣으세요

5. 저장 (Cmd + S) 후 닫기

---

## STEP 4: 터미널에서 설치 실행

### 터미널 여는 법
1. **Cmd + 스페이스바** 누르기 (Spotlight 검색)
2. `터미널` 입력 후 엔터

### 명령어 실행

아래 명령어를 **한 줄씩** 복사해서 터미널에 붙여넣으세요 (Cmd + V):

**바탕화면에 놓은 경우:**
```
cd ~/Desktop/wooseok-dashboard
```

그 다음:
```
bash setup.sh
```

"매일 오전 10시 자동 실행 설정을 진행할까요?" 라고 물어보면 → `y` 입력 후 엔터

---

## STEP 5: 테스트

터미널에 입력:
```
python3 app.py
```

→ 브라우저가 자동으로 열리면서 대시보드가 보이면 성공!

**종료하려면**: 터미널에서 `Ctrl + C`

---

## 매일 사용법

설정 완료 후에는 매일 오전 10시에 자동으로 대시보드가 열립니다.

수동으로 열고 싶으면:
```
cd ~/Desktop/wooseok-dashboard && python3 app.py
```

---

## 문제가 생기면?

| 문제 | 해결법 |
|------|--------|
| "No module named requests" | 터미널에서 `pip3 install requests` 실행 |
| "토큰 오류" / 빈 화면 | config.py 열어서 토큰이 제대로 들어갔는지 확인 |
| 따옴표가 이상하게 보임 | 따옴표를 직접 타이핑 (`Shift + '`) |
| 10시에 안 열림 | 터미널에서 `launchctl list | grep dashboard` 확인 |
| 여전히 안 되면 | 수현님에게 스크린샷 찍어서 보내주세요! |

## 대시보드에서 할 수 있는 것

- **오늘 할 일**: Notion에서 자동으로 가져옴
- **이번주 업무**: 주간 마감 업무 확인
- **모닝 체크리스트**: 직접 항목 추가/체크 가능
- **국순당 파이프라인**: 시리즈별 콘텐츠 진행 현황
- **CH MAG 메모**: 자유롭게 메모 (자동 저장)
- **알림**: 마감 임박/공유 단계 미달 자동 감지
