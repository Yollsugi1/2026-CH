## 1. Plugin 설치 방법

### Step 1: Claude Cowork 접속
- Claude Desktop 앱을 열고 Cowork 모드로 진입합니다

### Step 2: Plugin 설치
- 폴더에 있는 `kuksundang-workflow.plugin` 파일을 Claude Cowork 대화창에 드래그앤드롭
- 또는 대화에서 "이 플러그인 설치해줘"라고 말하면서 파일을 첨부
- Plugin 미리보기가 나타나면 **수락(Accept)** 버튼 클릭

### Step 3: 커넥터 연결 확인
Plugin이 제대로 작동하려면 아래 커넥터가 연결되어 있어야 합니다:

| 커넥터 | 용도 | 확인 방법 |
|--------|------|----------|
| **Notion** | DB 조회, 페이지 생성 | Cowork 설정 → Connectors에서 확인 |
| **Slack** | 리포트 발송, 알림 | Cowork 설정 → Connectors에서 확인 |

> 커넥터가 연결 안 되어 있으면 Claude에게 "Notion 연결해줘" 또는 "Slack 연결해줘"라고 요청하세요.

## 2. 매일 사용하는 커맨드

### 🌅 오전 루틴: 데일리 리포트
```
/daily
```
또는 자연어로: "오늘 현황 보내줘", "데일리 리포트"

**결과**: Slack outsource-1-국순당 채널에 2개 메시지 자동 발송
1. 내부 데일리 현황 (오늘 Due, 내부 확인 필요, 다가오는 일정)
2. 클라이언트 공유 액션 가이드 (공유 단계, 피드백 기한)

### 📅 일정 확인
```
/schedule
```
또는: "일정 확인", "다음 공유 언제?", "마감 체크"

**결과**: 전체 콘텐츠의 D-N 역산 현황을 알려줍니다.

### 📊 주간 리포트 (매주 월요일)
```
/weekly
```
또는: "주간 리포트 보내줘", "이번주 현황"

**결과**: 시리즈별 진행 상황 + 주의 필요 + 미팅 논의 사항을 Slack으로 발송

### 📋 클라이언트 업데이트 (매주 금요일)
```
/client-update
```
또는: "클라이언트 보고", "연정님 보고 만들어줘"

**결과**: Notion 협업페이지에 보고 페이지 생성 + Slack 알림

### 🎬 콘텐츠 기획
```
/plan 순당포착 새 에피소드 아이디어
```
또는: "새 에피소드 기획해줘", "이 가게로 순당포착 기획해줘"

**결과**: A-B-C 프레임워크 기반 기획안 작성

## 3. 핵심 개념: 3단계 공유 시스템

국순당 담당자(연정님)에게는 30%-80%-100% 3단계로 공유합니다.
실제 내부에서는 각각 20%-50%-80% 단계에서 준비합니다.

| 우리 내부 | 국순당 공유 | 시점 | 피드백 기한 | 피드백 요청 |
|---|---|---|---|---|
| **20%** | 30% (1차) | 업로드 D-14 | D+2 | "방향성과 흐름 피드백" |
| **50%** | 80% (2차) | 업로드 D-10 | D+2 | "자막, 색감, 전환 등 디테일 피드백" |
| **80%** | 100% (최종) | 업로드 D-5 | D-3 | "최종 확인 후 업로드" |

**중요**: 1차에서 흐름 확정 → 2차에서 디테일 확정 → 최종에서 확인만

## 4. 시리즈 가이드

| 시리즈 | 컨셉 | 톤 |
|--------|------|-----|
| 🔵 **순당포착** | 가게/사장님의 진짜 이야기 | 따뜻한 다큐 감성 |
| 🟡 **취하지아니한가** | 막걸리와 함께하는 문화적 순간 | 감성적, 공감 |
| 🟠 **1막1장** | 막걸리가 떠오르는 감각적 장면 | 시네마틱, ASMR |

## 6. 연동된 Notion DB

| DB 이름 | 용도 |
|---------|------|
| 2026 국순당 콘텐츠 | 마스터 콘텐츠 목록 (시리즈, 업로드일) |
| 내부 업무관리 | 제작 단계별 일정 관리 |
| 국순당 작업 현황 | 클라이언트 공유 보드 |
| 협업페이지 | 클라이언트 보고 페이지 생성 위치 |

## 7. 트러블슈팅

| 문제 | 해결 |
|------|------|
| "Notion 조회 실패" | Notion 커넥터 재연결 (설정 → Connectors) |
| "Slack 발송 실패" | Slack 커넥터 재연결, 권한을 "항상 허용"으로 설정 |
| 데이터가 안 나옴 | Notion DB에 날짜 필드가 비어있지 않은지 확인 |
| 커맨드가 안 보임 | Plugin이 설치되었는지 확인 (설정 → Plugins) |
